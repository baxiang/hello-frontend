## L1 理解层

**最简示例（1-3行）：**

```tsx
const ExpensiveList = React.memo(({ items }) => <ul>{items.map(i => <li key={i.id}>{i.name}</li>)}</ul>)
const sorted = useMemo(() => items.sort(byName), [items])
const handleClick = useCallback(() => setCount(c => c + 1), [])
```

**详细示例：**

```tsx
import { memo, useMemo, useCallback, lazy, Suspense } from 'react'

// 1. React.memo — 避免不必要的渲染
const TodoItem = memo(({ todo, onToggle }: { todo: Todo; onToggle: () => void }) => {
  console.log('TodoItem render')
  return <li onClick={onToggle}>{todo.text}</li>
})

// 2. useMemo — 缓存计算结果
function TodoList({ todos, filter }: { todos: Todo[]; filter: string }) {
  const filtered = useMemo(() => todos.filter(t => t.text.includes(filter)), [todos, filter])
  return <ul>{filtered.map(t => <TodoItem key={t.id} todo={t} onToggle={() => {}} />)}</ul>
}

// 3. useCallback — 缓存函数引用
function TodoForm({ onAdd }: { onAdd: (text: string) => void }) {
  const [text, setText] = useState('')
  const handleSubmit = useCallback((e: React.FormEvent) => {
    e.preventDefault()
    onAdd(text)
    setText('')
  }, [onAdd, text])

  return <form onSubmit={handleSubmit}><input value={text} onChange={e => setText(e.target.value)} /></form>
}

// 4. 组件懒加载
const About = lazy(() => import('./About'))
function App() {
  return (
    <Suspense fallback={<div>加载中...</div>}>
      <About />
    </Suspense>
  )
}
```

## L2 实践层

### 优化工具速查

| 工具 | 用途 | 何时用 |
|------|------|--------|
| `React.memo` | 避免子组件重复渲染 | 组件渲染频繁且 props 不变 |
| `useMemo` | 缓存计算结果 | 计算开销大 |
| `useCallback` | 缓存函数引用 | 传递给 memo 子组件 |
| `lazy` | 组件懒加载 | 减少首屏体积 |

### 反模式

```tsx
// ❌ 错误：过度使用 memo（增加复杂度）
const SimpleText = memo(({ text }: { text: string }) => <span>{text}</span>)

// ✅ 正确：只在必要时使用
const ExpensiveList = memo(({ items }: { items: Item[] }) => <LongList items={items} />)
```
