# Next.js 部署与优化 ⭐⭐

> Vercel 部署、性能优化、SEO

---

## L1 理解层

**最简示例：**

```bash
# 构建
npm run build

# 启动生产服务器
npm start

# 部署到 Vercel（推荐）
npx vercel
```

---

## L2 实践层

### 部署选项

| 平台 | 说明 | 适合 |
|------|------|------|
| **Vercel** | Next.js 官方平台，零配置 | 首选 |
| **Netlify** | 支持 Next.js | 已有 Netlify 项目 |
| **Docker** | 自托管 | 需要私有部署 |
| **Node.js 服务器** | 自建服务器 | 完全控制 |

### 性能优化

// 图片优化
