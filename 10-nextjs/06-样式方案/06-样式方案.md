# Next.js 样式方案 ⭐

> CSS Modules、Tailwind、全局样式

---

## L1 理解层

**最简示例：**

```tsx
// CSS Modules
// app/page.tsx
import styles from './page.module.css'
export default function Page() {
  return <div className={styles.container}>内容</div>
}
```

```css
/* app/page.module.css */
.container { padding: 2rem; }
.container h1 { color: #333; }
```

---

## L2 实践层

### 样式方案对比

| 方案 | 优点 | 适用 |
|------|------|------|
| CSS Modules | 作用域隔离，零配置 | 通用 |
| Tailwind | 快速开发，一致性 | 新项目推荐 |
| 全局 CSS | 简单 | 全局样式重置 |

```tsx
// Tailwind（Next.js 默认支持）
export default function Page() {
  return <div className="p-4 bg-white rounded-lg shadow">内容</div>
}

// 全局样式
// app/globals.css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
