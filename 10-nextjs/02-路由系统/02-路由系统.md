## L1 理解层：会用

### 文件路由

**最简示例（1-3行）：**

```
app/page.tsx           → /
app/about/page.tsx     → /about
app/posts/page.tsx     → /posts
```

**详细示例：**

```tsx
// app/layout.tsx — 根布局（所有页面共享）
export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="zh">
      <body>
        <nav><a href="/">首页</a></nav>
        {children}
      </body>
    </html>
  )
}

// app/posts/layout.tsx — 嵌套布局（仅 /posts/* 共享）
export default function PostsLayout({ children }: { children: React.ReactNode }) {
  return <div className="posts-layout">{children}</div>
}

// app/posts/[id]/page.tsx — 动态路由
import { notFound } from 'next/navigation'

export default async function PostPage({ params }: { params: Promise<{ id: string }> }) {
  const { id } = await params
  const post = await fetch(`https://api.example.com/posts/${id}`).then(r => r.json())
  if (!post) notFound()
  return <article><h1>{post.title}</h1><p>{post.content}</p></article>
}
```

---

### 特殊文件

| 文件 | 用途 |
|------|------|
| `page.tsx` | 页面组件（必需） |
| `layout.tsx` | 布局组件（共享 UI） |
| `loading.tsx` | 加载状态 |
| `error.tsx` | 错误边界 |
| `not-found.tsx` | 404 页面 |

**详细示例：**

```tsx
// app/posts/loading.tsx
export default function Loading() {
  return <div className="animate-pulse">加载中...</div>
}

// app/posts/error.tsx
'use client'
export default function Error({ error, reset }: { error: Error; reset: () => void }) {
  return (
    <div>
      <h2>出错了：{error.message}</h2>
      <button onClick={reset}>重试</button>
    </div>
  )
}
```

## L2 实践层：用好

### 路由模式速查

| 模式 | 路径 | 匹配 |
|------|------|------|
| 静态 | `app/about/page.tsx` | `/about` |
| 动态 | `app/posts/[id]/page.tsx` | `/posts/1`, `/posts/abc` |
| 捕获所有 | `app/docs/[...slug]/page.tsx` | `/docs/a/b/c` |
| 可选捕获 | `app/docs/[[...slug]]/page.tsx` | `/docs`, `/docs/a/b` |

### 导航

```tsx
import Link from 'next/link'
import { useRouter } from 'next/navigation'

// Link 组件（推荐）
<Link href="/posts/1">文章一</Link>

// 编程式导航
'use client'
const router = useRouter()
router.push('/posts/1')
router.replace('/posts/1')
router.back()
```

### 反模式

```tsx
// ❌ 错误：用 a 标签导航（会全页刷新）
<a href="/about">关于</a>

// ✅ 正确：用 Link 组件（客户端导航）
<Link href="/about">关于</Link>
```
