## 学习建议

**本章是 CSS 布局的核心**，建议：

1. **理解 Flexbox 和 Grid 的区别** - 一维 vs 二维布局
2. **动手实验** - 用开发者工具实时调试布局属性
3. **完成导航栏练习** - 验证 Flexbox 掌握程度
4. **对比定位方式** - 理解不同定位的使用场景
