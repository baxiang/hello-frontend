## 3.1 CSS 语法基础

### 基本语法

```css
/* 选择器 { 属性：值； } */
h1 {
    color: #333;
    font-size: 24px;
    text-align: center;
}
```

### 三种引入方式

```html
<!-- 方式 1：行内样式（不推荐） -->
<p style="color: red; font-size: 16px;">内容</p>

<!-- 方式 2：内部样式表 -->
<head>
    <style>
        p { color: blue; }
    </style>
</head>

<!-- 方式 3：外部样式表（推荐） -->
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

### CSS 层叠与优先级

```css
/* 优先级从低到高 */
*                    /* 通配符选择器 */
元素选择器             /* 如：div, p */
类选择器               /* 如：.class */
属性选择器             /* 如：[type="text"] */
伪类                  /* 如：:hover, :first-child */
ID 选择器              /* 如：#id */
行内样式              /* style="..." */
!important            /* 强制最高优先级 */
```

```css
/* 优先级计算示例 */
p { color: red; }                    /* 0,0,0,1 */
.text { color: blue; }               /* 0,0,1,0 */
#main { color: green; }              /* 0,1,0,0 */
div.text { color: purple; }          /* 0,0,1,1 */
#main .text { color: orange; }       /* 0,1,1,0 */
```

## 3.2 选择器详解

### 基础选择器

```css
/* 通配符选择器 */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* 元素选择器 */
h1, h2, h3 {
    font-family: 'Microsoft YaHei', sans-serif;
}

/* 类选择器 */
.btn {
    padding: 10px 20px;
    border-radius: 4px;
}

/* ID 选择器 */
#header {
    height: 60px;
    background: #fff;
}
```

### 关系选择器

```css
/* 后代选择器（所有后代） */
div p {
    color: red;
}

/* 子元素选择器（直接子元素） */
div > p {
    color: blue;
}

/* 相邻兄弟选择器（紧接其后） */
h1 + p {
    font-size: 18px;
}

/* 通用兄弟选择器（所有后面兄弟） */
h1 ~ p {
    color: #666;
}
```

### 伪类选择器

```css
/* 链接状态 */
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: orange; }

/* 结构伪类 */
li:first-child { font-weight: bold; }
li:last-child { border-bottom: none; }
li:nth-child(odd) { background: #f9f9f9; }
li:nth-child(even) { background: #fff; }
li:nth-child(3n+1) { /* 每 3 个第 1 个 */ }

/* 表单伪类 */
input:focus { border-color: #007bff; }
input:checked + label { color: #007bff; }
input:disabled { background: #eee; }
input:valid { border-color: green; }
input:invalid { border-color: red; }

/* UI 伪类 */
p:not(.highlight) { color: #666; }
.btn:disabled { cursor: not-allowed; }
```

### 伪元素选择器

```css
/* 前后插入内容 */
p::before {
    content: "► ";
    color: #007bff;
}

p::after {
    content: " ◄";
    color: #007bff;
}

/* 首行首字 */
p::first-line {
    font-weight: bold;
}

p::first-letter {
    font-size: 2em;
    color: #007bff;
}

/* 选中文本 */
::selection {
    background: #007bff;
    color: #fff;
}
```

### 属性选择器

```css
/* 存在属性 */
[target] {
    color: red;
}

/* 精确匹配 */
[target="_blank"] {
    color: blue;
}

/* 开头匹配 */
[href^="https"] {
    color: green;
}

/* 结尾匹配 */
[href$=".pdf"] {
    color: orange;
}

/* 包含匹配 */
[class*="btn-"] {
    display: inline-block;
}

/* 单词匹配 */
[lang~="en"] {
    font-style: italic;
}
```

## 3.3 盒模型

### 盒模型组成

```
┌─────────────────────────────────────┐
│           Margin（外边距）            │
│  ┌───────────────────────────────┐  │
│  │      Border（边框）            │  │
│  │  ┌─────────────────────────┐  │  │
│  │  │    Padding（内边距）     │  │  │
│  │  │  ┌───────────────────┐  │  │  │
│  │  │  │   Content（内容）  │  │  │  │
│  │  │  └───────────────────┘  │  │  │
│  │  └─────────────────────────┘  │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

### 盒模型属性

```css
.box {
    /* 尺寸 */
    width: 300px;
    height: 200px;

    /* 内边距 */
    padding: 10px;              /* 上下左右 */
    padding: 10px 20px;         /* 上下 左右 */
    padding: 10px 20px 30px;    /* 上 左右 下 */
    padding: 10px 20px 30px 40px; /* 上 右 下 左 */

    /* 边框 */
    border: 1px solid #333;     /* 简写 */
    border-width: 1px;
    border-style: solid;        /* solid, dashed, dotted, none */
    border-color: #333;
    border-radius: 8px;         /* 圆角 */

    /* 外边距 */
    margin: 10px;
    margin: 10px auto;          /* 上下 10px，左右居中 */
}
```

### 盒模型类型

```css
/* 标准盒模型（content-box） */
.box {
    width: 300px;
    box-sizing: content-box; /* 默认，width = content */
    /* 实际宽度 = 300 + padding + border */
}

/* IE 盒模型（border-box） */
.box {
    width: 300px;
    box-sizing: border-box;  /* width = content + padding + border */
    /* 实际宽度 = 300 */
}

/* 推荐全局设置 */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

### 外边距合并

```css
/* 垂直方向的外边距会合并，取较大值 */
.div1 { margin-bottom: 20px; }
.div2 { margin-top: 30px; }
/* 实际间距 = 30px（不是 50px） */

/* 防止外边距合并 */
.parent {
    overflow: hidden; /* 或 display: flow-root */
}

/* 或使用 padding/border 分隔 */
.parent {
    padding-top: 1px;
    margin-top: -1px;
}
```

## 3.4 文本样式

### 字体属性

```css
.text {
    /* 字体族 */
    font-family: 'Helvetica Neue', Helvetica, Arial, sans-serif;

    /* 字体大小 */
    font-size: 16px;       /* 绝对单位 */
    font-size: 1rem;       /* 相对根元素 */
    font-size: 1.5em;      /* 相对父元素 */

    /* 字体粗细 */
    font-weight: normal;   /* 400 */
    font-weight: bold;     /* 700 */
    font-weight: 600;

    /* 字体样式 */
    font-style: normal;
    font-style: italic;

    /* 简写（顺序：style weight size/line-height family） */
    font: italic bold 16px/1.5 'Microsoft YaHei', sans-serif;
}
```

### 文本属性

```css
.text {
    /* 颜色 */
    color: #333;
    color: rgba(0, 0, 0, 0.5);

    /* 对齐 */
    text-align: left;
    text-align: center;
    text-align: right;
    text-align: justify;   /* 两端对齐 */

    /* 行高 */
    line-height: 1.5;      /* 推荐无单位 */
    line-height: 24px;

    /* 缩进 */
    text-indent: 2em;      /* 首行缩进 2 字符 */

    /* 装饰 */
    text-decoration: none;
    text-decoration: underline;
    text-decoration: line-through;
    text-decoration: overline;

    /* 转换 */
    text-transform: none;
    text-transform: uppercase;
    text-transform: lowercase;
    text-transform: capitalize;

    /* 阴影 */
    text-shadow: 1px 1px 2px rgba(0,0,0,0.3);
    text-shadow: 2px 2px 4px #000;
}
```

### 文本溢出处理

```css
/* 单行省略 */
.ellipsis {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
}

/* 多行省略（Webkit） */
.multi-ellipsis {
    display: -webkit-box;
    -webkit-line-clamp: 3;
    -webkit-box-orient: vertical;
    overflow: hidden;
}

/* 强制换行 */
.break-word {
    word-wrap: break-word;
    word-break: break-all;   /* 所有字符都可断行 */
    word-break: break-word;  /* 仅单词断行 */
}
```

## 3.5 背景样式

### 背景属性

```css
.box {
    /* 背景颜色 */
    background-color: #f5f5f5;
    background-color: rgba(0,0,0,0.1);

    /* 背景图片 */
    background-image: url('image.jpg');

    /* 背景重复 */
    background-repeat: no-repeat;
    background-repeat: repeat-x;
    background-repeat: repeat-y;

    /* 背景位置 */
    background-position: center center;
    background-position: left top;
    background-position: 50% 50%;
    background-position: 10px 20px;

    /* 背景尺寸 */
    background-size: cover;     /* 覆盖整个容器 */
    background-size: contain;   /* 完整显示 */
    background-size: 100% auto;

    /* 背景附着 */
    background-attachment: scroll;  /* 默认，随页面滚动 */
    background-attachment: fixed;   /* 固定背景（视差滚动） */

    /* 背景裁剪 */
    background-clip: border-box;    /* 默认，到边框 */
    background-clip: padding-box;   /* 到内边距 */
    background-clip: content-box;   /* 到内容 */

    /* 简写 */
    background: #fff url('bg.jpg') no-repeat center center / cover;
}
```

### 多重背景

```css
.box {
    background-image:
        linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
        url('image.jpg');
    background-size: cover;
    background-position: center;
}
```

### 渐变背景

```css
/* 线性渐变 */
.linear {
    background: linear-gradient(to right, #ff0000, #00ff00);
    background: linear-gradient(90deg, red, blue, green);
    background: linear-gradient(to bottom right, red, blue);
}

/* 径向渐变 */
.radial {
    background: radial-gradient(circle, red, blue);
    background: radial-gradient(ellipse at center, red 0%, blue 100%);
}

/* 锥形渐变 */
.conic {
    background: conic-gradient(red, yellow, green, blue, red);
}

/* 条纹背景 */
.stripe {
    background: repeating-linear-gradient(
        45deg,
        #606dbc,
        #606dbc 10px,
        #465298 10px,
        #465298 20px
    );
}
```

## 3.6 列表样式

```css
ul {
    /* 列表符号类型 */
    list-style-type: disc;      /* 实心圆（默认） */
    list-style-type: circle;    /* 空心圆 */
    list-style-type: square;    /* 实心方块 */
    list-style-type: none;      /* 无符号 */
    list-style-type: decimal;   /* 数字 */
    list-style-type: lower-alpha; /* a, b, c */
    list-style-type: upper-alpha; /* A, B, C */

    /* 符号位置 */
    list-style-position: outside; /* 默认，在外面 */
    list-style-position: inside;  /* 在里面 */

    /* 符号图片 */
    list-style-image: url('bullet.png');

    /* 简写 */
    list-style: none inside url('bullet.png');
}
```

## 3.7 CSS 变量

```css
/* 定义全局变量 */
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    --font-size-base: 16px;
    --spacing-unit: 8px;
    --border-radius: 4px;
}

/* 使用变量 */
.btn {
    background-color: var(--primary-color);
    color: #fff;
    padding: calc(var(--spacing-unit) * 2) calc(var(--spacing-unit) * 3);
    border-radius: var(--border-radius);
    font-size: var(--font-size-base);
}

/* 带后备值 */
.card {
    background: var(--card-bg, #fff);
}

/* 在媒体查询中使用 */
@media (max-width: 768px) {
    :root {
        --font-size-base: 14px;
        --spacing-unit: 4px;
    }
}
```

## 3.9 常见问题

### Q1: em 和 rem 有什么区别？

**答：**
- `em`：相对父元素的字体大小
- `rem`：相对根元素（html）的字体大小

```css
html { font-size: 16px; }
.parent { font-size: 20px; }
.child {
    font-size: 1.5em;   /* 20px * 1.5 = 30px */
    font-size: 1.5rem;  /* 16px * 1.5 = 24px */
}
```

### Q2: 如何垂直居中一个元素？

**答：** 多种方法：

```css
/* 方法 1：Flexbox（推荐） */
.parent {
    display: flex;
    align-items: center;
    justify-content: center;
}

/* 方法 2：Grid */
.parent {
    display: grid;
    place-items: center;
}

/* 方法 3：绝对定位 + transform */
.parent {
    position: relative;
}
.child {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
```

### Q3: 如何清除浮动？

**答：**

```css
/* 方法 1：clearfix（推荐） */
.clearfix::after {
    content: '';
    display: table;
    clear: both;
}

/* 方法 2：overflow */
.parent {
    overflow: hidden;
}

/* 方法 3：display: flow-root（现代） */
.parent {
    display: flow-root;
}
```

### Q4: display: none 和 visibility: hidden 有什么区别？

| 属性 | 显示 | 占位 | 继承 |
|------|------|------|------|
| `display: none` | 不显示 | 不占位 | 子元素可显示 |
| `visibility: hidden` | 不显示 | 占位 | 子元素也隐藏 |
