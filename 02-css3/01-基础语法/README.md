## 学习建议

**本章是 CSS 的起点**，建议：

1. **边学边写** - 每学一个属性就动手写代码
2. **用浏览器实时查看** - 即时反馈加深理解
3. **理解盒模型** - 这是 CSS 的核心概念
4. **完成卡片组件练习** - 验证学习成果
