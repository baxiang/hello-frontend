## 6.1 媒体查询

### 基本语法

```css
/* 当视口宽度大于等于 768px 时应用 */
@media (min-width: 768px) {
    .container {
        max-width: 720px;
        margin: 0 auto;
    }
}

/* 当视口宽度小于 768px 时应用 */
@media (max-width: 767px) {
    .container {
        padding: 0 15px;
    }
}

/* 当视口宽度在 768px 到 991px 之间 */
@media (min-width: 768px) and (max-width: 991px) {
    .sidebar {
        width: 200px;
    }
}
```

### 常用断点

```css
/* 常见设备断点 */
/* 超小设备（手机） */
@media (max-width: 575px) { }

/* 小设备（大手机/小平板） */
@media (min-width: 576px) and (max-width: 767px) { }

/* 中等设备（平板） */
@media (min-width: 768px) and (max-width: 991px) { }

/* 大设备（桌面） */
@media (min-width: 992px) and (max-width: 1199px) { }

/* 超大设备（大桌面） */
@media (min-width: 1200px) { }
```

### Bootstrap 断点（参考）

```css
/* Bootstrap 5 断点 */
/* xs: 0, sm: 576px, md: 768px, lg: 992px, xl: 1200px, xxl: 1400px */

@media (min-width: 576px) { }  /* sm */
@media (min-width: 768px) { }  /* md */
@media (min-width: 992px) { }  /* lg */
@media (min-width: 1200px) { } /* xl */
@media (min-width: 1400px) { } /* xxl */
```

### 媒体查询特性

```css
/* 方向 */
@media (orientation: portrait) { }  /* 纵向 */
@media (orientation: landscape) { } /* 横向 */

/* 分辨率 */
@media (resolution: 2dppx) { }      /* 2 倍屏 */
@media (resolution: 300dpi) { }     /* 300 DPI */

/* 颜色 */
@media (prefers-color-scheme: dark) { }   /* 深色模式 */
@media (prefers-color-scheme: light) { }  /* 浅色模式 */

/* 动态效果偏好 */
@media (prefers-reduced-motion: reduce) {
    * {
        animation: none !important;
        transition: none !important;
    }
}

/* 对比度偏好 */
@media (prefers-contrast: high) { }   /* 高对比度 */
@media (prefers-contrast: low) { }    /* 低对比度 */

/* 横竖屏 */
@media screen and (orientation: landscape) { }
```

## 6.2 移动优先

### 设计理念

```css
/* 先写移动端样式（基础样式） */
.container {
    padding: 0 15px;
    width: 100%;
}

/* 再为大屏设备添加样式 */
@media (min-width: 768px) {
    .container {
        padding: 0 30px;
        max-width: 720px;
    }
}

@media (min-width: 992px) {
    .container {
        max-width: 960px;
    }
}

@media (min-width: 1200px) {
    .container {
        max-width: 1140px;
    }
}
```

### 移动优先实践

```css
/* 导航栏：移动端汉堡菜单 */
.nav {
    display: flex;
    flex-direction: column;
    gap: 10px;
}

.nav-toggle {
    display: block;  /* 移动端显示 */
}

.nav-links {
    display: none;   /* 移动端隐藏 */
    flex-direction: column;
}

.nav-links.active {
    display: flex;
}

/* 平板及以上 */
@media (min-width: 768px) {
    .nav {
        flex-direction: row;
        justify-content: space-between;
    }

    .nav-toggle {
        display: none;  /* 隐藏汉堡菜单 */
    }

    .nav-links {
        display: flex;  /* 显示导航链接 */
        flex-direction: row;
        gap: 20px;
    }
}
```

## 6.3 响应式布局技术

### 流式布局（Fluid Layout）

```css
/* 使用百分比宽度 */
.container {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
}

.column {
    width: 50%;      /* 两列 */
    float: left;
    padding: 0 15px;
}

@media (max-width: 768px) {
    .column {
        width: 100%;  /* 移动端单列 */
    }
}
```

### Flexbox 响应式

```css
/* 卡片自动换行 */
.cards {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 1 1 300px;  /* 最小 300px，自动放大 */
    max-width: 100%;
}

/* 或者使用 calc */
.card {
    flex: 0 0 calc(33.333% - 20px);  /* 三列 */
}

@media (max-width: 992px) {
    .card {
        flex: 0 0 calc(50% - 15px);  /* 两列 */
    }
}

@media (max-width: 576px) {
    .card {
        flex: 0 0 100%;  /* 单列 */
    }
}
```

### Grid 响应式

```css
/* 自动填充列 */
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
    gap: 20px;
}

/* 响应式网格 */
.responsive-grid {
    display: grid;
    gap: 20px;
    grid-template-columns: 1fr;  /* 移动端单列 */
}

@media (min-width: 576px) {
    .responsive-grid {
        grid-template-columns: repeat(2, 1fr);  /* 两列 */
    }
}

@media (min-width: 992px) {
    .responsive-grid {
        grid-template-columns: repeat(4, 1fr);  /* 四列 */
    }
}

/* 使用 auto-fit 和 minmax */
.gallery {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 16px;
}
```

### 响应式容器

```css
/* 通用容器 */
.container {
    width: 100%;
    padding-left: 15px;
    padding-right: 15px;
    margin-left: auto;
    margin-right: auto;
}

@media (min-width: 576px) {
    .container { max-width: 540px; }
}
@media (min-width: 768px) {
    .container { max-width: 720px; }
}
@media (min-width: 992px) {
    .container { max-width: 960px; }
}
@media (min-width: 1200px) {
    .container { max-width: 1140px; }
}
```

## 6.4 响应式图片

### 使用 srcset

```html
<!-- 根据屏幕密度加载不同图片 -->
<img src="image-800.jpg"
     srcset="image-400.jpg 400w,
             image-800.jpg 800w,
             image-1200.jpg 1200w"
     sizes="(max-width: 600px) 400px,
            (max-width: 1200px) 800px,
            1200px"
     alt="响应式图片">

<!-- 简单用法 -->
<img src="small.jpg"
     srcset="small.jpg 1x, medium.jpg 2x, large.jpg 3x"
     alt="高清图片">
```

### 使用 picture 元素

```html
<!-- 艺术指导：不同屏幕尺寸显示不同裁切的图片 -->
<picture>
    <source media="(min-width: 1200px)" srcset="large.jpg">
    <source media="(min-width: 768px)" srcset="medium.jpg">
    <img src="small.jpg" alt="回退图片">
</picture>

<!-- 不同格式 -->
<picture>
    <source type="image/avif" srcset="image.avif">
    <source type="image/webp" srcset="image.webp">
    <img src="image.jpg" alt="图片">
</picture>
```

### CSS 响应式图片

```css
/* 流体图片 */
img {
    max-width: 100%;
    height: auto;
}

/* 背景图片响应式 */
.hero {
    background-image: url('hero-mobile.jpg');
    background-size: cover;
    background-position: center;
}

@media (min-width: 768px) {
    .hero {
        background-image: url('hero-desktop.jpg');
    }
}

/* 使用 image-set（现代浏览器） */
.hero {
    background-image: image-set(
        url('hero-1x.jpg') 1x,
        url('hero-2x.jpg') 2x,
        url('hero-3x.jpg') 3x
    );
}
```

## 6.5 响应式文本

### 流体字体

```css
/* 使用 clamp() */
html {
    font-size: clamp(14px, 1vw + 10px, 18px);
}

/* 标题字体 */
h1 {
    font-size: clamp(24px, 5vw, 48px);
}

h2 {
    font-size: clamp(20px, 4vw, 36px);
}

/* 使用 calc 和 vw */
h1 {
    font-size: calc(16px + 4vw);
}
```

### 响应式间距

```css
:root {
    --spacing-base: clamp(1rem, 2vw, 2rem);
}

.section {
    padding: calc(var(--spacing-base) * 2) 0;
}

.container {
    padding: 0 var(--spacing-base);
}
```

## 6.7 常见问题

### Q1: 断点应该如何选择？

**答：** 根据内容决定，而不是特定设备：

```css
/* 不推荐：针对特定设备 */
@media (max-width: 768px) { } /* iPad */

/* 推荐：根据内容 */
@media (min-width: 768px) {
    /* 当内容需要更多空间时 */
}
```

### Q2: viewport meta 标签怎么用？

```html
<!-- 标准写法 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 禁止缩放（不推荐，影响无障碍） -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

### Q3: rem 和 em 如何选择？

```css
/* rem：相对根元素，适合全局控制 */
html { font-size: 16px; }
body { font-size: 1rem; }  /* 16px */

/* em：相对父元素，适合组件内部 */
.button {
    font-size: 1em;
    padding: 0.5em 1em;  /* 相对按钮字体大小 */
}
```

### Q4: 如何处理安全区域（iPhone 刘海屏）？

```css
/* 使用 env() */
.safe-area {
    padding-top: env(safe-area-inset-top);
    padding-bottom: env(safe-area-inset-bottom);
    padding-left: env(safe-area-inset-left);
    padding-right: env(safe-area-inset-right);
}

/* 带后备值 */
.safe-area {
    padding-top: max(20px, env(safe-area-inset-top));
}
```
