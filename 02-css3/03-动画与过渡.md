# CSS3 动画与过渡

## 学习目标
- 掌握 transition 过渡的基本用法
- 理解 animation 动画的核心概念
- 学会 keyframes 关键帧的定义
- 了解 transform 在动画中的应用

## 5.1 transition 过渡

### 基本语法

```css
.box {
    /* 属性 时长 时序函数 延迟时间 */
    transition: transform 0.3s ease-in-out 0.1s;
}
```

### 各个属性详解

```css
.box {
    /* 过渡属性 */
    transition-property: all;  /* all | none | 具体属性 */

    /* 过渡时长 */
    transition-duration: 0.3s;  /* 秒 */
    transition-duration: 300ms; /* 毫秒 */

    /* 时序函数 */
    transition-timing-function: ease;        /* 默认，慢 - 快 - 慢 */
    transition-timing-function: linear;      /* 匀速 */
    transition-timing-function: ease-in;     /* 慢 - 快 */
    transition-timing-function: ease-out;    /* 快 - 慢 */
    transition-timing-function: ease-in-out; /* 慢 - 快 - 慢 */
    transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); /* 自定义 */

    /* 延迟时间 */
    transition-delay: 0s;  /* 无延迟 */
    transition-delay: 0.2s; /* 延迟 0.2 秒 */

    /* 简写 */
    transition: all 0.3s ease-in-out 0.1s;
}
```

### 多属性过渡

```css
.box {
    /* 多个属性用逗号分隔 */
    transition:
        transform 0.3s ease,
        opacity 0.2s linear,
        background-color 0.5s ease-in-out;
}
```

### Transition 实例

```html
<style>
/* 按钮悬停效果 */
.btn {
    padding: 12px 24px;
    background: #007bff;
    color: white;
    border: none;
    border-radius: 6px;
    cursor: pointer;
    transition: background-color 0.3s ease, transform 0.2s ease;
}

.btn:hover {
    background: #0056b3;
    transform: scale(1.05);
}

.btn:active {
    transform: scale(0.95);
}

/* 卡片悬停上浮 */
.card {
    background: white;
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.1);
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.card:hover {
    transform: translateY(-8px);
    box-shadow: 0 12px 24px rgba(0,0,0,0.15);
}

/* 输入框聚焦效果 */
.input {
    width: 100%;
    padding: 12px;
    border: 2px solid #ddd;
    border-radius: 8px;
    transition: border-color 0.3s ease, box-shadow 0.3s ease;
}

.input:focus {
    border-color: #007bff;
    box-shadow: 0 0 0 3px rgba(0,123,255,0.1);
    outline: none;
}

/* 开关切换 */
.switch {
    width: 60px;
    height: 30px;
    background: #ddd;
    border-radius: 15px;
    position: relative;
    transition: background 0.3s ease;
}

.switch.on {
    background: #28a745;
}

.switch::after {
    content: '';
    position: absolute;
    top: 2px;
    left: 2px;
    width: 26px;
    height: 26px;
    background: white;
    border-radius: 50%;
    transition: transform 0.3s ease;
}

.switch.on::after {
    transform: translateX(30px);
}
</style>
```

### 可过渡的属性

```css
/* 颜色类 */
color, background-color, border-color, box-shadow

/* 尺寸类 */
width, height, padding, margin

/* 位置类 */
top, right, bottom, left, transform

/* 视觉效果 */
opacity, filter, blur

/* 注意：display 不能过渡 */
/* 错误示例 */
.box {
    transition: display 0.3s; /* 无效 */
}
```

## 5.2 animation 动画

### 基本语法

```css
@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

.box {
    animation: slideIn 0.5s ease-in-out;
}
```

### 动画属性详解

```css
.box {
    /* 动画名称 */
    animation-name: slideIn;

    /* 动画时长 */
    animation-duration: 1s;

    /* 时序函数 */
    animation-timing-function: ease-in-out;

    /* 延迟时间 */
    animation-delay: 0.5s;

    /* 播放次数 */
    animation-iteration-count: 1;      /* 具体次数 */
    animation-iteration-count: infinite; /* 无限循环 */

    /* 播放方向 */
    animation-direction: normal;       /* 默认，正向 */
    animation-direction: reverse;      /* 反向 */
    animation-direction: alternate;    /* 交替（奇数正向，偶数反向） */
    animation-direction: alternate-reverse; /* 交替（奇数反向，偶数正向） */

    /* 填充模式 */
    animation-fill-mode: none;         /* 默认，不应用样式 */
    animation-fill-mode: forwards;     /* 保持结束状态 */
    animation-fill-mode: backwards;    /* 应用初始状态（延迟期间） */
    animation-fill-mode: both;         /* forwards + backwards */

    /* 播放状态 */
    animation-play-state: running;     /* 默认，播放 */
    animation-play-state: paused;      /* 暂停 */

    /* 简写 */
    animation: slideIn 1s ease-in-out 0.5s infinite alternate both;
}
```

### keyframes 关键帧

```css
/* 基础关键帧 */
@keyframes fadeIn {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}

/* 多关键帧 */
@keyframes bounce {
    0% {
        transform: translateY(0);
    }
    25% {
        transform: translateY(-30px);
    }
    50% {
        transform: translateY(0);
    }
    75% {
        transform: translateY(-15px);
    }
    100% {
        transform: translateY(0);
    }
}

/* 复杂动画 */
@keyframes spin-and-scale {
    0% {
        transform: rotate(0deg) scale(1);
        background: red;
    }
    50% {
        transform: rotate(180deg) scale(1.5);
        background: yellow;
    }
    100% {
        transform: rotate(360deg) scale(1);
        background: red;
    }
}
```

## 5.3 transform 变换

### 2D 变换

```css
/* 平移 */
.element {
    transform: translate(50px, 100px);
    transform: translateX(50px);
    transform: translateY(50px);
}

/* 缩放 */
.element {
    transform: scale(1.5);      /* 放大 1.5 倍 */
    transform: scale(0.5);      /* 缩小 */
    transform: scaleX(2);       /* X 轴放大 */
    transform: scaleY(0.5);     /* Y 轴缩小 */
    transform: scale(-1, 1);    /* 水平翻转 */
    transform: scale(1, -1);    /* 垂直翻转 */
}

/* 旋转 */
.element {
    transform: rotate(45deg);   /* 顺时针 45 度 */
    transform: rotate(-45deg);  /* 逆时针 */
}

/* 倾斜 */
.element {
    transform: skew(30deg, 20deg);
    transform: skewX(30deg);
    transform: skewY(20deg);
}

/* 矩阵（不常用） */
.element {
    transform: matrix(1, 0, 0, 1, 50, 100);
}
```

### 3D 变换

```css
.element {
    /* 启用 3D 空间 */
    transform-style: preserve-3d;
    perspective: 1000px;

    /* 3D 平移 */
    transform: translateZ(50px);
    transform: translate3d(50px, 100px, 150px);

    /* 3D 旋转 */
    transform: rotateX(45deg);
    transform: rotateY(45deg);
    transform: rotateZ(45deg);
    transform: rotate3d(1, 1, 1, 45deg);

    /* 3D 缩放 */
    transform: scaleZ(2);
    transform: scale3d(1, 1, 2);
}
```

### 变换组合

```css
/* 多个变换 */
.complex {
    transform: translateX(100px) rotate(45deg) scale(1.5);
}

/* 变换原点 */
.element {
    transform-origin: center center;  /* 默认 */
    transform-origin: top left;
    transform-origin: 10px 20px;
    transform-origin: 50% 50% 0;      /* 3D */
}
```

## 5.5 性能优化

### 使用 will-change

```css
/* 告诉浏览器哪些属性会变化 */
.animated {
    will-change: transform, opacity;
}

/* 使用后记得移除 */
.element {
    animation: slide 1s ease-out;
}
.element {
    animation-fill-mode: forwards;
}
.element::after {
    content: '';
    animation: cleanup 0s 1s; /* 动画结束后清理 */
}
@keyframes cleanup {
    to { will-change: auto; }
}
```

### 启用 GPU 加速

```css
/* 触发 GPU 加速 */
.gpu-accelerated {
    transform: translateZ(0);
    /* 或 */
    transform: translate3d(0, 0, 0);
    /* 或 */
    backface-visibility: hidden;
}
```

### 避免的属性

```css
/* 这些属性会触发重排，性能较差 */
.bad {
    transition: width 0.3s;      /* 重排 */
    transition: height 0.3s;     /* 重排 */
    transition: margin 0.3s;     /* 重排 */
    transition: top 0.3s;        /* 重排 */
}

/* 这些属性只触发重绘，性能较好 */
.good {
    transition: transform 0.3s;  /* 重绘 */
    transition: opacity 0.3s;    /* 重绘 */
    transition: filter 0.3s;     /* 重绘 */
}
```

## 5.6 常见问题

### Q1: transition 和 animation 有什么区别？

| 特性 | transition | animation |
|------|-----------|-----------|
| 触发方式 | 状态变化（hover、focus 等） | 自动播放 |
| 循环播放 | 不支持 | 支持 |
| 关键帧 | 只有始末状态 | 多关键帧 |
| 延迟执行 | 支持 | 支持 |

### Q2: 如何暂停动画？

```css
/* 暂停动画 */
.paused {
    animation-play-state: paused;
}

/* 鼠标悬停暂停 */
.element:hover {
    animation-play-state: paused;
}
```

### Q3: 如何制作无限循环动画？

```css
@keyframes spin {
    to { transform: rotate(360deg); }
}

.spinner {
    animation: spin 1s linear infinite;
}
```

### Q4: 动画闪烁怎么办？

```css
/* 添加这些属性 */
.element {
    backface-visibility: hidden;
    perspective: 1000px;
    transform: translateZ(0);
}
```
