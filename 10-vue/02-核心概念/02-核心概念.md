## 2.1 响应式系统

### ref 响应式引用

```vue
<template>
  <div>
    <p>{{ count }}</p>
    <button @click="increment">+1</button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

// ref 用于基础类型
const count = ref(0)

// 修改时需要 .value
const increment = () => {
  count.value++
}
</script>
```

### reactive 响应式对象

```vue
<template>
  <div>
    <p>{{ user.name }}</p>
    <p>{{ user.age }}</p>
    <button @click="user.age++">年龄+1</button>
  </div>
</template>

<script setup>
import { reactive } from 'vue'

// reactive 用于对象
const user = reactive({
  name: '张三',
  age: 25
})

// 直接修改属性，不需要 .value
</script>
```

### ref vs reactive

| 特性 | ref | reactive |
|------|-----|----------|
| 适用类型 | 基础类型、对象 | 对象、数组 |
| 访问方式 | 需要 `.value` | 直接访问 |
| 解构 | 保持响应式 | 丢失响应式 |
| 替换 | 可以整体替换 | 不能整体替换 |

```vue
<script setup>
import { ref, reactive, toRefs } from 'vue'

// ref：可以整体替换
const userRef = ref({ name: '张三' })
userRef.value = { name: '李四' }  // ✅ 有效

// reactive：不能整体替换
const userReactive = reactive({ name: '张三' })
userReactive = { name: '李四' }   // ❌ 丢失响应式

// 使用 toRefs 解构保持响应式
const { name, age } = toRefs(userReactive)
</script>
```

## 2.2 计算属性

### 基础用法

```vue
<template>
  <div>
    <p>原价：{{ price }}</p>
    <p>折扣价：{{ discountPrice }}</p>
    <p>总价：{{ total }}</p>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const price = ref(100)
const quantity = ref(5)

// 计算属性：自动追踪依赖
const discountPrice = computed(() => price.value * 0.8)
const total = computed(() => discountPrice.value * quantity.value)
</script>
```

### 可写计算属性

```vue
<template>
  <div>
    <input v-model="fullName" />
    <p>姓：{{ firstName }}</p>
    <p>名：{{ lastName }}</p>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const firstName = ref('张')
const lastName = ref('三')

// 可写计算属性
const fullName = computed({
  get() {
    return firstName.value + lastName.value
  },
  set(newValue) {
    firstName.value = newValue[0]
    lastName.value = newValue.slice(1)
  }
})
</script>
```

### computed vs 方法

```vue
<script setup>
import { ref, computed } from 'vue'

const list = ref([1, 2, 3, 4, 5])

// 计算属性：有缓存，依赖不变不重新计算
const sum = computed(() => {
  console.log('重新计算')
  return list.value.reduce((a, b) => a + b, 0)
})

// 方法：每次调用都执行
const getSum = () => {
  console.log('重新执行')
  return list.value.reduce((a, b) => a + b, 0)
}
</script>

<template>
  <!-- 计算属性：多次使用只计算一次 -->
  <p>{{ sum }}</p>
  <p>{{ sum }}</p>
  
  <!-- 方法：每次调用都执行 -->
  <p>{{ getSum() }}</p>
  <p>{{ getSum() }}</p>
</template>
```

## 2.3 侦听器

### watch 基础

```vue
<template>
  <div>
    <input v-model="question" />
    <p>{{ answer }}</p>
  </div>
</template>

<script setup>
import { ref, watch } from 'vue'

const question = ref('')
const answer = ref('等待提问...')

// 监听 ref
watch(question, (newValue, oldValue) => {
  if (newValue.includes('?')) {
    answer.value = '思考中...'
    // 模拟异步请求
    setTimeout(() => {
      answer.value = '答案是...'
    }, 1000)
  }
})
</script>
```

### 监听多个源

```vue
<script setup>
import { ref, watch } from 'vue'

const x = ref(0)
const y = ref(0)

// 监听多个 ref
watch([x, y], ([newX, newY], [oldX, oldY]) => {
  console.log(`x: ${oldX} -> ${newX}, y: ${oldY} -> ${newY}`)
})

// 监听 reactive 对象属性
const user = reactive({ name: '', age: 0 })
watch(
  () => user.age,  // 使用 getter 函数
  (newAge, oldAge) => {
    console.log(`age: ${oldAge} -> ${newAge}`)
  }
)
</script>
```

### watchEffect

```vue
<script setup>
import { ref, watchEffect } from 'vue'

const count = ref(0)

// 自动追踪依赖，立即执行
watchEffect(() => {
  console.log(`count is ${count.value}`)
  // 自动追踪 count，不需要指定依赖
})

// 停止侦听
const stop = watchEffect(() => {
  // ...
})
stop() // 手动停止
</script>
```

### watch vs watchEffect

| 特性 | watch | watchEffect |
|------|-------|-------------|
| 执行时机 | 惰性，依赖变化才执行 | 立即执行一次 |
| 依赖指定 | 手动指定 | 自动追踪 |
| 获取旧值 | ✅ 可以 | ❌ 不能 |
| 适用场景 | 需要旧值、特定依赖 | 自动追踪依赖 |

## 2.4 生命周期

### 生命周期钩子

```vue
<script setup>
import {
  onBeforeMount,
  onMounted,
  onBeforeUpdate,
  onUpdated,
  onBeforeUnmount,
  onUnmounted
} from 'vue'

onBeforeMount(() => {
  console.log('挂载前')
})

onMounted(() => {
  console.log('挂载后')
  // DOM 已可用
})

onBeforeUpdate(() => {
  console.log('更新前')
})

onUpdated(() => {
  console.log('更新后')
})

onBeforeUnmount(() => {
  console.log('卸载前')
})

onUnmounted(() => {
  console.log('卸载后')
  // 清理定时器、事件监听等
})
</script>
```

### 生命周期图示

```
┌─────────────────────────────────────────────────────────────┐
│  Vue 生命周期                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  创建 → onBeforeMount → onMounted                           │
│                                                             │
│  更新 → onBeforeUpdate → onUpdated                          │
│                                                             │
│  销毁 → onBeforeUnmount → onUnmounted                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 2.5 模板引用

### ref 获取 DOM

```vue
<template>
  <input ref="inputRef" />
  <button @click="focusInput">聚焦</button>
</template>

<script setup>
import { ref, onMounted } from 'vue'

// 同名 ref 自动绑定
const inputRef = ref(null)

onMounted(() => {
  inputRef.value.focus()
})

const focusInput = () => {
  inputRef.value.focus()
}
</script>
```

### v-for 中的 ref

```vue
<template>
  <div v-for="item in list" :key="item.id" :ref="setItemRef">
    {{ item.name }}
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'

const list = ref([{ id: 1, name: 'A' }, { id: 2, name: 'B' }])
const itemRefs = ref([])

const setItemRef = (el) => {
  if (el) {
    itemRefs.value.push(el)
  }
}

onMounted(() => {
  console.log(itemRefs.value) // 所有 DOM 元素
})
</script>
```

## 2.6 常见问题

### Q1: 为什么 reactive 解构后失去响应式？

```vue
<script setup>
import { reactive, toRefs } from 'vue'

const user = reactive({ name: '张三', age: 25 })

// ❌ 错误：直接解构失去响应式
const { name, age } = user

// ✅ 正确：使用 toRefs
const { name, age } = toRefs(user)
</script>
```

### Q2: computed 和 watch 有什么区别？

- `computed`：有缓存，返回值，用于派生数据
- `watch`：无缓存，执行副作用，用于响应变化

### Q3: 如何在 watch 中深度监听对象？

```vue
<script setup>
import { watch } from 'vue'

const user = reactive({ name: '', info: { age: 0 } })

// 深度监听
watch(
  () => user,
  (newUser) => {
    console.log('user changed')
  },
  { deep: true }
)

// 立即执行
watch(
  () => user.name,
  (name) => {
    console.log(name)
  },
  { immediate: true }
)
</script>
```
