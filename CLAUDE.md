This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a **frontend learning roadmap documentation project** - a comprehensive collection of educational Markdown documents covering the complete frontend development stack. Pure documentation, no build system or dependencies.

## Module Structure

```
hello-frontend/
├── 01-html5/                    # HTML5: 基础语法、表单详解、语义化标签
├── 02-css3/                     # CSS3: 基础语法、布局相关、动画与过渡、响应式设计
├── 03-javascript/               # JavaScript: 基础语法、DOM与BOM、异步编程、高级特性
├── 04-es6+/                     # ES6+: 变量与作用域、函数进阶、对象与数组、异步编程、模块与元编程
├── 05-typescript/               # TypeScript: 快速入门、类型基础、接口与对象、函数类型、泛型编程等
├── 06-nodejs/                   # Node.js: Node.js基础、HTTP与Web开发、数据库集成、用户认证、实用技术
├── 07-engineering/              # 工程化: 包管理器、构建工具、代码质量、测试工具
├── 08-react/                    # React: 快速入门、核心概念、Hooks详解、组件模式、路由管理、状态管理、样式方案、性能优化、测试、实战项目
└── 09-vue/                      # Vue: 快速入门、核心概念、组件基础、Composition API、路由管理、状态管理、样式方案、性能优化、测试、实战项目
```

## Learning Path

Recommended progression:
1. **基础入门**: 01-html5 → 02-css3 → 03-javascript
2. **进阶提升**: 04-es6+ → 05-typescript
3. **后端拓展**: 06-nodejs
4. **工程化**: 07-engineering
5. **框架学习**: 08-react 或 09-vue

## Document Structure Pattern

Each chapter follows this template:
- 学习目标 (Learning objectives)
- 理论讲解 + 代码示例 (Theory + Code examples)
- 实践练习 (Practice exercises with full code)
- 常见问题 (FAQ)
- 学习资源 (Resources)
- Navigation links (上一篇/下一篇) at the bottom

## Development Notes

- Pure Markdown documentation (no package.json, no build required)
- Code examples use HTML/CSS/JavaScript with Chinese comments
- Documents link to each other using relative paths
- Recommended: VS Code + Live Server for preview
