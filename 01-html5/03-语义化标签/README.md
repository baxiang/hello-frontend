## 学习建议

**本章是理解现代 HTML 的关键**，建议：

1. **理解"为什么"** - 不仅仅是标签用法，更要理解语义化的意义
2. **对比学习** - 对比非语义化和语义化代码，体会差异
3. **关注无障碍** - 理解语义化对视障用户的重要性
4. **动手重构** - 将一个 div 堆砌的页面重构为语义化结构
