# 01-基础语法 ⭐

> HTML5 文档结构与常用标签，零基础入门

---

## 学习目标

- 理解 HTML 文档的基本结构
- 掌握常用文本标签的使用
- 理解块级元素和行内元素的区别
- 能写出一个完整的页面结构

## 学习建议

**本章是前端开发的起点**，建议：

1. **边学边写** - 每学一个标签就动手写代码
2. **用浏览器查看效果** - 即时反馈加深理解
3. **完成练习** - 手写一个个人简介页面验证学习成果

---

## 检验标准

完成本章后，你应该能够：

- [ ] 写出一个完整的 HTML 文档结构
- [ ] 正确使用 h1-h6 标题标签（不跳级）
- [ ] 区分块级元素和行内元素
- [ ] 为图片添加 alt 属性
- [ ] 实现页面内锚点跳转
