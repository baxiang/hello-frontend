# HTML5 基础语法

## 学习目标
- 理解 HTML 文档的基本结构
- 掌握常用文本标签的使用
- 理解块级元素和行内元素的区别

---

## 1.1 HTML 文档结构

### 标准文档模板

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- 字符编码 -->
    <meta charset="UTF-8">

    <!-- 视口设置（响应式必备） -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!-- 页面标题 -->
    <title>页面标题</title>

    <!-- SEO 相关 -->
    <meta name="description" content="页面描述">
    <meta name="keywords" content="关键词 1，关键词 2">
    <meta name="author" content="作者名">

    <!-- Open Graph（社交媒体分享） -->
    <meta property="og:title" content="标题">
    <meta property="og:image" content="image.jpg">

    <!-- 引入 CSS -->
    <link rel="stylesheet" href="style.css">

    <!-- Favicon -->
    <link rel="icon" href="favicon.ico">
</head>
<body>
    <!-- 页面内容 -->
    <script src="script.js"></script>
</body>
</html>
```

### DOCTYPE 声明

```html
<!-- HTML5 -->
<!DOCTYPE html>

<!-- HTML 4.01 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

<!-- XHTML 1.0 -->
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
```

### lang 属性

```html
<html lang="zh-CN">      <!-- 中文简体 -->
<html lang="zh-TW">      <!-- 中文繁体 -->
<html lang="en">         <!-- 英文 -->
<html lang="ja">         <!-- 日文 -->
<html lang="ko">         <!-- 韩文 -->
```

## 1.2 文本标签

### 标题标签

```html
<h1>主要标题（一个页面只有一个）</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

**要点：**
- h1 最重要的标题，影响 SEO
- 标题层级应该有序，不要跳级
- 搜索引擎会加权 h1 标签的内容

### 段落与换行

```html
<p>这是一个段落。</p>
<p>这是另一个段落。</p>

<p>这是第一行<br>这是第二行（使用 br 换行）</p>

<p>这是第一段</p>
<hr>  <!-- 水平分割线 -->
<p>这是第二段</p>
```

### 行内文本样式

```html
<!-- 语义化标签（推荐） -->
<strong>重要文本（浏览器通常加粗）</strong>
<em>强调文本（浏览器通常斜体）</em>
<mark>高亮文本</mark>
<small>小号文本</small>
<del>删除文本</del>
<ins>插入文本</ins>
<sub>下标</sub>
<sup>上标</sup>

<!-- 纯样式标签（不推荐） -->
<b>粗体</b>
<i>斜体</i>
<u>下划线</u>
```

### 引用和代码

```html
<!-- 短引用 -->
<p>正如孔子所说：<q>学而时习之，不亦说乎</q></p>

<!-- 长引用（块级） -->
<blockquote>
    <p>这是一段引用的内容，会独立成段并有缩进。</p>
    <cite>—— 出自某本书</cite>
</blockquote>

<!-- 代码相关 -->
<code>console.log('hello')</code>

<pre>
<!-- 预格式化文本，保留空格和换行 -->
function hello() {
    console.log('world');
}
</pre>

<kbd>Ctrl</kbd> + <kbd>C</kbd>  <!-- 键盘输入 -->
<var>x</var>  <!-- 变量 -->
<samp>程序输出</samp>
```

## 1.3 块级元素 vs 行内元素

### 块级元素（Block-level）

```html
<!-- 特点：独占一行，可设置宽高 -->
<div>        <!-- 通用容器 -->
<p>          <!-- 段落 -->
<h1>-<h6>    <!-- 标题 -->
<ul>         <!-- 无序列表 -->
<ol>         <!-- 有序列表 -->
<li>         <!-- 列表项 -->
<dl>         <!-- 定义列表 -->
<dt>         <!-- 术语 -->
<dd>         <!-- 描述 -->
<section>    <!-- 区域 -->
<article>    <!-- 文章 -->
<header>     <!-- 页眉 -->
<footer>     <!-- 页脚 -->
<nav>        <!-- 导航 -->
<aside>      <!-- 侧边栏 -->
<blockquote> <!-- 引用 -->
<pre>        <!-- 预格式化 -->
<hr>         <!-- 分割线 -->
<form>       <!-- 表单 -->
<table>      <!-- 表格 -->
```

### 行内元素（Inline）

```html
<!-- 特点：不独占一行，宽高设置无效 -->
<span>       <!-- 通用行内容器 -->
<a>          <!-- 链接 -->
<strong>     <!-- 重要 -->
<em>         <!-- 强调 -->
<img>        <!-- 图片 -->
<input>      <!-- 输入框 -->
<label>      <!-- 标签 -->
<code>       <!-- 代码 -->
<q>          <!-- 短引用 -->
<sup>        <!-- 上标 -->
<sub>        <!-- 下标 -->
```

### 行内块元素（Inline-block）

```html
<!-- 特点：行内排列，可设置宽高 -->
<img>
<input>
<button>
<textarea>
<select>
```

### 对比示例

```html
<style>
.box {
    width: 200px;
    height: 100px;
    background: lightblue;
    margin: 10px;
}
</style>

<!-- 块级元素：每个 div 独占一行 -->
<div class="box">1</div>
<div class="box">2</div>
<div class="box">3</div>

<!-- 行内元素：span 无法设置宽高 -->
<span class="box">1</span>
<span class="box">2</span>
<span class="box">3</span>

<!-- 行内块：可设置宽高且行内排列 -->
<span class="box" style="display: inline-block">1</span>
<span class="box" style="display: inline-block">2</span>
```

## 1.4 列表

### 无序列表

```html
<ul>
    <li>苹果</li>
    <li>香蕉</li>
    <li>橙子</li>
</ul>

<!-- 自定义样式 -->
<style>
ul {
    list-style-type: disc;    /* disc | circle | square | none */
    list-style-position: inside;  /* inside | outside */
}
</style>
```

### 有序列表

```html
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>

<!-- 自定义编号样式 -->
<style>
ol {
    list-style-type: decimal;        /* 1, 2, 3 */
    list-style-type: lower-alpha;    /* a, b, c */
    list-style-type: upper-roman;    /* I, II, III */

    /* 从指定数字开始 */
    start: 5;

    /* 倒序 */
    reversed;
}
</style>
```

### 定义列表

```html
<dl>
    <dt>HTML</dt>
    <dd>超文本标记语言，用于创建网页结构。</dd>

    <dt>CSS</dt>
    <dd>层叠样式表，用于控制网页样式。</dd>

    <dt>JavaScript</dt>
    <dd>编程语言，用于实现网页交互。</dd>
</dl>
```

### 嵌套列表

```html
<ul>
    <li>水果
        <ul>
            <li>热带水果
                <ul>
                    <li>芒果</li>
                    <li>菠萝</li>
                </ul>
            </li>
            <li>温带水果
                <ul>
                    <li>苹果</li>
                    <li>梨</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>蔬菜</li>
</ul>
```

## 1.5 链接（Anchor）

### 基础用法

```html
<a href="https://example.com">普通链接</a>

<!-- 新窗口打开 -->
<a href="https://example.com" target="_blank" rel="noopener noreferrer">新窗口打开</a>

<!-- 下载文件 -->
<a href="file.pdf" download>下载 PDF</a>
<a href="file.pdf" download="报告.pdf">指定文件名下载</a>
```

### 链接类型

```html
<!-- 外部链接 -->
<a href="https://example.com">外部网站</a>

<!-- 内部链接 -->
<a href="/about">关于页面</a>
<a href="/products/item.html">产品页面</a>

<!-- 锚点链接 -->
<a href="#section1">跳转到第一节</a>
<section id="section1">
    <h2>第一节</h2>
</section>

<!-- 邮件链接 -->
<a href="mailto:test@example.com">发送邮件</a>
<a href="mailto:test@example.com?subject=主题&body=内容">带主题和正文</a>

<!-- 电话链接 -->
<a href="tel:+8613800138000">拨打电话</a>

<!-- 短信链接 -->
<a href="sms:10086?body=短信内容">发送短信</a>
```

### 链接状态伪类

```css
/* 未访问 */
a:link {
    color: blue;
}

/* 已访问 */
a:visited {
    color: purple;
}

/* 悬停 */
a:hover {
    color: red;
    text-decoration: underline;
}

/* 激活（点击时） */
a:active {
    color: orange;
}

/* 聚焦 */
a:focus {
    outline: 2px solid blue;
}

/* 推荐写法顺序：LVHAF */
a:link,
a:visited,
a:hover,
a:active,
a:focus { }
```

## 1.6 图片

### 基础用法

```html
<img src="image.jpg" alt="图片描述" width="300" height="200">
```

### 重要属性

```html
<!-- src: 图片路径 -->
<img src="images/photo.jpg">
<img src="/assets/logo.png">
<img src="https://example.com/image.jpg">

<!-- alt: 替代文本（无障碍和 SEO） -->
<img src="chart.png" alt="2024 年 Q1-Q4 销售额对比柱状图">

<!-- 尺寸控制 -->
<img src="photo.jpg" width="300" height="200">
<img src="photo.jpg" style="width: 100%; height: auto;">

<!-- 懒加载 -->
<img src="placeholder.jpg" loading="lazy" data-src="real.jpg">

<!-- 响应式图片 -->
<img src="small.jpg"
     srcset="small.jpg 480w, medium.jpg 768w, large.jpg 1200w"
     sizes="(max-width: 600px) 100vw, (max-width: 1200px) 50vw, 25vw"
     alt="响应式图片">
```

### 图片格式对比

| 格式 | 特点 | 适用场景 |
|------|------|----------|
| JPG | 有损压缩，不支持透明 | 照片、复杂图像 |
| PNG | 无损压缩，支持透明 | Logo、图标、透明背景 |
| GIF | 支持动画，颜色有限 | 简单动画 |
| WebP | 高压缩率，支持透明和动画 | 现代浏览器首选 |
| SVG | 矢量图，无限缩放 | 图标、Logo、图表 |
| AVIF | 最新格式，压缩率最高 | 支持的设备 |

### 图片优化实践

```html
<!-- 使用 picture 元素提供多种格式 -->
<picture>
    <source srcset="image.avif" type="image/avif">
    <source srcset="image.webp" type="image/webp">
    <img src="image.jpg" alt="回退图片">
</picture>

<!-- 自适应分辨率 -->
<img src="image-1x.jpg"
     srcset="image-1x.jpg 1x, image-2x.jpg 2x, image-3x.jpg 3x"
     alt="高清图片">

<!-- 艺术指导 -->
<picture>
    <source media="(min-width: 800px)" srcset="large.jpg">
    <source media="(min-width: 400px)" srcset="medium.jpg">
    <img src="small.jpg" alt="自适应图片">
</picture>
```

### 图片性能优化

```html
<!-- 懒加载（原生支持） -->
<img src="image.jpg" loading="lazy" alt="懒加载图片">

<!-- 低质量占位图 -->
<img src="placeholder.jpg"
     data-src="high-quality.jpg"
     class="lazy-load"
     alt="图片">

<script>
// 使用 Intersection Observer 实现懒加载
const images = document.querySelectorAll('img[data-src]');
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            const img = entry.target;
            img.src = img.dataset.src;
            observer.unobserve(img);
        }
    });
});
images.forEach(img => observer.observe(img));
</script>
```

## 1.8 常见问题

### Q1: div 和 span 有什么区别？

**答：** div 是块级元素，默认独占一行；span 是行内元素，可以与其他行内元素并排。

### Q2: 为什么要用语义化标签？

**答：**
- 提高可访问性（屏幕阅读器）
- 改善 SEO（搜索引擎理解内容结构）
- 代码更易读和维护

### Q3: alt 属性重要吗？

**答：** 非常重要！alt 用于：
- 图片无法加载时显示
- 屏幕阅读器朗读给视障用户
- 搜索引擎理解图片内容

### Q4: 图片模糊怎么办？

**答：**
- 使用足够高的分辨率
- 考虑使用 WebP/AVIF 格式
- 为高清屏提供 2x/3x 图片
