## 🎯 学习路径建议

### 第一阶段：基础入门（2-4 周）
```
01-html5/ → 02-css3/ → 03-javascript/
```
**目标：** 掌握网页制作基础，能够制作静态页面

### 第二阶段：进阶提升（2-3 周）
```
04-es6/ → 05-typescript/
```
**目标：** 学习现代 JavaScript，理解类型系统

### 第三阶段：后端拓展（2-3 周）
```
06-nodejs/
```
**目标：** 掌握服务端开发，理解全栈开发流程

### 第四阶段：工程化与框架（3-4 周）
```
07-engineering/ → 08-frameworks/
```
**目标：** 学习工程化实践，掌握主流框架

## 📖 如何使用

1. **按顺序学习** - 建议按照目录顺序，循序渐进
2. **动手实践** - 每个章节都有代码示例，务必亲手练习
3. **完成练习** - 每章末尾有实践练习，巩固所学知识
4. **查阅文档** - 遇到疑问及时查阅官方文档

## 🛠️ 开发环境准备

### 必备工具
- **编辑器**: [VS Code](https://code.visualstudio.com/)
- **浏览器**: [Chrome](https://www.google.com/chrome/) (安装 DevTools)
- **Node.js**: [下载地址](https://nodejs.org/) (建议 LTS 版本)
- **Git**: [下载地址](https://git-scm.com/)

### VS Code 推荐插件
- Live Server - 本地开发服务器
- Prettier - 代码格式化
- ESLint - 代码检查
- Auto Rename Tag - 自动重命名标签
- CSS Peek - CSS 快速查看

## 📝 学习建议

### ✅ 应该做的
- 多动手写代码，不要只看不练
- 善用浏览器开发者工具调试
- 阅读官方文档获取第一手资料
- 参与开源项目，积累实战经验
- 写技术博客，总结学习心得

### ❌ 应该避免的
- 不要只收藏不学习
- 不要盲目追求新框架，基础最重要
- 不要 copy 代码，要理解原理
- 不要闭门造车，多交流多提问
