## 2.1 安装 Node.js

### 什么是 Node.js

Node.js 是一个 JavaScript 运行环境，让我们可以在服务器端运行 JavaScript。

TypeScript 需要 Node.js 来安装和运行。

### 检查是否已安装

```bash
# 打开终端，输入以下命令
node --version

# 如果显示版本号，说明已安装
# 例如：v20.10.0
```

### 下载安装

如果没有安装，访问官网下载：https://nodejs.org/

```
推荐下载 LTS（长期支持）版本
```

### 使用 nvm 管理 Node.js（可选）

如果你需要同时使用多个 Node.js 版本，推荐使用 nvm：

```bash
# macOS / Linux
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Windows
# 下载：https://github.com/coreybutler/nvm-windows/releases
```

## 2.2 安装 TypeScript

### 全局安装

```bash
# 在终端中运行
npm install -g typescript

# 验证安装
tsc --version

# 应该显示：Version 5.x.x
```

### 项目内安装

```bash
# 创建项目文件夹
mkdir my-ts-project
cd my-ts-project

# 初始化项目
npm init -y

# 安装 TypeScript
npm install --save-dev typescript

# 验证安装
npx tsc --version
```

### npm 简介

| 命令 | 说明 |
|------|------|
| `npm install -g` | 全局安装，所有项目可用 |
| `npm install` | 安装到当前项目 |
| `--save-dev` / `-D` | 保存为开发依赖 |
| `npx` | 运行项目的局部命令 |

## 2.3 第一个 TypeScript 程序

### 创建项目

```bash
# 创建项目目录
mkdir hello-typescript
cd hello-typescript

# 初始化
npm init -y

# 安装 TypeScript
npm install --save-dev typescript
```

### 编写代码

创建文件 `src/index.ts`：

```typescript
// src/index.ts
const message: string = 'Hello, TypeScript!';
console.log(message);
```

### 编译运行

```bash
# 编译 TypeScript
npx tsc src/index.ts

# 运行编译后的 JavaScript
node src/index.js

# 输出：Hello, TypeScript!
```

### 目录结构

```
hello-typescript/
├── src/
│   ├── index.ts      # TypeScript 源码
│   └── index.js      # 编译后的 JavaScript
├── package.json
└── node_modules/
```

## 2.4 使用 ts-node 直接运行

### 安装 ts-node

```bash
npm install --save-dev ts-node
```

### 直接运行 .ts 文件

```bash
# 不需要手动编译，直接运行
npx ts-node src/index.ts

# 输出：Hello, TypeScript!
```

### 常用命令总结

| 命令 | 说明 |
|------|------|
| `tsc file.ts` | 编译单个文件 |
| `tsc --init` | 初始化 tsconfig.json |
| `tsc` | 编译整个项目 |
| `ts-node file.ts` | 直接运行 TypeScript 文件 |

## 2.5 VS Code 配置

### 安装 VS Code

访问 https://code.visualstudio.com/ 下载安装。

### TypeScript 支持

VS Code 内置了 TypeScript 支持，安装后自动启用。

### 推荐配置

打开 VS Code 设置（`Cmd + ,` 或 `Ctrl + ,`），添加以下配置：

```json
{
    // 保存时自动格式化
    "editor.formatOnSave": true,
    
    // 保存时自动修复
    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": "explicit"
    },
    
    // 使用单引号
    "javascript.preferences.quoteStyle": "single",
    "typescript.preferences.quoteStyle": "single",
    
    // Tab 改为 2 个空格
    "editor.tabSize": 2
}
```

### 常用快捷键

| 快捷键 | 说明 |
|--------|------|
| `Cmd + Shift + P` | 打开命令面板 |
| `F2` | 重命名符号 |
| `F12` | 跳转到定义 |
| `Cmd + 点击` | 跳转到定义 |
| `Cmd + 悬停` | 查看类型信息 |

### 插件推荐

| 插件 | 说明 |
|------|------|
| ESLint | 代码检查 |
| Prettier | 代码格式化 |
| Path Autocomplete | 路径自动补全 |

## 2.6 实践：搭建开发环境

### 步骤 1：创建项目

```bash
mkdir ts-project
cd ts-project
npm init -y
npm install --save-dev typescript ts-node @types/node
```

### 步骤 2：创建 tsconfig.json

```bash
npx tsc --init
```

修改 `tsconfig.json`：

```json
{
    "compilerOptions": {
        "target": "ES2022",
        "module": "ESNext",
        "strict": true,
        "esModuleInterop": true,
        "skipLibCheck": true,
        "forceConsistentCasingInFileNames": true,
        "outDir": "./dist",
        "rootDir": "./src"
    },
    "include": ["src/**/*"],
    "exclude": ["node_modules", "dist"]
}
```

### 步骤 3：编写代码

创建 `src/index.ts`：

```typescript
// 问候函数
function greet(name: string): string {
    return `Hello, ${name}!`;
}

// 输出
const result = greet('TypeScript');
console.log(result);
```

### 步骤 4：运行

```bash
# 方式一：编译后运行
npx tsc
node dist/index.js

# 方式二：直接运行
npx ts-node src/index.ts
```



