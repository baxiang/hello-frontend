# 类型注解初体验 ⭐

> 类型注解语法、类型推断、编译错误

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
let name: string = 'Tom';
let age: number = 25;
let arr: number[] = [1, 2, 3];
```

**详细示例：**

```typescript
// 基础类型
let message: string = 'Hello';
let count: number = 42;
let isActive: boolean = true;

// 数组
let numbers: number[] = [1, 2, 3];
let names: Array<string> = ['Tom', 'Jerry'];

// 对象
let user: { name: string; age: number } = { name: 'Tom', age: 25 };

// 函数
function add(a: number, b: number): number { return a + b; }

// 类型推断（省略标注）
let inferred = 'auto string';  // TS 自动推断为 string
```

## L2 实践层

### 常见编译错误

| 错误 | 原因 | 修复 |
|------|------|------|
| `Type 'string' is not assignable to type 'number'` | 类型不匹配 | 检查赋值类型 |
| `Property 'x' does not exist on type` | 访问不存在属性 | 检查类型定义 |

### 反模式

```typescript
// ❌ 错误：类型不匹配
let age: number = '25';

// ✅ 正确
let age: number = 25;
```
