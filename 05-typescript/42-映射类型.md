## 15.0 通俗理解映射类型

### 什么是映射类型？

```
映射类型 = 批量处理对象的属性
        = 循环遍历属性并转换

就像：
- 有一个清单（键的列表）
- 对每个项目做同样的操作
```

### 通俗比喻

```
场景：把所有人的名字都改成大写

不用映射类型：
interface User1 {
    name: string;
    nickname: string;
    email: string;
}

interface User2 {
    name: string;      // 手动写
    nickname: string;  // 手动写
    email: string;     // 手动写
}

用映射类型：
type UppercaseKeys<T> = {
    [P in keyof T]: string;  // 把所有属性变成 string
};
```

## 15.1 映射类型基础

### 语法

```typescript
{ [P in K]: T }
```

**解读：**
- `P` - 属性名（自己取的变量名）
- `K` - 键的联合类型
- `T` - 值的类型

### 简单例子

```typescript
// 定义要处理的键
type Keys = 'name' | 'age';

// 映射：把每个键映射到 string 类型
type Mapped = {
    [P in Keys]: string;
};

// 结果：
// {
//     name: string;
//     age: string;
// }
```

### 用 keyof 遍历所有属性

```typescript
interface User {
    id: number;
    name: string;
    email: string;
}

// keyof User = 'id' | 'name' | 'email'
// 遍历每个属性，类型保持不变

type MappedUser = {
    [P in keyof User]: User[P];
};

// 结果：
// {
//     id: number;
//     name: string;
//     email: string;
// }
```

## 15.2 映射修饰符

### readonly - 添加只读

```typescript
interface User {
    name: string;
    age: number;
}

// 添加 readonly
type Readonly<T> = {
    readonly [P in keyof T]: T[P];
};

type ReadonlyUser = Readonly<User>;

// 结果：
// {
//     readonly name: string;
//     readonly age: number;
// }
```

### ? - 添加可选

```typescript
interface User {
    name: string;
    age: number;
}

// 添加 ?
type Optional<T> = {
    [P in keyof T]?: T[P];
};

type OptionalUser = Optional<User>;

// 结果：
// {
//     name?: string;
//     age?: number;
// }
```

### - 移除修饰符

```typescript
interface User {
    readonly name?: string;
}

// 移除 readonly 和 ?
type Mutable<T> = {
    -readonly [P in keyof T]-?: T[P];
};

type MutableUser = Mutable<User>;

// 结果：
// {
//     name: string;
// }
```

## 15.3 键的重映射

### as 子句

```
as = "重新映射" = 改变键名
```

### 例子：把键名改成大写

```typescript
interface User {
    id: number;
    name: string;
    email: string;
}

// 把所有键变成大写
type UppercaseKeys<T> = {
    [P in keyof T as Uppercase<string & P>]: T[P];
};

type UppercaseUser = UppercaseKeys<User>;

// 结果：
// {
//     ID: number;
//     NAME: string;
//     EMAIL: string;
// }
```

### 例子：添加前缀

```typescript
interface Config {
    port: number;
    host: string;
}

// 给键添加前缀
type PrefixedConfig<T> = {
    [P in keyof T as `get${Capitalize<string & P>}`]: T[P];
};

type Prefixed = PrefixedConfig<Config>;

// 结果：
// {
//     getPort: number;
//     getHost: string;
// }
```

### 例子：过滤属性

```typescript
interface User {
    id: number;
    name: string;
    password: string;
    email: string;
}

// 排除 password
type SafeUser<T> = {
    [P in keyof T as P extends 'password' ? never : P]: T[P];
};

type Safe = SafeUser<User>;

// 结果：
// {
//     id: number;
//     name: string;
//     email: string;
// }
```

## 15.4 模板字面量类型

### 什么是模板字面量？

```
模板字面量 = 用反引号 `` 拼接字符串类型
```

### 例子

```typescript
// 基础
type Greeting = `Hello, ${string}`;
// 只能是 "Hello, xxx"

type Method = 'get' | 'post' | 'put' | 'delete';

// 组合
type API = `${Method}/${string}`;
// 'get/user' | 'post/user' | 'put/user' | 'delete/user'
```

### 在映射类型中使用

```typescript
interface Handler {
    onClick: () => void;
    onChange: () => void;
    onSubmit: () => void;
}

// 把 onXxx 变成 handleXxx
type HandlerToHandle<T> = {
    [P in keyof T as P extends `on${infer R}` ? `handle${R}` : P]: T[P];
};

type Result = HandlerToHandle<Handler>;

// 结果：
// {
//     handleClick: () => void;
//     handleChange: () => void;
//     handleSubmit: () => void;
// }
```

## 15.5 完整示例

```typescript
// 综合示例：实现 Partial

// 1. 基础版
type MyPartial<T> = {
    [P in keyof T]?: T[P];
};

// 2. 深度版
type DeepPartial<T> = {
    [P in keyof T]?: T[P] extends object ? DeepPartial<T[P]> : T[P];
};

// 测试
interface Company {
    name: string;
    address: {
        city: string;
        street: string;
    };
}

type PartialCompany = DeepPartial<Company>;

// 结果：
// {
//     name?: string;
//     address?: {
//         city?: string;
//         street?: string;
//     };
// }
```

## 本章小结

| 概念 | 语法 | 说明 |
|------|------|------|
| 映射类型 | `[P in K]: T` | 批量处理属性 |
| keyof | 获取所有键 | `'a' \| 'b'` |
| readonly | 添加只读 | `readonly [P in K]` |
| ? | 添加可选 | `[P in K]?` |
| as | 键重映射 | 改变键名 |
| 模板字面量 | `` `${string}` `` | 拼接字符串 |
