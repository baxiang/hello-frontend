## 15.1 映射类型基础

### 语法

```typescript
{ [P in K]: T }
```

- `P`：属性名（映射变量）
- `K`：键的联合类型
- `T`：属性类型

### 示例

```typescript
type Keys = 'name' | 'age';

type Mapped = {
    [P in Keys]: string;
};

// 结果：{ name: string; age: string; }
```

## 15.2 使用 keyof

### 基本用法

```typescript
interface User {
    id: number;
    name: string;
    email: string;
}

type UserKeys = keyof User;  // 'id' | 'name' | 'email'

type MappedUser = {
    [P in keyof User]: User[P];
};

// 结果：{ id: number; name: string; email: string; }
```

## 15.3 映射修饰符

### readonly

```typescript
// 添加 readonly
type Readonly<T> = {
    readonly [P in keyof T]: T[P];
};

// 移除 readonly
type Mutable<T> = {
    -readonly [P in keyof T]: T[P];
};
```

### ?

```typescript
// 添加 ?
type Optional<T> = {
    [P in keyof T]?: T[P];
};

// 移除 ?
type Required<T> = {
    [P in keyof T]-?: T[P];
};
```

## 15.4 键的重映射

### as 子句

```typescript
interface User {
    id: number;
    name: string;
    email: string;
}

// 将所有键转为大写
type UppercaseKeys<T> = {
    [P in keyof T as Uppercase<string & P>]: T[P];
};

type UppercaseUser = UppercaseKeys<User>;
// { ID: number; NAME: string; EMAIL: string; }
```

### 实际应用

```typescript
// 移除可选属性
type RemoveOptional<T> = {
    [P in keyof T as {} extends T[P] ? never : P]: T[P];
};

interface PartialUser {
    id?: number;
    name?: string;
    age: number;
}

type RequiredUser = RemoveOptional<PartialUser>;
// { age: number }
```

## 15.5 模板字面量类型

### 基本用法

```typescript
type EventName = 'click' | 'focus' | 'blur';
type Handler = `on${Capitalize<EventName>}`;
// 'onClick' | 'onFocus' | 'onBlur'
```

### 组合多个类型

```typescript
type Path = 'users' | 'posts';
type Method = 'get' | 'post' | 'put' | 'delete';
type API = `${Method}/${Path}`;
// 'get/users' | 'get/posts' | 'post/users' | ...
```

## 15.6 实践练习

### 练习 1：实现 Partial

```typescript
type MyPartial<T> = {
    [P in keyof T]?: T[P];
};
```

### 练习 2：实现 Pick

```typescript
type MyPick<T, K extends keyof T> = {
    [P in K]: T[P];
};
```

### 练习 3：键重映射

```typescript
// 添加前缀
type PrefixKeys<T, P extends string> = {
    [K in keyof T as `${P}${K & string}`]: T[K];
};

interface Config {
    port: number;
    host: string;
}

type PrefixedConfig = PrefixKeys<Config, 'app_'>;
// { app_port: number; app_host: string; }
```

## 本章小结

| 概念 | 语法 | 说明 |
|------|------|------|
| 映射类型 | `[P in K]: T` | 遍历键 |
| keyof | 获取键的联合类型 | `keyof T` |
| readonly | 只读修饰符 | 添加/移除 |
| ? | 可选修饰符 | 添加/移除 |
| as | 键重映射 | 改变键名 |
| 模板字面量 | `` `${Method}/${Path}` `` | 组合类型 |
