# 类型守卫与类型收窄 ⭐⭐

> typeof、instanceof、in、类型谓词

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
function process(x: string | number) {
    if (typeof x === 'string') x.toUpperCase();
    else x.toFixed(2);
}
```

**详细示例：**

```typescript
// typeof
function pad(value: string | number, width: number) {
    if (typeof value === 'string') return value.padStart(width);
    return String(value).padStart(width);
}

// instanceof
class Dog { bark() { return 'woof'; } }
class Cat { meow() { return 'meow'; } }
function sound(animal: Dog | Cat) {
    if (animal instanceof Dog) return animal.bark();
    return animal.meow();
}

// in
interface Fish { swim(): void }
interface Bird { fly(): void }
function move(animal: Fish | Bird) {
    if ('swim' in animal) animal.swim();
    else animal.fly();
}

// 类型谓词
interface User { name: string }
function isUser(x: unknown): x is User {
    return typeof x === 'object' && x !== null && 'name' in x;
}
```

## L2 实践层

### 类型守卫速查

| 守卫 | 适用 | 示例 |
|------|------|------|
| `typeof` | 基础类型 | `typeof x === 'string'` |
| `instanceof` | 类实例 | `x instanceof Dog` |
| `in` | 属性检查 | `'name' in x` |
| 类型谓词 | 自定义 | `function isUser(x): x is User` |

---

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **类型守卫** | 运行时检查类型的表达式 | `typeof x === 'string'` |
| **类型收窄** | 缩小变量的可能类型范围 | `if (typeof x === 'string')` |
| **类型谓词** | 自定义类型守卫的返回类型 | `x is User` |
