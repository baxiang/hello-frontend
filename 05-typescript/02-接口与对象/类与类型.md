## L1 理解层

**最简示例（1-3行）：**

```typescript
class User { constructor(public name: string) {} greet() { return `Hi, ${this.name}`; } }
const user = new User('Tom'); user.greet();
```

**详细示例：**

```typescript
class User {
    public name: string;
    private age: number;
    readonly id: number;

    constructor(id: number, name: string, age: number) {
        this.id = id;
        this.name = name;
        this.age = age;
    }

    greet(): string { return `Hi, ${this.name}`; }
}

// 简写：参数属性
class User2 {
    constructor(
        public id: number,
        public name: string,
        private age: number
    ) {}
}

// 抽象类
abstract class Shape {
    abstract area(): number;
    toString() { return `Shape: ${this.area()}`; }
}

class Circle extends Shape {
    constructor(private radius: number) { super(); }
    area() { return Math.PI * this.radius ** 2; }
}

// 实现接口
interface Printable { print(): void }
class Report implements Printable {
    print() { console.log('Printing report'); }
}
```

## L2 实践层

### 访问修饰符

| 修饰符 | 类内 | 子类 | 外部 |
|--------|:---:|:---:|:---:|
| `public` | ✅ | ✅ | ✅ |
| `protected` | ✅ | ✅ | ❌ |
| `private` | ✅ | ❌ | ❌ |

---

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **访问修饰符** | 控制属性/方法可见性 | public/protected/private |
| **抽象类** | 不能被实例化的基类 | `abstract class Shape` |
| **implements** | 类实现接口 | `class A implements B` |
