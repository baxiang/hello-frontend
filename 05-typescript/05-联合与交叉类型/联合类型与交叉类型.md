# 联合类型与交叉类型 ⭐⭐

> 联合、交叉、可辨识联合

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
type ID = string | number;              // 联合：或
type Admin = User & { role: 'admin' };  // 交叉：且
```

**详细示例：**

```typescript
// 联合类型 — 多个类型之一
type Status = 'pending' | 'success' | 'error';
type Result = { data: string } | { error: string };

// 交叉类型 — 合并多个类型
type HasId = { id: number };
type HasName = { name: string };
type User = HasId & HasName;  // { id: number; name: string }

// 可辨识联合
interface Circle { kind: 'circle'; radius: number }
interface Square { kind: 'square'; side: number }
type Shape = Circle | Square;

function area(shape: Shape): number {
    switch (shape.kind) {
        case 'circle': return Math.PI * shape.radius ** 2;
        case 'square': return shape.side ** 2;
    }
}
```

---

## L2 实践层

### 联合 vs 交叉

| 操作符 | 含义 | 结果 |
|--------|------|------|
| `\|` | 联合（或） | 满足任一类型 |
| `&` | 交叉（且） | 满足所有类型 |

---

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **联合类型** | 多个类型之一 | `string \| number` |
| **交叉类型** | 合并所有类型 | `A & B` |
| **可辨识联合** | 通过共同字段区分的联合 | `switch (shape.kind)` |
| **字面量类型** | 具体值作为类型 | `'success' \| 'error'` |
