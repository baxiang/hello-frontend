## 5.1 type 基础用法

### 什么是类型别名？

```
类型别名（type alias）= 给类型起个"昵称"

作用：
- 简化复杂类型的书写
- 让类型更有意义
- 方便复用

就像：
- 给长名字起外号
- 给复杂地址起简称
```

### 基本用法

```typescript
// 基础类型别名
type Name = string;
type Age = number;
type IsDone = boolean;

// 使用
const name: Name = '张三';
const age: Age = 25;
const isDone: IsDone = true;

/** 用户 ID 类型 */
type UserID = string | number;
```

### 对象类型别名

```typescript
// 对象类型
type User = {
    id: number;
    name: string;
    email: string;
};

const user: User = {
    id: 1,
    name: '张三',
    email: 'zhangsan@example.com'
};

// 嵌套对象
type Address = {
    street: string;
    city: string;
    zipcode: string;
    country?: string;
};

type Person = {
    name: string;
    age: number;
    address: Address;
};
```

## 5.2 联合类型

### 什么是联合类型？

```
联合类型 = "或"关系 = 多种类型之一

语法：类型1 | 类型2 | 类型3

就像：
- 成绩：优秀 | 良好 | 及格 | 不及格
- 状态：pending | success | error
- 性别：male | female | other
```

### 基本用法

```typescript
// 字符串联合类型
type Status = 'pending' | 'success' | 'error';

const status1: Status = 'pending';
const status2: Status = 'success';
// const status3: Status = 'unknown';  // ❌ 错误

// 数字联合类型
type ResultCode = 200 | 201 | 400 | 401 | 404 | 500;

const code: ResultCode = 200;

// 混合联合类型
type ID = string | number;

const userId: ID = 'abc123';
const postId: ID = 12345;
```

### 联合类型数组

```typescript
let arr: (string | number)[] = [1, 'a', 2, 'b'];
```

### 联合类型的实际应用

```typescript
// 1. API 响应
interface ApiResponse {
    status: 'loading' | 'success' | 'error';
    data?: any;
    error?: string;
}

// 2. 配置值
type Theme = 'light' | 'dark' | 'auto';
const theme: Theme = 'dark';

// 3. 回调类型
type Callback = (error: Error | null, result?: any) => void;
```

## 5.3 可辨识联合

### 什么是可辨识联合？

```
可辨识联合 = 用一个"标签"区分不同类型

核心思想：
- 每个类型都有一个共同的"标识"属性（通常是 kind 或 type）
- 通过检查这个标识来区分类型

优点：
- 代码更清晰
- 易于扩展
- 配合 switch 语句更优雅
```

### 基本用法

```typescript
interface Square {
    kind: 'square';
    size: number;
}

interface Circle {
    kind: 'circle';
    radius: number;
}

interface Triangle {
    kind: 'triangle';
    base: number;
    height: number;
}

type Shape = Square | Circle | Triangle;

function getArea(shape: Shape): number {
    switch (shape.kind) {
        case 'square':
            return shape.size * shape.size;
        case 'circle':
            return Math.PI * shape.radius * shape.radius;
        case 'triangle':
            return 0.5 * shape.base * shape.height;
    }
}
```

### 实际应用：状态管理

```typescript
interface LoadingState {
    status: 'loading';
}

interface SuccessState {
    status: 'success';
    data: string[];
}

interface ErrorState {
    status: 'error';
    error: Error;
}

type AsyncState = LoadingState | SuccessState | ErrorState;

function handleState(state: AsyncState): void {
    switch (state.status) {
        case 'loading':
            console.log('Loading...');
            break;
        case 'success':
            console.log('Data:', state.data);
            break;
        case 'error':
            console.log('Error:', state.error);
            break;
    }
}
```

## 5.4 交叉类型

### 什么是交叉类型？

```
交叉类型 = "与"关系 = 多种类型都要满足

语法：类型1 & 类型2 & 类型3

就像：
- 既是学生又是班干部
- 既是教师又是父亲
- 既有优点又有缺点
```

### 基本用法

```typescript
type Person = {
    name: string;
    age: number;
};

type Employee = {
    company: string;
    salary: number;
};

// 交叉类型：既是 Person 又是 Employee
type PersonEmployee = Person & Employee;

const person: PersonEmployee = {
    name: '张三',
    age: 30,
    company: 'ABC公司',
    salary: 20000
};
```

### 实际应用：混入

```typescript
interface CanRun {
    run(): void;
}

interface CanSwim {
    swim(): void;
}

type Amphibian = CanRun & CanSwim;

class Frog implements Amphibian {
    run(): void {
        console.log('Running...');
    }
    swim(): void {
        console.log('Swimming...');
    }
}
```

## 5.5 联合 vs 交叉 对比

| 类型 | 符号 | 含义 | 记忆方法 |
|------|------|------|----------|
| 联合 | `|` | 或（满足其一） | Union = 联合（合并） |
| 交叉 | `&` | 且（满足全部） | Intersection = 交叉（重叠） |

### 图示

```
联合类型 A | B：
┌───────┐     ┌───────┐
│   A   │ OR  │   B   │
└───────┘     └───────┘
   结果：可以是 A 或 B

交叉类型 A & B：
┌───────┐     ┌───────┐
│   A   │ AND │   B   │
└───────┘     └───────┘
   结果：同时是 A 和 B
```

### 示例对比

```typescript
// 联合：可以是 string 或 number
type StringOrNumber = string | number;
let value: StringOrNumber = 'hello';  // ✅
value = 123;                           // ✅

// 交叉：同时是 { a: string } 和 { b: number }
type Both = { a: string } & { b: number };
let obj: Both = { a: 'hello', b: 123 };  // ✅
```

## 5.6 实践练习

### 练习 1：联合类型

```typescript
// 参数可以是字符串或数字
function format(value: string | number): string {
    if (typeof value === 'string') {
        return value.toUpperCase();
    }
    return value.toFixed(2);
}

format('hello');  // 'HELLO'
format(123.456);  // '123.46'
```

### 练习 2：可辨识联合

```typescript
interface Circle {
    kind: 'circle';
    radius: number;
}

interface Rectangle {
    kind: 'rectangle';
    width: number;
    height: number;
}

interface Triangle {
    kind: 'triangle';
    base: number;
    height: number;
}

type Shape = Circle | Rectangle | Triangle;

function getArea(shape: Shape): number {
    switch (shape.kind) {
        case 'circle':
            return Math.PI * shape.radius ** 2;
        case 'rectangle':
            return shape.width * shape.height;
        case 'triangle':
            return 0.5 * shape.base * shape.height;
    }
}
```

### 练习 3：交叉类型

```typescript
interface Loggable {
    log(): void;
}

interface Serializable {
    serialize(): string;
}

type LoggableAndSerializable = Loggable & Serializable;

class User implements LoggableAndSerializable {
    name: string;

    constructor(name: string) {
        this.name = name;
    }

    log(): void {
        console.log(`User: ${this.name}`);
    }

    serialize(): string {
        return JSON.stringify({ name: this.name });
    }
}
```

## 本章小结

| 概念 | 符号 | 说明 |
|------|------|------|
| 联合类型 | `|` | 值可以是多种类型之一 |
| 可辨识联合 | 通过 kind 属性区分 | 安全的联合类型处理 |
| 交叉类型 | `&` | 值同时具备多种类型 |
