# 工具类型详解 ⭐⭐

> Partial、Required、Pick、Omit、Record、ReturnType

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
type PartialUser = Partial<User>;
type PickName = Pick<User, 'name'>;
type OmitId = Omit<User, 'id'>;
```

**详细示例：**

```typescript
interface User { id: number; name: string; email: string; age: number }

// 使所有属性可选
type PartialUser = Partial<User>;  // { id?: number; name?: string; ... }

// 使所有属性必填
type RequiredUser = Required<PartialUser>;

// 使所有属性只读
type ReadonlyUser = Readonly<User>;

// 挑选属性
type UserName = Pick<User, 'name' | 'email'>;  // { name: string; email: string }

// 排除属性
type UserWithoutId = Omit<User, 'id'>;  // { name: string; email: string; age: number }

// 键值映射
type StringRecord = Record<string, number>;  // { [key: string]: number }

// 函数类型工具
type Fn = (a: number, b: number) => string;
type Params = Parameters<Fn>;     // [number, number]
type Ret = ReturnType<Fn>;        // string
```

## L2 实践层

### 工具类型速查

| 工具类型 | 说明 | 示例 |
|----------|------|------|
| `Partial<T>` | 所有属性可选 | `Partial<User>` |
| `Required<T>` | 所有属性必填 | `Required<Partial<User>>` |
| `Readonly<T>` | 所有属性只读 | `Readonly<User>` |
| `Pick<T, K>` | 挑选属性 | `Pick<User, 'name'>` |
| `Omit<T, K>` | 排除属性 | `Omit<User, 'id'>` |
| `Record<K, V>` | 键值映射 | `Record<string, number>` |
| `ReturnType<T>` | 函数返回值类型 | `ReturnType<() => string>` |
