## 8.1 类基础

### 定义类

```typescript
class User {
    name: string;
    age: number;
    
    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }
    
    greet(): string {
        return `Hello, I'm ${this.name}`;
    }
}

const user = new User('Tom', 25);
console.log(user.greet());  // "Hello, I'm Tom"
```

### 类作为类型

```typescript
class User {
    name: string;
    age: number;
}

const user: User = {
    name: 'Tom',
    age: 25
};
```

## 8.2 访问修饰符

### public - 公开

默认修饰符，类内部和外部都可以访问：

```typescript
class User {
    public name: string;
    
    constructor(name: string) {
        this.name = name;
    }
}

const user = new User('Tom');
console.log(user.name);  // ✅ 'Tom'
```

### private - 私有

只能在类内部访问：

```typescript
class User {
    private age: number;
    
    constructor(age: number) {
        this.age = age;
    }
    
    getAge(): number {
        return this.age;  // ✅ 类内部可以访问
    }
}

const user = new User(25);
console.log(user.getAge());  // ✅ 25
// user.age  // ❌ 错误：私有属性无法访问
```

### protected - 受保护

类内部和子类可以访问：

```typescript
class Animal {
    protected name: string;
    
    constructor(name: string) {
        this.name = name;
    }
}

class Dog extends Animal {
    bark(): void {
        console.log(`${this.name} is barking`);  // ✅ 子类可以访问
    }
}

const dog = new Dog('Buddy');
// dog.name  // ❌ 错误：受保护属性无法访问
```

## 8.3 readonly 修饰符

### 只读属性

```typescript
class User {
    readonly id: number;
    name: string;
    
    constructor(id: number, name: string) {
        this.id = id;
        this.name = name;
    }
}

const user = new User(1, 'Tom');
user.id = 2;    // ❌ 错误：只读属性不能修改
user.name = 'Jerry';  // ✅
```

## 8.5 抽象类

### 什么是抽象类

抽象类不能直接实例化，只能被继承：

```typescript
abstract class Animal {
    abstract makeSound(): void;  // 抽象方法
    
    move(): void {
        console.log('Moving...');
    }
}

class Dog extends Animal {
    makeSound(): void {
        console.log('Woof!');
    }
}

const dog = new Dog();  // ✅
const animal = new Animal();  // ❌ 错误：抽象类不能实例化
```

### 实际应用

```typescript
abstract class Shape {
    abstract area(): number;
    abstract perimeter(): number;
}

class Circle extends Shape {
    constructor(private radius: number) {
        super();
    }
    
    area(): number {
        return Math.PI * this.radius ** 2;
    }
    
    perimeter(): number {
        return 2 * Math.PI * this.radius;
    }
}

class Rectangle extends Shape {
    constructor(private width: number, private height: number) {
        super();
    }
    
    area(): number {
        return this.width * this.height;
    }
    
    perimeter(): number {
        return 2 * (this.width + this.height);
    }
}
```

## 8.6 类的继承

### extends 关键字

```typescript
class Animal {
    name: string;
    
    constructor(name: string) {
        this.name = name;
    }
    
    eat(): void {
        console.log(`${this.name} is eating`);
    }
}

class Dog extends Animal {
    breed: string;
    
    constructor(name: string, breed: string) {
        super(name);
        this.breed = breed;
    }
    
    bark(): void {
        console.log(`${this.name} is barking`);
    }
}

const dog = new Dog('Buddy', 'Golden Retriever');
dog.eat();   // "Buddy is eating"
dog.bark();  // "Buddy is barking"
```

### 方法重写

```typescript
class Animal {
    speak(): string {
        return 'Some sound';
    }
}

class Dog extends Animal {
    speak(): string {
        return 'Woof!';
    }
}

const dog = new Dog();
console.log(dog.speak());  // "Woof!"
```

## 8.7 静态成员

### static 关键字

```typescript
class MathHelper {
    static PI: number = 3.14159;
    
    static add(a: number, b: number): number {
        return a + b;
    }
}

console.log(MathHelper.PI);  // 3.14159
console.log(MathHelper.add(1, 2));  // 3
```

## 8.8 实践练习

### 练习 1：用户类

```typescript
class User {
    constructor(
        public readonly id: number,
        public name: string,
        private _age: number,
        protected email: string
    ) {}
    
    get age(): number {
        return this._age;
    }
    
    set age(value: number) {
        if (value > 0) {
            this._age = value;
        }
    }
}

const user = new User(1, 'Tom', 25, 'tom@example.com');
console.log(user.name);    // "Tom"
console.log(user.age);     // 25
user.age = 30;
console.log(user.age);     // 30
```

### 练习 2：抽象类

```typescript
abstract class Logger {
    abstract log(message: string): void;
    
    format(message: string): string {
        return `[${new Date().toISOString()}] ${message}`;
    }
}

class ConsoleLogger extends Logger {
    log(message: string): void {
        console.log(this.format(message));
    }
}

const logger = new ConsoleLogger();
logger.log('Hello');  // "[2024-01-01T00:00:00.000Z] Hello"
```

### 练习 3：继承

```typescript
class Person {
    constructor(
        public name: string,
        public age: number
    ) {}
    
    introduce(): string {
        return `I'm ${this.name}, ${this.age} years old`;
    }
}

class Student extends Person {
    constructor(
        name: string,
        age: number,
        public grade: string
    ) {
        super(name, age);
    }
    
    introduce(): string {
        return `${super.introduce()}, I'm in grade ${this.grade}`;
    }
}

const student = new Student('Tom', 15, '10th');
console.log(student.introduce());
// "I'm Tom, 15 years old, I'm in grade 10th"
```

## 本章小结

| 修饰符 | 访问范围 | 关键字 |
|--------|----------|--------|
| public | 任何地方 | 默认 |
| private | 类内部 | private |
| protected | 类和子类 | protected |
| readonly | 只读 | readonly |

| 概念 | 说明 |
|------|------|
| 抽象类 | 不能实例化，只能被继承 |
| 继承 | extends 关键字 |
| 静态成员 | static 关键字，类级别 |
| getter/setter | 访问器属性 |
