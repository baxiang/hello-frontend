# 泛型编程 ⭐⭐⭐

> 泛型函数、接口、约束、keyof

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
function identity<T>(x: T): T { return x; }
identity(5);       // T = number
identity('hello'); // T = string
```

**详细示例：**

```typescript
// 泛型函数
function first<T>(arr: T[]): T | undefined { return arr[0]; }
first([1, 2, 3]);    // number
first(['a', 'b']);   // string

// 泛型接口
interface Repository<T> {
    findById(id: number): T | undefined;
    save(item: T): void;
}

// 泛型类
class Stack<T> {
    private items: T[] = [];
    push(item: T) { this.items.push(item); }
    pop(): T | undefined { return this.items.pop(); }
}

// 泛型约束
function logLength<T extends { length: number }>(x: T): void {
    console.log(x.length);
}

// keyof 约束
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
    return obj[key];
}
getProperty({ name: 'Tom', age: 25 }, 'name');  // 'Tom'
```

## L2 实践层

### 泛型速查

| 语法 | 说明 | 示例 |
|------|------|------|
| `<T>` | 类型参数 | `function fn<T>(x: T)` |
| `extends` | 泛型约束 | `<T extends { id: number }>` |
| `keyof T` | 键的联合类型 | `keyof User` → `'id' \| 'name'` |
| `T[K]` | 索引访问类型 | `T['name']` |

## L3 专家层

### 泛型在 React 中的应用

```typescript
function List<T>({ items, render }: { items: T[]; render: (item: T) => React.ReactNode }) {
    return <ul>{items.map(render)}</ul>;
}

// API 响应
interface ApiResponse<T> { data: T; status: number }
async function fetchAPI<T>(url: string): Promise<ApiResponse<T>> {
    const res = await fetch(url);
    return res.json();
}
```

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **泛型** | 类型参数化，编写通用代码 | `<T>(x: T): T` |
| **泛型约束** | 限制泛型必须满足的条件 | `<T extends { length: number }>` |
| **keyof** | 获取对象键的联合类型 | `keyof User` |
