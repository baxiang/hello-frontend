# TypeScript 零基础到进阶学习大纲

> 从入门到独立开发，约 40 小时

---

## 学习路径

```
入门(3h) → 基础(10h) → 进阶(12h) → 高级(10h) → 实战(5h)
```

## 各阶段内容

### 第一阶段：入门（3 小时）

- TypeScript 是什么（与 JS 的关系）
- 安装和环境搭建
- 类型注解基础
- 编译和运行

### 第二阶段：基础（10 小时）

- 基础类型（string、number、boolean）
- 接口和类型别名
- 函数类型
- 类和访问修饰符

### 第三阶段：进阶（12 小时）

- 泛型编程
- 联合类型与交叉类型
- 类型守卫与类型收窄
- 工具类型（Partial、Pick、Omit）

### 第四阶段：高级（10 小时）

- 条件类型与 infer
- 映射类型
- 模板字面量类型
- 装饰器

### 第五阶段：实战（5 小时）

- React/Vue + TypeScript
- 项目结构最佳实践
- JS 迁移指南
