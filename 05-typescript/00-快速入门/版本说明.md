# TypeScript 版本说明（2026 年）

> 本教程基于最新稳定版本编写

---

## 推荐版本

| 工具 | 推荐版本 | 说明 |
|------|---------|------|
| **TypeScript** | `6.0.x` | 最新稳定版 |
| **Node.js** | `v22.x` 或 `v24.x` | LTS 长期支持版 |
| **VSCode** | 最新版 | 内置 TypeScript 支持 |

### 安装推荐版本

```bash
# 安装 TypeScript 6.0
npm install -g typescript@latest

# 验证版本
tsc --version    # 6.0.x
```

---

## TypeScript 5.x vs 6.x 主要变化

| 特性 | 5.x | 6.x |
|------|-----|-----|
| 模块解析 | `node` / `bundler` | 默认 `bundler` |
| 装饰器 | 实验性 (`experimentalDecorators`) | 标准化装饰器 |
| `using` 关键字 | ❌ | ✅ 支持 |
| 类型推断 | 严格 | 更精确 |

**新项目直接用 6.x，老项目保持 5.x 即可。**

---

## 本教程的版本适配

本教程代码同时兼容 TypeScript 5.x 和 6.x。如果有版本差异，会在代码注释中标注：

```typescript
// TypeScript 6.x 新语法
using resource = acquireResource();  // 6.x 支持

// TypeScript 5.x 兼容写法
const resource = acquireResource();
try { /* ... */ } finally { resource.release(); }
```
