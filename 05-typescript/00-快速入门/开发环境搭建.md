# 开发环境搭建

> 安装 TypeScript、配置 VS Code、编写第一个程序

---

## L1 理解层

**最简示例：**

```bash
npm install -g typescript
tsc --version    # 检查版本
echo 'let x: number = 5;' > hello.ts
tsc hello.ts     # 编译
node hello.js    # 运行
```

**详细示例：**

```bash
# 1. 安装 Node.js（前提）
# 官网: https://nodejs.org/
node --version   # v18+

# 2. 安装 TypeScript
npm install -g typescript
tsc --version

# 3. 初始化项目
mkdir my-app && cd my-app
npm init -y
npm install -D typescript
npx tsc --init   # 生成 tsconfig.json

# 4. 编写代码
# hello.ts
function greet(name: string): string {
    return `Hello, ${name}!`;
}
console.log(greet('TypeScript'));

# 5. 编译运行
npx tsc hello.ts    # 生成 hello.js
node hello.js       # 运行

# 或直接运行（需要 ts-node）
npx ts-node hello.ts
```

## L2 实践层

### VS Code 配置

```json
// .vscode/settings.json
{
    "typescript.preferences.importModuleSpecifier": "relative",
    "editor.formatOnSave": true,
    "editor.defaultFormatter": "esbenp.prettier-vscode"
}
```

### tsconfig.json 核心配置

```json
{
    "compilerOptions": {
        "target": "ES2020",
        "module": "commonjs",
        "strict": true,
        "outDir": "./dist",
        "rootDir": "./src",
        "esModuleInterop": true
    }
}
```
