# TypeScript 类型别名与联合类型

## 学习目标
- 掌握 type 基础用法
- 理解联合类型和交叉类型
- 学会使用映射类型工具
- 能够自定义工具类型

## 3.1 type 基础用法

### 基本类型别名

```typescript
// 基础类型别名
type Name = string;
type Age = number;
type IsDone = boolean;

// 使用
const name: Name = '张三';
const age: Age = 25;
const isDone: IsDone = true;
```

### 对象类型别名

```typescript
type User = {
    id: number;
    name: string;
    email: string;
};

const user: User = {
    id: 1,
    name: '张三',
    email: 'zhangsan@example.com'
};

// 嵌套对象
type Address = {
    street: string;
    city: string;
    zipcode: string;
};

type Person = {
    name: string;
    age: number;
    address: Address;
};

const person: Person = {
    name: '李四',
    age: 30,
    address: {
        street: '朝阳路',
        city: '北京',
        zipcode: '100000'
    }
};
```

### 泛型类型别名

```typescript
// 泛型类型别名
type Container<T> = {
    value: T;
};

const stringContainer: Container<string> = {
    value: 'hello'
};

const numberContainer: Container<number> = {
    value: 42
};

// 多个泛型参数
type Pair<T, U> = {
    first: T;
    second: U;
};

const pair: Pair<string, number> = {
    first: 'hello',
    second: 42
};
```

## 3.2 联合类型

### 基础用法

```typescript
// 可以是多种类型之一
type ID = string | number;

const id1: ID = 'abc123';
const id2: ID = 123456;

// 错误：不能是其他类型
// const id3: ID = true;

// 多个类型
type Status = 'pending' | 'success' | 'error';

const status1: Status = 'pending';
const status2: Status = 'success';

// 错误
// const status3: Status = 'failed';
```

### 联合类型的属性访问

```typescript
interface Bird {
    name: string;
    fly(): void;
}

interface Fish {
    name: string;
    swim(): void;
}

type Animal = Bird | Fish;

function move(animal: Animal) {
    // 只能访问共有的属性和方法
    console.log(animal.name);  // OK

    // animal.fly();  // 错误：Fish 没有 fly
    // animal.swim();  // 错误：Bird 没有 swim
}
```

### 类型守卫

```typescript
type Animal = Bird | Fish;

// 使用 typeof
function isBird(animal: Animal): animal is Bird {
    return 'fly' in animal;
}

function move(animal: Animal) {
    if (isBird(animal)) {
        animal.fly();  // OK，类型 narrowed 为 Bird
    } else {
        animal.swim();  // OK，类型 narrowed 为 Fish
    }
}

// 使用 instanceof
class BirdClass {
    fly() {
        console.log('鸟在飞');
    }
}

class FishClass {
    swim() {
        console.log('鱼在游');
    }
}

function move2(animal: BirdClass | FishClass) {
    if (animal instanceof BirdClass) {
        animal.fly();
    } else {
        animal.swim();
    }
}
```

### 可辨识联合

```typescript
// 带有字面量类型单例接口的联合类型
interface Square {
    kind: 'square';
    side: number;
}

interface Circle {
    kind: 'circle';
    radius: number;
}

interface Triangle {
    kind: 'triangle';
    base: number;
    height: number;
}

type Shape = Square | Circle | Triangle;

function getArea(shape: Shape): number {
    switch (shape.kind) {
        case 'square':
            return shape.side * shape.side;
        case 'circle':
            return Math.PI * shape.radius ** 2;
        case 'triangle':
            return (shape.base * shape.height) / 2;
        default:
            //  exhaustive check
            const _exhaustiveCheck: never = shape;
            return _exhaustiveCheck;
    }
}
```

## 3.3 交叉类型

### 基础用法

```typescript
// 合并多个类型
interface A {
    a: string;
}

interface B {
    b: number;
}

type C = A & B;

const obj: C = {
    a: 'hello',
    b: 42
};
```

### 实用场景

```typescript
// 组件 Props 合并
interface BaseProps {
    className?: string;
    style?: React.CSSProperties;
}

interface ButtonProps extends BaseProps {
    label: string;
    onClick: () => void;
    disabled?: boolean;
}

// 使用交叉类型添加额外属性
type ExtendedButtonProps = ButtonProps & {
    icon: string;
    loading?: boolean;
};

// 多个接口交叉
interface Timestamp {
    createdAt: number;
    updatedAt: number;
}

interface SoftDelete {
    deletedAt?: number;
}

type BaseEntity = { id: number } & Timestamp & SoftDelete;

const entity: BaseEntity = {
    id: 1,
    createdAt: Date.now(),
    updatedAt: Date.now(),
    deletedAt: undefined
};
```

### 交叉类型的冲突处理

```typescript
// 同名属性，类型相同
interface A {
    name: string;
}

interface B {
    name: string;
}

type C = A & B;  // name: string

// 同名属性，类型不同
interface X {
    name: string;
}

interface Y {
    name: number;
}

type Z = X & Y;  // name: string & number = never

const obj: Z = {
    // name: 'hello'  // 错误
    // name: 42  // 错误
};

// 对象类型的交叉
interface Person {
    name: string;
}

interface Contact {
    email: string;
}

type Employee = Person & Contact;  // { name: string; email: string; }
```

## 3.4 映射类型基础

### 基础语法

```typescript
// 基于已有类型创建新类型
type Readonly<T> = {
    readonly [P in keyof T]: T[P];
};

type Partial<T> = {
    [P in keyof T]?: T[P];
};

type Required<T> = {
    [P in keyof T]-?: T[P];
};

// 使用
interface User {
    id: number;
    name: string;
    email: string;
}

type ReadonlyUser = Readonly<User>;
// { readonly id: number; readonly name: string; readonly email: string; }

type PartialUser = Partial<User>;
// { id?: number; name?: string; email?: string; }

type RequiredUser = Required<PartialUser>;
// { id: number; name: string; email: string; }
```

### 修改修饰符

```typescript
// 添加可选
type Mutable<T> = {
    -readonly [P in keyof T]: T[P];
};

// 移除可选
type Required<T> = {
    [P in keyof T]-?: T[P];
};

// 使用
interface Config {
    readonly timeout: number;
    retries?: number;
}

type MutableConfig = Mutable<Config>;
// { timeout: number; retries?: number; }

type StrictConfig = Required<Config>;
// { timeout: number; retries: number; }
```

## 3.5 常用工具类型

### Pick - 选取部分属性

```typescript
// 内置定义
type Pick<T, K extends keyof T> = {
    [P in K]: T[P];
};

// 使用
interface User {
    id: number;
    name: string;
    email: string;
    password: string;
}

// 只选取 id 和 name
type UserBasic = Pick<User, 'id' | 'name'>;
// { id: number; name: string; }

// API 响应中不包含密码
type UserResponse = Pick<User, 'id' | 'name' | 'email'>;
```

### Omit - 省略部分属性

```typescript
// 内置定义
type Omit<T, K extends keyof any> = Pick<T, Exclude<keyof T, K>>;

// 使用
interface User {
    id: number;
    name: string;
    email: string;
    password: string;
}

// 创建用户时不需要 id
type CreateUserInput = Omit<User, 'id'>;
// { name: string; email: string; password: string; }

// 更新用户时不需要 id 和 password
type UpdateUserInput = Omit<User, 'id' | 'password'>;
// { name: string; email: string; }
```

### Exclude / Extract / NonNullable

```typescript
// Exclude - 从联合类型中排除某些类型
type T0 = Exclude<'a' | 'b' | 'c', 'a'>;  // 'b' | 'c'
type T1 = Exclude<'a' | 'b' | 'c', 'a' | 'b'>;  // 'c'

// Extract - 从联合类型中提取某些类型
type T2 = Extract<'a' | 'b' | 'c', 'a' | 'b'>;  // 'a' | 'b'

// NonNullable - 排除 null 和 undefined
type T3 = NonNullable<string | number | null | undefined>;  // string | number

// 实用示例
type EventTypes = 'click' | 'scroll' | 'mousemove' | 'touchstart';
type MouseEventTypes = Exclude<EventTypes, 'scroll'>;  // 'click' | 'mousemove' | 'touchstart'
```

### Parameters / ReturnType / InstanceType

```typescript
// Parameters - 获取函数参数类型
type T0 = Parameters<(a: string, b: number) => void>;
// [string, number]

// ReturnType - 获取函数返回类型
type T1 = ReturnType<() => { id: number; name: string }>;
// { id: number; name: string; }

// InstanceType - 获取构造函数实例类型
class Person {
    constructor(public name: string, public age: number) {}
}

type T2 = InstanceType<typeof Person>;
// Person

// 实用示例
function createUser(name: string, email: string) {
    return { id: 1, name, email };
}

type CreateUserData = Parameters<typeof createUser>;
// [string, string]

type CreateUserResult = ReturnType<typeof createUser>;
// { id: number; name: string; email: string; }
```

### Record - 定义键值对类型

```typescript
// 内置定义
type Record<K extends keyof any, T> = {
    [P in K]: T;
};

// 使用
// 定义枚举映射
enum Status {
    Pending = 'pending',
    Success = 'success',
    Error = 'error'
}

const statusLabels: Record<Status, string> = {
    [Status.Pending]: '待处理',
    [Status.Success]: '成功',
    [Status.Error]: '错误'
};

// 定义配置对象
type RouteConfig = Record<string, { path: string; component: string }>;

const routes: RouteConfig = {
    home: { path: '/', component: 'HomePage' },
    about: { path: '/about', component: 'AboutPage' }
};

// 从联合类型创建
type Role = 'admin' | 'user' | 'guest';
type Permissions = Record<Role, string[]>;

const permissions: Permissions = {
    admin: ['read', 'write', 'delete'],
    user: ['read'],
    guest: ['read']
};
```

## 3.6 自定义工具类型

### DeepPartial - 深度可选

```typescript
type DeepPartial<T> = {
    [P in keyof T]?: T[P] extends object ? DeepPartial<T[P]> : T[P];
};

interface User {
    id: number;
    profile: {
        name: string;
        address: {
            city: string;
            street: string;
        };
    };
}

type UpdateUserInput = DeepPartial<User>;
// {
//     id?: number;
//     profile?: {
//         name?: string;
//         address?: {
//             city?: string;
//             street?: string;
//         };
//     };
// }
```

### DeepReadonly - 深度只读

```typescript
type DeepReadonly<T> = {
    readonly [P in keyof T]: T[P] extends object ? DeepReadonly<T[P]> : T[P];
};

interface Config {
    api: {
        url: string;
        timeout: number;
    };
}

type ReadonlyConfig = DeepReadonly<Config>;
// { readonly api: { readonly url: string; readonly timeout: number; } }
```

### Mutable - 移除只读

```typescript
type Mutable<T> = {
    -readonly [P in keyof T]: T[P];
};

interface ReadonlyUser {
    readonly id: number;
    readonly name: string;
}

type User = Mutable<ReadonlyUser>;
// { id: number; name: string; }
```

### ValueOf - 获取对象的值类型

```typescript
type ValueOf<T> = T[keyof T];

const colors = {
    red: '#ff0000',
    green: '#00ff00',
    blue: '#0000ff'
} as const;

type ColorName = keyof typeof colors;  // 'red' | 'green' | 'blue'
type ColorValue = ValueOf<typeof colors>;  // '#ff0000' | '#00ff00' | '#0000ff'
```

### Nullable / NonNullable

```typescript
type Nullable<T> = T | null;
type NonNullable<T> = T extends null | undefined ? never : T;

type T0 = Nullable<string>;  // string | null
type T1 = NonNullable<string | null>;  // string
type T2 = NonNullable<string | null | undefined>;  // string
```

## 3.7 模板字面量类型

### 基础用法

```typescript
// 字符串字面量联合类型
type EmailTemplate = 'welcome' | 'reset-password' | 'verify-email';
type EmailEvent = `${EmailTemplate}-email`;
// 'welcome-email' | 'reset-password-email' | 'verify-email-email'

// 多个联合类型组合
type Method = 'get' | 'post' | 'put' | 'delete';
type Resource = 'users' | 'posts' | 'comments';
type Endpoint = `${Method} /api/${Resource}`;
// 24 种组合
```

### 实用场景

```typescript
// CSS 属性
type CSSProperties = {
    [K in `margin${'' | 'Top' | 'Right' | 'Bottom' | 'Left'}`]?: string | number;
};

const style: CSSProperties = {
    margin: '10px',
    marginTop: '20px',
    marginRight: '15px'
};

// 事件处理器
type Event = 'click' | 'change' | 'submit';
type EventHandler = {
    [K in `on${Capitalize<Event>}`]: () => void;
};

interface ButtonProps extends EventHandler {
    label: string;
}

const props: ButtonProps = {
    label: 'Submit',
    onClick: () => {},
    onSubmit: () => {}
};
```

### 内置字符串工具类型

```typescript
// Capitalize - 首字母大写
type T0 = Capitalize<'hello'>;  // 'Hello'

// Lowercase - 转小写
type T1 = Lowercase<'HELLO'>;  // 'hello'

// Uppercase - 转大写
type T2 = Uppercase<'hello'>;  // 'HELLO'

// Uncapitalize - 首字母小写
type T3 = Uncapitalize<'Hello'>;  // 'hello'

// 实用示例
type Events = 'click' | 'focus' | 'blur';
type ReactEvents = {
    [K in `on${Capitalize<Events>}`]: () => void;
};
// { onClick: () => void; onFocus: () => void; onBlur: () => void; }
```

## 3.8 条件类型

### 基础语法

```typescript
// T extends U ? X : Y
type IsString<T> = T extends string ? true : false;

type T0 = IsString<string>;  // true
type T1 = IsString<number>;  // false
```

### 分布式条件类型

```typescript
// 当 T 是联合类型时，条件类型会分布应用
type ToArray<T> = T extends any ? T[] : never;

type T0 = ToArray<string>;  // string[]
type T1 = ToArray<string | number>;  // string[] | number[]

// 非分布式（使用元组包装）
type ToArrayNonDistributed<T> = [T] extends [any] ? T[] : never;

type T2 = ToArrayNonDistributed<string | number>;  // (string | number)[]
```

### infer 关键字

```typescript
// 推断类型
type ReturnType<T> = T extends (...args: any[]) => infer R ? R : any;

type T0 = ReturnType<() => string>;  // string
type T1 = ReturnType<(a: number) => Promise<number>>;  // Promise<number>

// 推断数组元素类型
type ElementType<T> = T extends (infer U)[] ? U : never;

type T2 = ElementType<string[]>;  // string
type T3 = ElementType<(string | number)[]>;  // string | number

// 推断 Promise 的值类型
type UnwrapPromise<T> = T extends Promise<infer U> ? U : T;

type T4 = UnwrapPromise<Promise<string>>;  // string
type T5 = UnwrapPromise<number>;  // number
```

### 内置条件工具类型

```typescript
// Extract<T, U> - 提取可赋值给 U 的类型
type T0 = Extract<'a' | 'b' | 'c', 'a' | 'b'>;  // 'a' | 'b'

// Exclude<T, U> - 排除可赋值给 U 的类型
type T1 = Exclude<'a' | 'b' | 'c', 'a'>;  // 'b' | 'c'

// NonNullable<T> - 排除 null 和 undefined
type T2 = NonNullable<string | null | undefined>;  // string

// ReturnType<T> - 获取函数返回类型
type T3 = ReturnType<() => { id: number }>;  // { id: number; }

// InstanceType<T> - 获取构造函数实例类型
class Person {}
type T4 = InstanceType<typeof Person>;  // Person

// ThisType<T> - 获取 this 类型
interface DB {
    log(): void;
}

interface User {
    id: number;
    name: string;
}

interface UserDB extends DB {
    getUser: (id: number) => {
        id: number;
        name: string;
    };
}
```

type FieldType = 'text' | 'number' | 'email' | 'password' | 'select' | 'checkbox';

type ValidationRule<T = any> = {

### 练习 3：状态管理类型

```typescript
// Action 类型
type Action<T extends string, P = void> = P extends void
    ? { type: T }
    : { type: T; payload: P };

// 组合 Actions
type UserActions =
    | Action<'LOGIN', { username: string; token: string }>
    | Action<'LOGOUT'>
    | Action<'UPDATE_PROFILE', Partial<{ username: string; avatar: string }>>;

// Reducer 类型
type Reducer<S, A> = (state: S, action: A) => S;

// State 类型
type UserState = {
    user: {
        id: number;
        username: string;
        avatar?: string;
    } | null;
    token: string | null;
    isAuthenticated: boolean;
    loading: boolean;
};

// 使用示例
const userReducer: Reducer<UserState, UserActions> = (state, action) => {
    switch (action.type) {
        case 'LOGIN':
            return {
                ...state,
                user: { id: 1, username: action.payload.username },
                token: action.payload.token,
                isAuthenticated: true
            };
        case 'LOGOUT':
            return {
                ...state,
                user: null,
                token: null,
                isAuthenticated: false
            };
        default:
            return state;
    }
};

// Dispatch 类型
type Dispatch<A> = (action: A) => void;

// Store 类型
type Store<S, A> = {
    state: S;
    dispatch: Dispatch<A>;
    subscribe: (listener: (state: S) => void) => () => void;
    getState: () => S;
};
```

## 3.10 常见问答

### Q1: interface 和 type 如何选择？

**答：**
- 需要声明合并、被类实现 → interface
- 联合类型、映射类型、复杂类型转换 → type

### Q2: 如何创建不可变类型？

```typescript
// 使用 Readonly
type Immutable<T> = Readonly<T>;

// 深度不可变
type DeepImmutable<T> = {
    readonly [P in keyof T]: T[P] extends object
        ? DeepImmutable<T[P]>
        : T[P];
};
```

### Q3: 如何从联合类型中排除某些类型？

```typescript
// 使用 Exclude
type Status = 'success' | 'error' | 'pending';
type FinalStatus = Exclude<Status, 'pending'>;  // 'success' | 'error'

// 自定义排除
type ExcludeNull<T> = Exclude<T, null>;
type T0 = ExcludeNull<string | null>;  // string
```
