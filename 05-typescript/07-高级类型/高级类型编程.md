## L1 理解层

**最简示例（1-3行）：**

```typescript
type DeepReadonly<T> = { readonly [K in keyof T]: DeepReadonly<T[K]> };
type EventName<T extends string> = `on${Capitalize<T>}`;
```

**详细示例：**

```typescript
// 递归类型
type DeepPartial<T> = T extends object
    ? { [K in keyof T]?: DeepPartial<T[K]> }
    : T;

// 模板字面量类型
type Alignment = 'top' | 'bottom' | 'left' | 'right';
type Position = `${Alignment}-${Alignment}`;  // 'top-top' | 'top-bottom' | ...

type EventName<T extends string> = `on${Capitalize<T>}`;
type ClickEvent = EventName<'click'>;  // 'onClick'

// 类型编程挑战
type Trim<S extends string> = S extends ` ${infer R}` ? Trim<R> : S extends `${infer L} ` ? Trim<L> : S;
type CamelCase<S extends string> = S extends `${infer A}-${infer B}${infer C}` ? `${A}${Uppercase<B>}${CamelCase<C>}` : S;
```

## L2 实践层

### 常用高级类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `DeepReadonly<T>` | 深层只读 | `{ readonly [K]: DeepReadonly<T[K]> }` |
| `DeepPartial<T>` | 深层可选 | `{ [K]?: DeepPartial<T[K]> }` |
| 模板字面量 | 字符串模式 | `` `on${Capitalize<T>}` `` |
