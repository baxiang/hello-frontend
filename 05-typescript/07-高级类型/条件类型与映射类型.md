# 条件类型与映射类型 ⭐⭐⭐

> 条件类型、infer、映射类型

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
type IsString<T> = T extends string ? true : false;
type NonNullable<T> = T extends null | undefined ? never : T;
type Readonly<T> = { readonly [K in keyof T]: T[K] };
```

**详细示例：**

```typescript
// 条件类型
type IsArray<T> = T extends any[] ? true : false;
type Unpack<T> = T extends (infer U)[] ? U : T;
type Flatten<T> = T extends any[] ? T[number] : T;

// infer — 推断类型
type ReturnType<T> = T extends (...args: any[]) => infer R ? R : never;
type Parameters<T> = T extends (...args: infer P) => any ? P : never;
type First<T> = T extends [infer F, ...any[]] ? F : never;

// 映射类型
type Readonly<T> = { readonly [K in keyof T]: T[K] };
type Partial<T> = { [K in keyof T]?: T[K] };
type Required<T> = { [K in keyof T]-?: T[K] };

// 键重映射
type Getters<T> = { [K in keyof T & string as `get${Capitalize<K>}`]: () => T[K] };
```

## L2 实践层

### 条件类型速查

| 语法 | 说明 | 示例 |
|------|------|------|
| `T extends U ? X : Y` | 条件类型 | `T extends string ? true : false` |
| `infer R` | 推断类型 | `T extends (...args: any[]) => infer R` |
| `[K in keyof T]` | 映射类型 | `{ [K in keyof T]: T[K] }` |
| `as` | 键重映射 | `[K in keyof T as \`get${K}\`]` |

---

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **条件类型** | 类型的三元表达式 | `T extends U ? X : Y` |
| **infer** | 在条件类型中推断类型 | `infer R` |
| **映射类型** | 从一个类型映射出新类型 | `{ [K in keyof T]: T[K] }` |
