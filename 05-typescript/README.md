# TypeScript 模块化学习路线

## 章节划分

### 第 1 章：基础类型
- 布尔、数字、字符串
- 数组和元组
- 枚举
- any、unknown、void、never
- 类型推断和断言
- 实践练习

### 第 2 章：接口
- 接口基础用法
- 可选属性和只读属性
- 函数类型接口
- 接口继承
- 实践练习

### 第 3 章：类型别名与联合类型
- type 基础用法
- 联合类型
- 交叉类型
- 映射类型（Partial/Required/Pick/Omit 等）
- 实践练习

### 第 4 章：泛型
- 泛型函数
- 泛型接口和类
- 泛型约束
- 泛型工具类型
- 实践练习

### 第 5 章：类型守卫与高级类型
- typeof 守卫
- instanceof 守卫
- 类型谓词
- 可辨识联合
- 条件类型
- 模板字面量类型
- 实践练习

### 第 6 章：装饰器（选学）
- 类装饰器
- 方法装饰器
- 属性装饰器
- 参数装饰器
- 实践练习

---

## 学习建议

1. **按顺序学习** - 章节之间有依赖关系
2. **每章练习** - 完成每章的实践代码
3. **实际应用** - 在项目中尝试使用 TypeScript
4. **理解优先** - 理解类型系统的设计思想
