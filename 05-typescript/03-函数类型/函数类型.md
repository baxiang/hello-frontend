# 函数类型 ⭐⭐

> 参数类型、返回值、函数重载、高阶函数

---

## L1 理解层

**最简示例（1-3行）：**

```typescript
function add(a: number, b: number): number { return a + b; }
const mul = (a: number, b: number): number => a * b;
```

**详细示例：**

```typescript
// 参数类型
function greet(name: string, age?: number): string {
    return age ? `${name}, ${age}岁` : name;
}

// 默认参数
function create(name: string, role = 'user') { return { name, role }; }

// 剩余参数
function sum(...nums: number[]): number { return nums.reduce((a, b) => a + b, 0); }

// 函数类型
type MathFn = (a: number, b: number) => number;
const add: MathFn = (a, b) => a + b;

// 函数重载
function parse(value: string): number;
function parse(value: number): string;
function parse(value: string | number): string | number {
    return typeof value === 'string' ? Number(value) : String(value);
}
```

## L2 实践层

### 函数类型速查

| 语法 | 说明 | 示例 |
|------|------|------|
| `(a: T) => R` | 函数类型 | `(x: number) => string` |
| `?` | 可选参数 | `name?: string` |
| `=` | 默认参数 | `role = 'user'` |
| `...args` | 剩余参数 | `...nums: number[]` |
