# 02-数据类型 - 数据的种类

> 理解 JavaScript 中不同类型的数据及其特点

## 学习目标

- ✅ 掌握 5 种基本数据类型
- ✅ 理解 Symbol 和 BigInt 特殊类型
- ✅ 掌握引用类型（对象、数组、函数）
- ✅ 学会类型检测和类型转换

## 2.1 数据类型总览

JavaScript 的数据类型分为两大类：

```
JavaScript 数据类型
│
├── 基本类型（值类型）- 7 种
│   ├── string（字符串）
│   ├── number（数字）
│   ├── boolean（布尔）
│   ├── null（空值）
│   ├── undefined（未定义）
│   ├── symbol（符号）- ES6 新增
│   └── bigint（大整数）- ES2020 新增
│
└── 引用类型 - 3 种
    ├── Object（对象）
    ├── Array（数组）
    └── Function（函数）
```

### 基本类型 vs 引用类型

```
基本类型：值直接存储在变量中
┌─────────────┐
│    25       │  ← 变量中直接存储值
└─────────────┘

引用类型：变量存储的是内存地址（引用）
┌─────────────┐     内存中
│   0x001     │ ──→ ┌─────────────┐
└─────────────┘     │  { name:    │
                    │   '张三' }  │
                    └─────────────┘
```

## 2.2 字符串（String）

### 三种声明方式

```javascript
// 单引号
let str1 = 'Hello';

// 双引号
let str2 = "World";

// 反引号（模板字符串）- ES6
let str3 = `JavaScript`;
```

### 模板字符串

```javascript
let name = '张三';
let age = 25;

// 传统拼接 - 麻烦
let message1 = '我叫' + name + '，今年' + age + '岁';

// 模板字符串 - 简洁
let message2 = `我叫${name}，今年${age}岁`;

// 多行字符串
let html = `
<div>
  <h1>标题</h1>
  <p>内容</p>
</div>
`;
```

### 常用字符串方法

```javascript
let str = 'Hello, World!';

// 获取长度
str.length;           // 13

// 大小写转换
str.toLowerCase();    // 'hello, world!'
str.toUpperCase();    // 'HELLO, WORLD!'

// 查找
str.indexOf('World'); // 7
str.includes('World'); // true

// 截取
str.slice(0, 5);      // 'Hello'
str.substring(0, 5);  // 'Hello'

// 替换
str.replace('World', 'JavaScript');  // 'Hello, JavaScript!'

// 分割
str.split(', ');      // ['Hello', 'World!']

// 去除空格
'  hello  '.trim();   // 'hello'
```

## 2.3 数字（Number）

### 数字字面量

```javascript
// 整数
let integer = 42;

// 小数
let float = 3.14;

// 负数
let negative = -10;

// 科学计数法
let big = 1.23e10;    // 12300000000

// 特殊值
let infinity = Infinity;   // 无穷大
let notNumber = NaN;       // 不是数字（Not a Number）
```

### 数字分隔符（ES2021）

```javascript
// 让大数字更易读
let salary = 1_000_000;     // 100 万
let phone = 138_0013_8000;  // 手机号分段
let bytes = 0xFF_FF_FF_FF;  // 十六进制
```

### 数字方法

```javascript
// 转字符串
(123).toString();       // '123'
(123).toString(2);      // '1111011'（二进制）
(123).toString(16);     // '7b'（十六进制）

// 保留小数
let num = 3.14159;
num.toFixed(2);         // '3.14'
num.toPrecision(4);     // '3.142'

// 安全整数
Number.MAX_SAFE_INTEGER;  // 9007199254740991
Number.MIN_SAFE_INTEGER;  // -9007199254740991
```

### NaN 详解

```javascript
// NaN 的产生
0 / 0;                // NaN
Math.sqrt(-1);        // NaN
parseInt('abc');      // NaN

// NaN 的特点
NaN === NaN;          // false（特殊：不等于任何值，包括自己）
Number.isNaN(NaN);    // true（正确的检测方式）
isNaN(NaN);           // true（老方法，不推荐）
```

## 2.4 布尔（Boolean）

### 布尔值

```javascript
// 只有两个值
let isTrue = true;    // 真
let isFalse = false;  // 假
```

### 实际应用场景

```javascript
// 状态标记
let isLoggedIn = true;     // 是否登录
let hasPermission = false; // 是否有权限
let isActive = true;       // 是否激活
let isVisible = false;     // 是否可见

// 条件判断
if (isLoggedIn) {
    showDashboard();
} else {
    showLoginPage();
}
```

### 真假值转换

```javascript
// 假值（falsy）- 只有这 6 个
false
0
-0
0n                    // BigInt 零
''                    // 空字符串
null
undefined
NaN

// 其他所有值都是真值（truthy）
'hello'               // true
42                    // true
[]                    // true（空数组也是真值）
{}                    // true（空对象也是真值）
```

## 2.5 null 和 undefined

### undefined - 未定义

```javascript
// 变量声明但未赋值
let notDefined;
console.log(notDefined);  // undefined

// 函数没有返回值
function noReturn() {}
console.log(noReturn());  // undefined

// 对象属性不存在
let obj = {};
console.log(obj.name);    // undefined
```

### null - 有意的空

```javascript
// 有意的空值
let empty = null;

// 表示"这个变量将来会有值，但现在没有"
let user = null;
// 稍后可能赋值
user = { name: '张三' };
```

### 区别对比

| 场景 | undefined | null |
|------|-----------|------|
| 含义 | 未定义，还没有这个值 | 有意的空值 |
| 产生 | 系统自动赋值 | 手动赋值 |
| 类型 | 'undefined' | 'object'（历史 bug） |
| 相等 | null == undefined → true | null === undefined → false |

```javascript
// 实际开发中的建议
// 用 undefined 表示"未定义"
// 用 null 表示"空值"或"无"

// 检测空值
if (value === null || value === undefined) {
    // 处理空值
}

// 或使用空值合并运算符
let result = value ?? '默认值';
```

## 2.6 Symbol - 唯一标识符

### 创建 Symbol

```javascript
// Symbol 是 ES6 引入的新类型
let id1 = Symbol('id');
let id2 = Symbol('id');

console.log(id1 === id2);  // false
// 即使描述相同，也是不同的 Symbol
```

### 使用场景

```javascript
// 1. 作为对象的唯一键
const ID = Symbol('id');
let user = {
    [ID]: 123,
    name: '张三'
};
console.log(user[ID]);  // 123

// 2. 避免属性名冲突
let obj = {};
let key1 = Symbol('key');
let key2 = Symbol('key');
obj[key1] = 'value1';
obj[key2] = 'value2';
// 两个属性不会冲突
```

### Symbol 的特性

```javascript
// Symbol 作为键不会被遍历
let obj = {
    [Symbol('id')]: 123,
    name: '张三'
};

Object.keys(obj);           // ['name']
Object.getOwnPropertySymbols(obj);  // [Symbol(id)]
```

## 2.7 BigInt - 大整数

### 为什么需要 BigInt？

```javascript
// JavaScript 数字的安全范围
Number.MAX_SAFE_INTEGER;  // 9007199254740991

// 超出范围会丢失精度
9007199254740999;         // 9007199254741000（精度丢失）

// BigInt 可以表示任意大的整数
let bigInt = 9007199254740991n;  // 后面加 n 表示 BigInt
```

### BigInt 的使用

```javascript
// 创建方式
let big1 = 123n;
let big2 = BigInt(123);
let big3 = BigInt('900719925474099999999');

// 运算
let a = 10n;
let b = 3n;
a + b;    // 13n
a * b;    // 30n
a / b;    // 3n（BigInt 除法取整）
a % b;    // 1n

// 注意：BigInt 和普通数字不能混用
// 10n + 5;  // ❌ 报错
10n + BigInt(5);  // ✅ 15n
```

## 2.8 引用类型 - 对象

### 对象字面量

```javascript
let user = {
    name: '张三',
    age: 25,
    email: 'zhang@example.com',
    sayHi: function() {
        console.log('你好！');
    }
};

// 访问属性
console.log(user.name);      // '张三'
console.log(user['name']);   // '张三'

// 修改属性
user.age = 26;

// 添加属性
user.city = '北京';

// 删除属性
delete user.email;
```

### 对象的方法

```javascript
let obj = {
    // 传统方法
    method1: function() {
        console.log('方法 1');
    },

    // 简写（ES6）
    method2() {
        console.log('方法 2');
    },

    // 箭头函数（注意 this）
    method3: () => {
        console.log('方法 3');
    }
};
```

## 2.9 引用类型 - 数组

### 数组字面量

```javascript
// 创建数组
let numbers = [1, 2, 3, 4, 5];
let mixed = [1, 'hello', true, null, { name: 'obj' }];

// 访问元素
console.log(numbers[0]);  // 1（从 0 开始）

// 修改元素
numbers[0] = 10;

// 数组长度
console.log(numbers.length);  // 5
```

### 数组常用方法

```javascript
let arr = [1, 2, 3, 4, 5];

// 增删
arr.push(6);           // 末尾添加
arr.pop();             // 末尾删除
arr.unshift(0);        // 开头添加
arr.shift();           // 开头删除

// 查找
arr.indexOf(3);        // 找索引
arr.includes(3);       // 是否包含
arr.find(x => x > 3);  // 找第一个符合条件的

// 转换
arr.join('-');         // 转字符串
arr.reverse();         // 反转
arr.sort((a, b) => a - b);  // 排序

// 映射过滤归约
arr.map(x => x * 2);       // [2, 4, 6, 8, 10]
arr.filter(x => x > 2);    // [3, 4, 5]
arr.reduce((sum, x) => sum + x, 0);  // 15
```

## 2.10 引用类型 - 函数

### 函数声明

```javascript
// 函数声明
function greet(name) {
    return `你好，${name}！`;
}

// 函数表达式
const greet2 = function(name) {
    return `你好，${name}！`;

// 箭头函数（ES6）
const greet3 = (name) => `你好，${name}！`;
```

## 2.11 类型检测

### typeof 运算符

```javascript
typeof 'hello'           // 'string'
typeof 42                // 'number'
typeof true              // 'boolean'
typeof undefined         // 'undefined'
typeof null              // 'object'（历史 bug）
typeof {}                // 'object'
typeof []                // 'object'（数组也是对象）
typeof function() {}     // 'function'
typeof Symbol()          // 'symbol'
typeof 123n              // 'bigint'
```

### 其他检测方法

```javascript
// Array.isArray - 检测数组
Array.isArray([]);       // true
Array.isArray({});       // false

// Object.prototype.toString - 最准确
Object.prototype.toString.call([]);   // '[object Array]'
Object.prototype.toString.call({});   // '[object Object]'
Object.prototype.toString.call(null); // '[object Null]'
Object.prototype.toString.call(undefined); // '[object Undefined]'
```

## 2.12 类型转换

### 转字符串

```javascript
String(123);             // '123'
(123).toString();        // '123'
` ${123}`;               // '123'（模板字符串）
String(true);            // 'true'
String(null);            // 'null'
String(undefined);       // 'undefined'
```

### 转数字

```javascript
Number('123');           // 123
Number('123px');         // NaN（包含非数字字符）
parseInt('123px');       // 123（解析整数）
parseFloat('3.14px');    // 3.14（解析小数）
+'123';                  // 123（一元加号）
'123' * 1;               // 123（乘除也会转换）
```

### 转布尔

```javascript
Boolean(0);              // false
Boolean('');             // false
Boolean(null);           // false
Boolean(undefined);      // false
Boolean(NaN);            // false
Boolean('hello');        // true
Boolean(1);              // true
Boolean({});             // true
Boolean([]);             // true
```

## 2.13 实践练习

### 练习 1：数据类型识别

```javascript
// 判断以下值的类型
typeof 'Hello';          // 'string'
typeof 42;               // 'number'
typeof true;             // 'boolean'
```

### 练习 2：类型转换

```javascript
// 字符串转数字
let str = '123';

let price = 99;
```

### 练习 3：对象和数组操作

```javascript
// 对象操作
let user = { name: '张三', age: 25 };

console.log(Object.keys(user));  // ['name', 'email']

// 数组操作
let arr = [1, 2, 3, 4, 5];
let doubled = arr.map(x => x * 2);

let sum = arr.reduce((acc, x) => acc + x, 0);
console.log(doubled);  // [2, 4, 6, 8, 10]
console.log(evens);    // [2, 4]
console.log(sum);      // 15
```

## 2.14 常见问答

### Q1: 为什么 typeof null 返回 'object'？

这是 JavaScript 早期的一个 bug，为了兼容性一直保留。

```javascript
// 正确的检测 null 的方法
if (value === null) {
    // 处理 null
}

// 或使用 lodash 的 isNull
_.isNull(value);
```

### Q2: 如何准确检测数组？

```javascript
// 不推荐
typeof [];              // 'object'

// 推荐
Array.isArray([]);      // true
```

### Q3: 基本类型和引用类型有什么区别？

```javascript
// 基本类型：值传递
let a = 5;
let b = a;
b = 10;
console.log(a);  // 5（不受影响）

// 引用类型：引用传递
let obj1 = { name: '张三' };
let obj2 = obj1;
obj2.name = '李四';
console.log(obj1.name);  // '李四'（会受影响）
```
