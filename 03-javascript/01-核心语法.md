## 1.1 变量与常量 - 存储数据的容器

### 什么是变量？

**变量就像一个带标签的盒子**，用来存储数据。

```
┌─────────────────────────────┐
│      变量：age              │
│  ┌─────────────────────┐    │
│  │         25          │    │
│  └─────────────────────┘    │
└─────────────────────────────┘
   标签（变量名）   盒子里的值
```

### 三种声明方式

#### var - 老式声明（不推荐）

```javascript
// var 是 JavaScript 早期的声明方式
var name = '张三';
var name = '李四';  // 可以重复声明，不会报错

// 问题：容易意外覆盖
var age = 25;
if (true) {
    var age = 30;  // 覆盖了外面的 age
}
console.log(age);  // 30（可能不是你想要的）
```

#### let - 现代声明（推荐）

```javascript
// let 声明的变量可以修改
let age = 25;
age = 26;  // ✅ 可以重新赋值

// 不能重复声明
let name = '张三';
// let name = '李四';  // ❌ 报错

// 块级作用域 - 只在 {} 内有效
let outside = '外面';
if (true) {
    let inside = '里面';
    console.log(outside);  // ✅ 可以访问
}
console.log(inside);  // ❌ 报错：inside 只在块内有效
```

#### const - 常量（推荐用于不变的值）

```javascript
// const 声明后不能重新赋值
const PI = 3.14159;
// PI = 3;  // ❌ 报错：常量不能修改

// 但对象的属性可以修改
const user = { name: '张三' };
user.name = '李四';  // ✅ 可以修改对象属性
// user = {};  // ❌ 不能重新赋值

// 数组也一样
const arr = [1, 2, 3];
arr.push(4);  // ✅ 可以修改数组
// arr = [];  // ❌ 不能重新赋值
```

### 生活例子

```
想象你在搬家，需要给箱子贴标签：

var 箱子 = '衣服'
var 箱子 = '书'  // 可以撕掉标签重新贴（不推荐）

let 箱子 = '衣服'
let 箱子 = '书'  // ❌ 一个箱子不能有两个标签

const 易碎品 = '玻璃杯'
// 易碎品 = '书本'  // ❌ 易碎品标签不能改

// 但箱子里的东西可以换
const 箱子 = ['衣服']
箱子.push('裤子')  // ✅ 可以往箱子里加东西
```

### 命名规则

```javascript
// ✅ 合法的变量名
let userName;      // 驼峰命名（推荐）
let $element;      // 可以包含 $
let _private;      // 可以包含下划线
let 中文;          // 可以用中文（但不推荐）

// ❌ 非法的变量名
// let 123abc;     // 不能数字开头
// let user-name;  // 不能包含连字符
// let class;      // 不能用保留字

// 常量命名：全部大写 + 下划线
const MAX_COUNT = 100;
const API_URL = 'https://api.example.com';
```

## 1.2 数据类型 - 数据的种类

### 基本数据类型 - 不可分割的简单值

**基本数据类型就像"原子"**，是不可再分的基本单位。

```
生活例子：
- 字符串 = 文字信息
- 数字 = 数值
- 布尔 = 是非判断
- null = 空盒子
- undefined = 还没放东西的盒子
```

#### 字符串（String）- 文本内容

```javascript
// 三种写法
let str1 = '单引号';
let str2 = "双引号";
let str3 = `反引号（模板字符串）`;

// 模板字符串 - 可以嵌入变量
let name = '张三';
let age = 25;

// 传统拼接 - 麻烦
let message1 = '我叫' + name + '，今年' + age + '岁';

// 模板字符串 - 简洁
let message2 = `我叫${name}，今年${age}岁`;

// 多行字符串
let html = `
<div>
  <h1>标题</h1>
  <p>内容</p>
</div>
`;
```

#### 数字（Number）- 所有数值

```javascript
// JavaScript 只有一种数字类型
let integer = 42;       // 整数
let float = 3.14;       // 小数
let negative = -10;     // 负数

// 特殊值
let infinity = Infinity;    // 无穷大
let notNumber = NaN;        // 不是数字（Not a Number）

// 数字分隔符（ES2021）- 让大数字更易读
let salary = 1_000_000;     // 100 万，方便数零
let phone = 138_0013_8000;  // 手机号分段
```

#### 布尔（Boolean）- 是非判断

```javascript
// 只有两个值
let isTrue = true;   // 真
let isFalse = false; // 假

// 实际应用场景
let isLoggedIn = true;     // 是否登录
let hasPermission = false; // 是否有权限
let isActive = true;       // 是否激活
```

#### null 和 undefined - 空值

```javascript
// null - 有意的空，"盒子里是空的"
let empty = null;

// undefined - 未定义，"还没有这个盒子"
let notDefined;
console.log(notDefined);  // undefined

// 区别
let user1 = null;     // 用户存在，但没有数据
let user2;            // 用户变量还没声明
```

#### Symbol 和 BigInt - 特殊类型

```javascript
// Symbol - 唯一的标识符
let id1 = Symbol('id');
let id2 = Symbol('id');
console.log(id1 === id2);  // false（即使描述相同，也是不同的）

// BigInt - 超大整数
let bigInt = 9007199254740991n;  // 超出安全整数范围
```

### 引用数据类型 - 复杂的数据结构

**引用类型就像"储物柜"**，存储的是指向数据的"钥匙"（引用）。

```
基本类型：盒子里直接放值
┌─────────────┐
│    25       │
└─────────────┘

引用类型：盒子里放地址，实际数据在别处
┌─────────────┐     内存中
│   0x001     │ ──→ ┌─────────────┐
└─────────────┘     │  { name:    │
                    │   '张三' }  │
                    └─────────────┘
```

#### 对象（Object）- 键值对集合

```javascript
// 对象字面量
let user = {
    name: '张三',
    age: 25,
    email: 'zhang@example.com',
    sayHi: function() {
        console.log('你好！');
    }
};

// 访问属性
console.log(user.name);      // '张三'
console.log(user['name']);   // '张三'

// 修改属性
user.age = 26;

// 添加属性
user.city = '北京';

// 删除属性
delete user.email;
```

#### 数组（Array）- 有序列表

```javascript
// 数组字面量
let numbers = [1, 2, 3, 4, 5];
let mixed = [1, 'hello', true, null, { name: 'obj' }];

// 访问元素
console.log(numbers[0]);  // 1（从 0 开始）

// 修改元素
numbers[0] = 10;

// 数组长度
console.log(numbers.length);  // 5
```

#### 函数（Function）- 可执行的代码块

```javascript
// 函数声明
function greet(name) {
    return `你好，${name}！`;
}

// 箭头函数（ES6）
const greet2 = (name) => `你好，${name}！`;
```

#### Date 和 RegExp

```javascript
// Date - 日期时间
let now = new Date();
console.log(now.getFullYear());  // 2024

// RegExp - 正则表达式
let regex = /pattern/gi;  // 全局、忽略大小写匹配
```

### 类型检测

```javascript
// typeof - 检测基本类型
typeof 'hello'           // 'string'
typeof 42                // 'number'
typeof true              // 'boolean'
typeof undefined         // 'undefined'
typeof null              // 'object'（历史 bug）
typeof {}                // 'object'
typeof []                // 'object'（数组也是对象）
typeof function() {}     // 'function'

// Array.isArray - 检测数组
Array.isArray([]);       // true
Array.isArray({});       // false

// Object.prototype.toString - 最准确
Object.prototype.toString.call([]);  // '[object Array]'
Object.prototype.toString.call({});  // '[object Object]'
Object.prototype.toString.call(null); // '[object Null]'
```

### 类型转换

```javascript
// 转字符串
String(123);             // '123'
(123).toString();        // '123'
` ${123}`;               // '123'（模板字符串）

// 转数字
Number('123');           // 123
parseInt('123px');       // 123（解析整数）
parseFloat('3.14px');    // 3.14（解析小数）
+'123';                  // 123（一元加号）

// 转布尔
Boolean(0);              // false（0 是假值）
Boolean('');             // false（空字符串是假值）
Boolean(null);           // false
Boolean(undefined);      // false
Boolean(NaN);            // false
Boolean('hello');        // true（非空字符串是真值）
Boolean(1);              // true
Boolean({});             // true（对象是真值）
Boolean([]);             // true（数组是真值）
```

## 1.3 运算符 - 对数据进行操作

### 算术运算符 - 数学计算

```javascript
let a = 10, b = 3;

a + b      // 13   加法
a - b      // 7    减法
a * b      // 30   乘法
a / b      // 3.333... 除法
a % b      // 1    取余（求余数）
a ** b     // 1000 幂运算（10 的 3 次方）

// 自增自减
let x = 5;
x++;       // 6  后置递增（先用后加）
++x;       // 7  前置递增（先加后用）
x--;       // 6  后置递减
--x;       // 5  前置递减
```

### 比较运算符 - 比较大小

```javascript
// 相等比较
5 == '5'       // true  宽松相等（会类型转换）
5 === '5'      // false 严格相等（不会类型转换）

// 为什么 5 == '5' 是 true？
// 因为 == 会把字符串 '5' 转成数字 5，然后比较
// 所以 5 == 5 是 true

// 推荐使用 ===（严格相等）
5 !== '5'      // true  严格不相等
5 != '5'       // false 宽松不相等

// 大小比较
5 > 3          // true
5 < 3          // false
5 >= 5         // true（大于或等于）
5 <= 4         // false
```

### 逻辑运算符 - 逻辑判断

```javascript
// 与（&&）- 两个都真才真
true && true   // true
true && false  // false

// 或（||）- 有一个真就真
true || false  // true
false || false // false

// 非（!）- 取反
!true          // false
!false         // true

// 短路求值 - 实际开发中常用
let name = userInput || '默认用户';
// 如果 userInput 有值，就用 userInput
// 如果 userInput 是空，就用 '默认用户'

let admin = isLoggedIn && isAdmin;
// 只有登录了 且 是管理员，才是 true
```

### 空值合并运算符（??）- ES2020 新特性

```javascript
// || 的问题：0、空字符串也会被当作默认值
let count = userCount || 10;
// 如果 userCount 是 0，会得到 10（可能不是你想要的）

// ?? 只有 null 或 undefined 才用默认值
let count2 = userCount ?? 10;
// 如果 userCount 是 0，会得到 0（正确）
// 如果 userCount 是 null/undefined，才得到 10

// 对比
'' || 'default'      // 'default'
'' ?? 'default'      // ''（空字符串保留）

0 || 10              // 10
0 ?? 10              // 0（0 保留）
```

## 1.4 控制流 - 控制代码执行顺序

### 条件语句 - 根据条件执行不同代码

#### if-else 语句

```javascript
// 生活例子：考试成绩评定
let score = 85;

if (score >= 90) {
    console.log('优秀');
} else if (score >= 80) {
    console.log('良好');
} else if (score >= 60) {
    console.log('及格');
} else {
    console.log('不及格');
}

// 实际应用场景
if (isLoggedIn) {
    showUserProfile();
} else {
    showLoginButton();
}
```

#### 三元运算符 - 简化 if-else

```javascript
// 传统写法
let result;
if (score >= 60) {
    result = '及格';
} else {
    result = '不及格';
}

// 三元运算符 - 简洁写法
let result = score >= 60 ? '及格' : '不及格';

// 语法：条件 ? 真值 : 假值
```

#### switch 语句 - 多条件分支

```javascript
// 生活例子：星期几
let day = 3;

switch (day) {
    case 1:
        console.log('周一');
        break;  // 重要：跳出 switch
    case 2:
        console.log('周二');
        break;
    case 3:
        console.log('周三');
        break;
    default:
        console.log('其他');
}

// 实际应用场景
switch (orderStatus) {
    case 'pending':
        showPendingUI();
        break;
    case 'paid':
        showPaidUI();
        break;
    case 'shipped':
        showShippedUI();
        break;
    default:
        showUnknownUI();
}
```

### 循环语句 - 重复执行代码

#### for 循环 - 最常用的循环

```javascript
// 语法：for (初始值; 条件; 更新)
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// 生活例子：跑圈
// for (let 圈数 = 0; 圈数 < 10; 圈数++) {
//     跑一圈();
// }
```

#### while 循环 - 条件满足就继续

```javascript
// 语法：while (条件)
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// 实际应用场景
while (!gameOver) {
    updateGame();
    render();
}
```

#### do-while 循环 - 至少执行一次

```javascript
// 语法：do { ... } while (条件)
let j = 0;
do {
    console.log(j);
    j++;
} while (j < 5);

// 和 while 的区别：
// while：先判断，再执行
// do-while：先执行，再判断
```

#### for...in 和 for...of

```javascript
// for...in - 遍历对象的键
let obj = { a: 1, b: 2, c: 3 };
for (let key in obj) {
    console.log(key, obj[key]);
    // 'a' 1
    // 'b' 2
    // 'c' 3
}

// for...of - 遍历数组的值（ES6）
let arr = [1, 2, 3, 4, 5];
for (let value of arr) {
    console.log(value);
    // 1, 2, 3, 4, 5
}

// 对比
// for...in 得到的是键（属性名）
// for...of 得到的是值
```

#### break 和 continue

```javascript
for (let i = 0; i < 10; i++) {
    if (i === 3) continue;  // 跳过本次，继续下一次
    if (i === 7) break;     // 直接跳出循环
    console.log(i);
}
// 输出：0, 1, 2, 4, 5, 6
// 3 被跳过，7 及以后不执行
```

## 1.5 函数 - 可复用的代码块

### 什么是函数？

**函数就像"机器"**，输入原料（参数），经过加工（函数体），产出产品（返回值）。

```
┌─────────────────────────────────┐
│          函数：add              │
│                                 │
│  输入 → [ 加工过程 ] → 输出      │
│   a,b  → [  a + b   ] → 结果    │
└─────────────────────────────────┘
```

### 函数声明

```javascript
// 函数声明 - 会提升（可以在声明前调用）
function add(a, b) {
    return a + b;
}

// 函数表达式 - 不会提升
const subtract = function(a, b) {
    return a - b;
};

// 箭头函数 - ES6（简洁）
const multiply = (a, b) => a * b;

// 多语句箭头函数
const divide = (a, b) => {
    if (b === 0) {
        throw new Error('除数不能为零');
    }
    return a / b;
};
```

### 函数参数

#### 默认参数

```javascript
// ES6 之前 - 麻烦
function greet(name) {
    name = name || '访客';
    console.log(`你好，${name}`);
}

// ES6 - 简洁
function greet(name = '访客') {
    console.log(`你好，${name}`);
}

greet();           // 你好，访客
greet('张三');     // 你好，张三
```

#### 剩余参数

```javascript
// 把多余的参数收集到数组中
function sum(...numbers) {
    return numbers.reduce((acc, cur) => acc + cur, 0);
}

sum(1, 2, 3);       // 6
sum(1, 2, 3, 4, 5); // 15

// ...numbers 会把所有参数收集到数组中
// numbers = [1, 2, 3]
```

#### 解构参数

```javascript
// 直接从对象中解构
function printUser({ name, age }) {
    console.log(`${name}, ${age}岁`);
}

const user = { name: '张三', age: 25 };
printUser(user);  // 张三，25 岁
```

## 1.6 作用域与闭包

### 什么是作用域？

**作用域就是变量的"活动范围"**。

```
生活例子：
- 全局作用域 = 广场（谁都能访问）
- 函数作用域 = 房间（只有房间里的人能访问）
- 块级作用域 = 保险箱（只有特定代码能访问）
```

#### 全局作用域

```javascript
// 在任何地方都能访问
let globalVar = '全局变量';

function func() {
    console.log(globalVar);  // ✅ 可以访问
}
```

#### 函数作用域（var）

```javascript
function outer() {
    var outerVar = '外部变量';
    
    function inner() {
        var innerVar = '内部变量';
        console.log(outerVar);  // ✅ 可以访问外部
    }
    
    // console.log(innerVar);  // ❌ 不能访问内部
}
```

#### 块级作用域（let/const）

```javascript
// let/const 有块级作用域
if (true) {
    let blockVar = '块级变量';
    console.log(blockVar);  // ✅ 块内可以访问
}
// console.log(blockVar);  // ❌ 块外不能访问

for (let i = 0; i < 5; i++) {
    console.log(i);
}
// console.log(i);  // ❌ i 只在循环内有效
```

### 什么是闭包？

**闭包就是函数记住并访问其创建时的环境**。

```javascript
// 生活例子：
// 你离开家（外部函数执行完），
// 但你口袋里还带着家里的钥匙（闭包），
// 随时可以回家（访问外部变量）

function createCounter() {
    let count = 0;  // 外部函数的变量
    
    // 内部函数（闭包）
    return function() {
        count++;
        return count;
    };
}

const counter = createCounter();
console.log(counter());  // 1
console.log(counter());  // 2
console.log(counter());  // 3

// count 变量一直被保留着
```

#### 闭包的实际应用

```javascript
// 1. 数据私有化
function createBankAccount(initialBalance) {
    let balance = initialBalance;  // 私有变量
    
    return {
        deposit: function(amount) {
            balance += amount;
        },
        withdraw: function(amount) {
            if (amount > balance) {
                throw new Error('余额不足');
            }
            balance -= amount;
        },
        getBalance: function() {
            return balance;
        }
    };
}

const account = createBankAccount(1000);
account.deposit(500);
console.log(account.getBalance());  // 1500
// 不能直接访问 balance，只能通过方法操作
```

## 1.7 对象详解

### 创建对象的方式

```javascript
// 1. 对象字面量（最常用）
let obj = {
    name: '张三',
    age: 25,
    sayHi() {
        console.log(`Hi, I'm ${this.name}`);
    }
};

// 2. 构造函数
function Person(name, age) {
    this.name = name;
    this.age = age;
}
let p1 = new Person('李四', 30);

// 3. Object.create
let proto = { greet() { console.log('Hello'); } };
let obj2 = Object.create(proto);

// 4. class 语法（ES6）
class Animal {
    constructor(name) {
        this.name = name;
    }
    speak() {
        console.log(`${this.name} makes a sound`);
    }
}
```

### 访问属性

```javascript
let user = { name: '张三', age: 25 };

// 点表示法
user.name;  // '张三'

// 方括号表示法
user['name'];  // '张三'

// 动态属性名
let key = 'age';
user[key];  // 25

// 区别：
// 点表示法：属性名固定
// 方括号：属性名可以是变量
```

### 对象方法

```javascript
let obj = {
    // 传统方法
    method1: function() {
        console.log('方法 1');
    },
    
    // 简写（ES6）
    method2() {
        console.log('方法 2');
    },
    
    // 箭头函数（注意 this）
    method3: () => {
        console.log('方法 3');
    }
```

### 数组方法 - 增删改查

```javascript
let arr = [1, 2, 3, 4, 5];

// ========== 增删 ==========

// push - 末尾添加
arr.push(6);       // [1, 2, 3, 4, 5, 6]

// pop - 末尾删除
arr.pop();         // 返回 6，arr 变成 [1, 2, 3, 4, 5]

// unshift - 开头添加
arr.unshift(0);    // [0, 1, 2, 3, 4, 5]

// shift - 开头删除
arr.shift();       // 返回 0，arr 变成 [1, 2, 3, 4, 5]

// ========== 切片 ==========

// slice - 截取（不修改原数组）
arr.slice(1, 3);   // [2, 3]

// splice - 删除/插入（修改原数组）
arr.splice(1, 2);      // 从位置 1 删除 2 个元素
arr.splice(1, 0, 'a'); // 在位置 1 插入'a'

// ========== 查找 ==========

// indexOf - 找索引
arr.indexOf(3);    // 2

// includes - 是否包含
arr.includes(3);   // true

// find - 找第一个符合条件的
arr.find(x => x > 3);  // 4

// findIndex - 找索引
arr.findIndex(x => x > 3);  // 3

// ========== 转换 ==========

// join - 转字符串
arr.join('-');     // '1-2-3-4-5'

// reverse - 反转
arr.reverse();     // [5, 4, 3, 2, 1]

// sort - 排序
arr.sort((a, b) => a - b);  // [1, 2, 3, 4, 5]
```

### 数组方法 - 映射过滤归约

```javascript
let arr = [1, 2, 3, 4, 5];

// map - 映射（每个元素都处理）
arr.map(x => x * 2);       // [2, 4, 6, 8, 10]

// filter - 过滤（只保留符合条件的）
arr.filter(x => x > 2);    // [3, 4, 5]

// reduce - 归约（汇总成一个值）
arr.reduce((sum, x) => sum + x, 0);  // 15

// forEach - 遍历（不返回值）
arr.forEach(x => console.log(x));

// some - 是否有符合条件的
arr.some(x => x > 3);      // true

// every - 是否都符合条件
arr.every(x => x > 0);     // true
```

## 1.10 实践练习

### 练习 1：简易计算器

```javascript
class Calculator {
    constructor() {
        this.history = [];
    }

    add(a, b) {
        return this._record(a + b, `${a} + ${b}`);
    }

    subtract(a, b) {
        return this._record(a - b, `${a} - ${b}`);
    }

    multiply(a, b) {
        return this._record(a * b, `${a} × ${b}`);
    }

    divide(a, b) {
        if (b === 0) {
            throw new Error('除数不能为零');
        }
        return this._record(a / b, `${a} ÷ ${b}`);
    }

    _record(result, expression) {
        this.history.push({ expression, result });
        return result;
    }

    getHistory() {
        return this.history;
    }

    clearHistory() {
        this.history = [];
    }
}

// 使用
const calc = new Calculator();
console.log(calc.add(5, 3));      // 8
console.log(calc.multiply(4, 7)); // 28
console.log(calc.getHistory());
```

### 练习 2：待办事项管理

```javascript
class TodoManager {
    constructor() {
        this.todos = [];
        this.nextId = 1;
    }

    add(text, priority = 'medium') {
        const todo = {
            id: this.nextId++,
            text,
            priority,
            completed: false,
            createdAt: new Date()
        };
        this.todos.push(todo);
        return todo;
    }

    remove(id) {
        const index = this.todos.findIndex(t => t.id === id);
        if (index === -1) {
            throw new Error(`未找到 ID 为 ${id} 的待办事项`);
        }
        return this.todos.splice(index, 1)[0];
    }

    toggle(id) {
        const todo = this.todos.find(t => t.id === id);
        if (!todo) {
            throw new Error(`未找到 ID 为 ${id} 的待办事项`);
        }
        todo.completed = !todo.completed;
        return todo;
    }

    list(options = {}) {
        let result = [...this.todos];

        if (options.completed !== undefined) {
            result = result.filter(t => t.completed === options.completed);
        }

        if (options.priority) {
            result = result.filter(t => t.priority === options.priority);
        }

        return result;
    }

    getStats() {
        const total = this.todos.length;
        const completed = this.todos.filter(t => t.completed).length;
        return {
            total,
            completed,
            pending: total - completed,
            completionRate: total === 0 ? 0 : (completed / total * 100).toFixed(1) + '%'
        };
    }
}

// 使用
const todo = new TodoManager();
todo.add('学习 JavaScript', 'high');
todo.add('完成练习', 'medium');
todo.toggle(1);
console.log(todo.list());
console.log(todo.getStats());
```

## 1.11 常见问答

### Q1: var、let、const 有什么区别？

| 特性 | var | let | const |
|------|-----|-----|-------|
| 作用域 | 函数作用域 | 块级作用域 | 块级作用域 |
| 变量提升 | 是 | 否 | 否 |
| 重复声明 | 允许 | 不允许 | 不允许 |
| 重新赋值 | 允许 | 允许 | 不允许 |

**推荐：** 优先用 const，需要修改时用 let，避免用 var。

### Q2: == 和 === 有什么区别？

```javascript
// == 会进行类型转换
5 == '5'      // true（字符串转数字）
0 == false    // true（布尔转数字）
'' == false   // true（空字符串转数字）
null == undefined  // true

// === 不会进行类型转换
5 === '5'    // false（类型不同）
0 === false  // false（类型不同）
```

**推荐：** 总是用 === 和 !==。

### Q3: 什么是闭包？有什么用途？

**闭包**是函数记住并访问其创建时的词法作用域。

**用途：**
- 数据私有化
- 函数柯里化
- 防抖节流
- 迭代器

```javascript
// 数据私有化示例
function createCounter() {
    let count = 0;  // 私有变量
    return {
        increment: () => ++count,
        getCount: () => count
    };
}
```
