## 实践项目

学完本模块后，建议完成以下项目巩固知识：

### 项目 1：简易计算器

```javascript
class Calculator {
    constructor() {
        this.history = [];
    }

    add(a, b) {
        return this._record(a + b, `${a} + ${b}`);
    }

    subtract(a, b) {
        return this._record(a - b, `${a} - ${b}`);
    }

    multiply(a, b) {
        return this._record(a * b, `${a} × ${b}`);
    }

    divide(a, b) {
        if (b === 0) {
            throw new Error('除数不能为零');
        }
        return this._record(a / b, `${a} ÷ ${b}`);
    }

    _record(result, expression) {
        this.history.push({ expression, result });
        return result;
    }

    getHistory() {
        return this.history;
    }
}
```

**涉及知识点：** 变量、函数、对象、错误处理

### 项目 2：待办事项管理

```javascript
class TodoManager {
    constructor() {
        this.todos = [];
        this.nextId = 1;
    }

    add(text, priority = 'medium') {
        const todo = {
            id: this.nextId++,
            text,
            priority,
            completed: false,
            createdAt: new Date()
        };
        this.todos.push(todo);
        return todo;
    }

    remove(id) {
        const index = this.todos.findIndex(t => t.id === id);
        if (index === -1) {
            throw new Error(`未找到 ID 为 ${id} 的待办事项`);
        }
        return this.todos.splice(index, 1)[0];
    }

    toggle(id) {
        const todo = this.todos.find(t => t.id === id);
        if (!todo) {
            throw new Error(`未找到 ID 为 ${id} 的待办事项`);
        }
        todo.completed = !todo.completed;
        return todo;
    }

    list(options = {}) {
        let result = [...this.todos];
        if (options.completed !== undefined) {
            result = result.filter(t => t.completed === options.completed);
        }
        if (options.priority) {
            result = result.filter(t => t.priority === options.priority);
        }
        return result;
    }

    getStats() {
        const total = this.todos.length;
        const completed = this.todos.filter(t => t.completed).length;
        return {
            total,
            completed,
            pending: total - completed,
            completionRate: total === 0 ? 0 : (completed / total * 100).toFixed(1) + '%'
        };
    }
}
```

**涉及知识点：** 数组、对象、函数、错误处理

## 常见问答

### Q1: var、let、const 有什么区别？

| 特性 | var | let | const |
|------|-----|-----|-------|
| 作用域 | 函数作用域 | 块级作用域 | 块级作用域 |
| 变量提升 | 是 | 否 | 否 |
| 重复声明 | 允许 | 不允许 | 不允许 |
| 重新赋值 | 允许 | 允许 | 不允许 |

**推荐：** 优先用 const，需要修改时用 let，避免用 var。

### Q2: == 和 === 有什么区别？

```javascript
// == 会进行类型转换
5 == '5'      // true（字符串转数字）
0 == false    // true（布尔转数字）

// === 不会进行类型转换
5 === '5'    // false（类型不同）
0 === false  // false（类型不同）
```

**推荐：** 总是用 === 和 !==。

### Q3: 什么是闭包？有什么用途？

**闭包**是函数记住并访问其创建时的词法作用域。

**用途：**
- 数据私有化
- 函数柯里化
- 防抖节流
- 迭代器

```javascript
// 数据私有化示例
function createCounter() {
    let count = 0;  // 私有变量
    return {
        increment: () => ++count,
        getCount: () => count
    };
}
```

```javascript
// 1. this 绑定不同
```
