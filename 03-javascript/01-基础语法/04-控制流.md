## 生活化比喻

**控制流就像"交通指挥系统"**：

```
比喻对应：

┌──────────────────────────────────────────────────────┐
│                  交通指挥                             │
├──────────────────────────────────────────────────────┤
│                                                      │
│    if-else = 红绿灯                                   │
│    ─────────────                                     │
│    绿灯 → 直行                                       │
│    红灯 → 停车                                       │
│    黄灯 → 减速准备                                   │
│    根据条件选择不同的道路                           │
│                                                      │
│    switch = 多岔路口                                  │
│    ─────────────                                     │
│    路牌指示：去 A 走左，去 B 走右，去 C 直走          │
│    一个入口，多个出口                                 │
│                                                      │
│    for 循环 = 跑圈                                    │
│    ─────────────                                     │
│    从起点出发 → 跑到第 N 圈 → 结束                   │
│    明确知道要跑多少圈                                 │
│                                                      │
│    while = 等公交                                     │
│    ─────────────                                     │
│    车没来就继续等 → 来了就上车                       │
│    不知道要等多久，但条件满足就停                     │
│                                                      │
│    break = 提前离场                                   │
│    ─────────────                                     │
│    事情做完了，不用继续了                             │
│                                                      │
│    continue = 跳过这一轮                              │
│    ─────────────                                     │
│    这一圈不算，直接进入下一圈                         │
│                                                      │
└──────────────────────────────────────────────────────┘
```

## L1 理解层：会用

### 条件语句

**语法结构图：**

```
if-else 结构：

if (条件) {
    // 条件为真时执行
} else if (条件2) {
    // 条件2 为真时执行
} else {
    // 以上都不满足时执行
}

switch 结构：

switch (值) {
    case 值1: 执行1; break;
    case 值2: 执行2; break;
    default:  默认执行;
}
```

**最简示例（1-3行）：**

```javascript
if (age >= 18) console.log('成年');
else console.log('未成年');

const grade = score >= 60 ? '及格' : '不及格';
```

**详细示例：**

```javascript
// if-else
const score = 85;
if (score >= 90) console.log('优秀');
else if (score >= 80) console.log('良好');
else if (score >= 60) console.log('及格');
else console.log('不及格');

// switch — 适合固定值的比较
const day = 3;
switch (day) {
    case 1: console.log('周一'); break;
    case 2: console.log('周二'); break;
    case 3: console.log('周三'); break;
    default: console.log('其他');
}

// 多个 case 合并
switch (month) {
    case 3: case 4: case 5: season = '春'; break;
    case 6: case 7: case 8: season = '夏'; break;
    default: season = '冬';
}
```

### for 循环

**语法结构图：**

```
for 循环结构：

for (初始值; 条件; 更新) {
    // 循环体
}

执行流程：
1. 执行初始值（只一次）
2. 检查条件 → 为假则结束
3. 执行循环体
4. 执行更新
5. 回到步骤 2
```

**最简示例：**

```javascript
for (let i = 0; i < 5; i++) console.log(i);  // 0, 1, 2, 3, 4
```

**详细示例：**

```javascript
// 基本 for
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// 遍历数组
const fruits = ['苹果', '香蕉', '橙子'];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// for...of — 遍历值（推荐）
for (const fruit of fruits) {
    console.log(fruit);
}

// for...in — 遍历键（对象用）
const user = { name: '张三', age: 25 };
for (const key in user) {
    console.log(key, user[key]);
}
```

### while / do-while

**最简示例：**

```javascript
let i = 0;
while (i < 3) { console.log(i); i++; }  // 0, 1, 2

let j = 0;
do { console.log(j); j++; } while (j < 3);  // 0, 1, 2
```

**详细示例：**

```javascript
// while — 先判断再执行
let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

// do-while — 至少执行一次
let input = '';
do {
    input = prompt('请输入内容：');
} while (!input);

// break — 跳出循环
for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i);  // 0, 1, 2, 3, 4
}

// continue — 跳过本次
for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) continue;
    console.log(i);  // 1, 3, 5, 7, 9
}
```

## L2 实践层：用好

### 条件语句选择

| 场景 | 推荐方案 | 原因 |
|------|---------|------|
| 2-3 个分支 | if-else | 简洁 |
| 多个固定值比较 | switch | 语义清晰 |
| 范围判断 | if-else if | switch 不支持范围 |
| 复杂条件 | if-else | switch 只支持相等比较 |
| 简单二选一 | 三元运算符 | 一行搞定 |

### 循环选择

| 场景 | 推荐方案 | 原因 |
|------|---------|------|
| 已知次数 | for | 明确计数 |
| 遍历数组值 | for...of | 简洁 |
| 遍历对象键 | for...in | 专用 |
| 条件满足就继续 | while | 不知道循环次数 |
| 至少执行一次 | do-while | 先执行再判断 |
| 数组方法可替代 | map/filter/reduce | 声明式更清晰 |

### 反模式：不要这样做

```javascript
// ❌ 错误：忘记 break（case 穿透）
switch (day) {
    case 1: console.log('周一');
    case 2: console.log('周二');  // 也会执行！
}

// ✅ 正确：每个 case 加 break
switch (day) {
    case 1: console.log('周一'); break;
    case 2: console.log('周二'); break;
}
```

```javascript
// ❌ 错误：用 for...in 遍历数组
for (const i in [1, 2, 3]) {
    console.log(typeof i);  // 'string'（索引是字符串）
}

// ✅ 正确：用 for...of
for (const value of [1, 2, 3]) {
    console.log(value);  // 1, 2, 3
}
```

```javascript
// ❌ 错误：用循环做可以用数组方法做的事
const doubled = [];
for (let i = 0; i < arr.length; i++) {
    doubled.push(arr[i] * 2);
}

// ✅ 正确：用 map
const doubled = arr.map(x => x * 2);
```

## L3 专家层：深入

### 跳出多层循环

```javascript
// 方法 1：标签（label）
outer: for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 3; j++) {
        if (i === 1 && j === 1) break outer;
        console.log(i, j);
    }
}

// 方法 2：函数返回（更推荐）
function findElement() {
    for (let i = 0; i < 3; i++) {
        for (let j = 0; j < 3; j++) {
            if (condition) return { i, j };
        }
    }
    return null;
}
```

### 性能考量

| 操作 | 性能影响 | 说明 |
|------|---------|------|
| for 循环 | 极低 | 最快的遍历方式 |
| for...of | 低 | 比 for 稍慢，但更清晰 |
| for...in | 中等 | 会遍历原型链上的属性 |
| forEach | 中等 | 函数调用开销 |
| map/filter | 中等 | 创建新数组 |

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **控制流** | 代码执行的顺序和路径 | if、for、while 等 |
| **分支** | 根据条件选择不同执行路径 | if-else、switch |
| **循环** | 重复执行一段代码 | for、while |
| **break** | 立即跳出循环或 switch | `if (found) break;` |
| **continue** | 跳过本次循环，进入下一次 | `if (skip) continue;` |
| **for...of** | 遍历可迭代对象的值 | `for (const x of arr)` |
| **for...in** | 遍历对象的可枚举属性键 | `for (const k in obj)` |
| **case 穿透** | switch 中忘记 break 导致继续执行下一个 case | 常见 bug |

## 实践练习

### 练习：控制流应用

```javascript
// 练习 1：斐波那契数列
function fibonacci(n) {
    let a = 0, b = 1;
    for (let i = 0; i < n; i++) {
        console.log(a);
        [a, b] = [b, a + b];
    }
}
fibonacci(7);  // 0, 1, 1, 2, 3, 5, 8

// 练习 2：找出数组最大值
function findMax(arr) {
    let max = arr[0];
    for (let i = 1; i < arr.length; i++) {
        if (arr[i] > max) max = arr[i];
    }
    return max;
}
console.log(findMax([3, 7, 2, 9, 1]));  // 9

// 练习 3：99 乘法表
for (let i = 1; i <= 9; i++) {
    let line = '';
    for (let j = 1; j <= i; j++) {
        line += `${j}×${i}=${i * j}\t`;
    }
    console.log(line);
}
```

## 常见问题

### Q1：for...in 和 for...of 有什么区别？

```javascript
const arr = ['a', 'b', 'c'];

for (const i in arr) console.log(i);      // '0', '1', '2'（索引，字符串）
for (const v of arr) console.log(v);      // 'a', 'b', 'c'（值）
```
**规则：遍历数组用 for...of，遍历对象用 for...in。**

### Q2：while 和 do-while 用哪个？

**看是否需要至少执行一次：**

```javascript
// 可能一次都不执行 → while
while (hasNext()) { processNext(); }

// 至少执行一次 → do-while
do { retry(); } while (shouldRetry());
```

### Q3：如何简化冗长的 if-else？

```javascript
// ❌ 冗长
if (type === 'a') handleA();
else if (type === 'b') handleB();
else if (type === 'c') handleC();

// ✅ 用对象映射
const handlers = { a: handleA, b: handleB, c: handleC };
handlers[type]?.();
```
