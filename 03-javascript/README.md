## 常用代码片段

### 数组操作

```javascript
// map - 映射
const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2);  // [2, 4, 6]

// filter - 过滤
const evens = nums.filter(n => n % 2 === 0);  // [2]

// reduce - 归并
const sum = nums.reduce((acc, n) => acc + n, 0);  // 6

// find - 查找
const found = nums.find(n => n > 1);  // 2

// some/every - 判断
const hasEven = nums.some(n => n % 2 === 0);  // true
const allPositive = nums.every(n => n > 0);   // true
```

### 对象操作

```javascript
// 对象解构
const user = { name: '张三', age: 25 };
const { name, age } = user;

// 展开运算符
const newObj = { ...user, city: '北京' };

// Object 方法
const keys = Object.keys(user);      // ['name', 'age']
const values = Object.values(user);  // ['张三', 25]
const entries = Object.entries(user); // [['name', '张三'], ['age', 25]]
```

### 事件处理

```javascript
// 添加事件监听器
element.addEventListener('click', (e) => {
    console.log('点击了', e.target);
});

// 事件委托
document.querySelector('ul').addEventListener('click', (e) => {
    if (e.target.tagName === 'LI') {
        console.log('点击了列表项', e.target.textContent);
    }
});

// 防抖函数
function debounce(fn, delay) {
    let timer = null;
    return function(...args) {
        clearTimeout(timer);
        timer = setTimeout(() => fn.apply(this, args), delay);
    };
}
```

## 最佳实践

### ✅ 推荐做法

```javascript
// 1. 使用 let/const 代替 var
const MAX_COUNT = 100;  // 常量
let count = 0;          // 变量
// var name = '张三';   // ❌ 避免使用

// 2. 使用模板字符串
const name = '张三';
console.log(`你好，${name}`);  // ✅
console.log('你好，' + name);  // ❌

// 3. 使用箭头函数
const add = (a, b) => a + b;

// 4. 使用解构赋值
const user = { name: '张三', age: 25 };
const { name, age } = user;

// 5. 使用展开运算符
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];

// 6. 使用 async/await
async function fetchData() {
    try {
        const response = await fetch('/api/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('请求失败:', error);
    }
}

// 7. 使用可选链
const city = user?.address?.city;

// 8. 使用空值合并运算符
const name = userName ?? '匿名用户';
```

### ❌ 避免做法

```javascript
// 1. 避免使用 var
var name = '张三';  // ❌
let name = '张三';  // ✅

// 2. 避免使用 == 比较
if (x == 0) {}  // ❌
if (x === 0) {} // ✅

// 3. 避免回调地狱
getData(() => {
    getMoreData(() => {
        getEvenMoreData(() => {
            // 嵌套过深 ❌
        });
    });
});
// ✅ 使用 Promise 链或 async/await

// 4. 避免直接修改原型
Array.prototype.myMethod = () => {};  // ❌

// 5. 避免忽略错误
fetch('/api').then(res => res.json());  // ❌ 没有 catch
fetch('/api').then(res => res.json()).catch(console.error);  // ✅

// 6. 避免内存泄漏
// 及时清理事件监听器和定时器
```

## 常见问题

### Q1: var、let、const 的区别？

**答：**

| 特性 | var | let | const |
|------|-----|-----|-------|
| 作用域 | 函数作用域 | 块级作用域 | 块级作用域 |
| 变量提升 | 是 | 否（TDZ） | 否（TDZ） |
| 重复声明 | 允许 | 不允许 | 不允许 |
| 重新赋值 | 允许 | 允许 | 不允许 |

---

### Q2: 箭头函数和普通函数的区别？

**答：**

1. **this 绑定**：箭头函数的 this 是词法作用域，普通函数的 this 是动态绑定
2. **arguments**：箭头函数没有 arguments，需要使用剩余参数
3. **构造函数**：箭头函数不能作为构造函数
4. **yield**：箭头函数不能用作生成器

---

### Q3: 什么是闭包？有什么用？

**答：** 闭包是函数和函数创建时的词法环境的组合。

用途：
- 数据私有化
- 函数柯里化
- 防抖节流
- 迭代器
