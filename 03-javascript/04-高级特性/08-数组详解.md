## 生活化比喻

**数组就像"一排储物柜"**：

```
比喻对应：

┌──────────────────────────────────────────────────────┐
│                  一排储物柜                           │
├──────────────────────────────────────────────────────┤
│                                                      │
│    索引 = 柜子编号                                    │
│    ─────────────                                     │
│    从 0 开始编号，每个位置对应一个柜子                │
│    arr[0] → 第 1 个柜子                              │
│                                                      │
│    push/pop = 从最右边放/拿东西                       │
│    ─────────────────                                 │
│    push：往队尾放东西（返回新长度）                   │
│    pop：从队尾拿走最后一个（返回拿走的东西）          │
│                                                      │
│    unshift/shift = 从最左边放/拿东西                  │
│    ─────────────────                                 │
│    unshift：往队头塞东西（后面的全要挪位置）          │
│    shift：从队头拿走第一个（后面的全要挪位置）        │
│                                                      │
│    map = 给每个柜子换标签                            │
│    ─────────────                                     │
│    原来的柜子不动，生成一排新柜子                     │
│    每个柜子里的内容经过同样的处理                     │
│                                                      │
│    filter = 筛选符合条件的柜子                       │
│    ─────────────                                     │
│    只保留满足条件的柜子，生成新一排                   │
│                                                      │
│    reduce = 把所有柜子里的东西汇总成一个               │
│    ─────────────                                     │
│    从左到右，逐个合并，最后得到一个结果               │
│                                                      │
└──────────────────────────────────────────────────────┘
```

## L1 理解层：会用

### 数组基础

**语法结构图：**

```
创建数组：

字面量（推荐）:  let arr = [1, 2, 3];
Array.of:       let arr = Array.of(1, 2, 3);   ← 避免 new Array 的陷阱
Array.from:     let arr = Array.from('hello'); ← 从可迭代对象创建

访问元素:
arr[0]          → 第一个元素
arr.at(-1)      → 最后一个（ES2022）
arr.length      → 数组长度

判断是否数组:
Array.isArray(arr)  → true/false
```

**最简示例（1-3行）：**

```javascript
let arr = [1, 2, 3];
arr[0];            // 1
Array.isArray(arr); // true
```

**详细示例：**

```javascript
// 创建
const numbers = [1, 2, 3, 4, 5];
const mixed = [1, 'hello', true, null, { name: 'obj' }];
const range = Array.from({ length: 5 }, (_, i) => i);  // [0, 1, 2, 3, 4]

// 访问
numbers[0];        // 1
numbers.at(-1);    // 5（ES2022）
numbers.length;    // 5

// 修改
numbers[0] = 10;   // [10, 2, 3, 4, 5]
numbers.push(6);   // [10, 2, 3, 4, 5, 6]，返回 6
numbers.pop();     // 返回 6，数组变回 [...5]

// 遍历
for (const item of numbers) console.log(item);
numbers.forEach((item, i) => console.log(i, item));
```

### 核心方法：map / filter / reduce

**语法结构图：**

```
数据处理三件套：

map    → 转换每个元素，返回新数组
  [1,2,3].map(x => x * 2)  → [2, 4, 6]

filter → 筛选符合条件的元素，返回新数组
  [1,2,3,4].filter(x => x % 2 === 0)  → [2, 4]

reduce → 归约为单个值
  [1,2,3].reduce((acc, x) => acc + x, 0)  → 6

链式调用：
  data.filter(...).map(...).reduce(...)
```

**最简示例：**

```javascript
[1,2,3].map(x => x * 2);          // [2, 4, 6]
[1,2,3,4].filter(x => x > 2);     // [3, 4]
[1,2,3].reduce((a, b) => a + b);  // 6
```

**详细示例：**

```javascript
const users = [
    { name: '张三', age: 25, score: 80 },
    { name: '李四', age: 17, score: 90 },
    { name: '王五', age: 30, score: 70 }
];

// map：提取属性
const names = users.map(u => u.name);  // ['张三', '李四', '王五']

// filter：筛选
const adults = users.filter(u => u.age >= 18);  // 张三、王五

// reduce：统计
const totalScore = users.reduce((sum, u) => sum + u.score, 0);  // 240

// 链式：数据处理管道
const adultNames = users
    .filter(u => u.age >= 18)
    .sort((a, b) => b.score - a.score)
    .map(u => u.name);
// ['张三', '王五']
```

### 增删操作

**语法结构图：**

```
修改原数组的方法：

push(elem)     末尾添加    → 返回新长度
pop()          末尾删除    → 返回删除的元素
unshift(elem)  开头添加    → 返回新长度（慢）
shift()        开头删除    → 返回删除的元素（慢）
splice(start, count, ...items)  → 任意位置增删改

不修改原数组的方法：

slice(start, end)   → 返回截取的新数组
concat(...arrays)   → 返回合并的新数组
[...arr1, ...arr2]  → 展开运算符合并（推荐）
```

**最简示例：**

```javascript
let arr = [1, 2, 3];
arr.push(4);       // [1, 2, 3, 4]
arr.slice(0, 2);   // [1, 2]（原数组不变）
[...arr, 5, 6];    // [1, 2, 3, 4, 5, 6]
```

## L2 实践层：用好

### 方法速查表

| 方法 | 修改原数组？ | 返回值 | 场景 |
|------|:---:|------|------|
| `push/pop` | ✅ | 长度/元素 | 栈操作 |
| `unshift/shift` | ✅ | 长度/元素 | 队列操作（大数组慢） |
| `splice` | ✅ | 删除的元素 | 任意位置增删改 |
| `slice` | ❌ | 新数组 | 安全截取 |
| `map` | ❌ | 新数组 | 数据转换 |
| `filter` | ❌ | 新数组 | 数据筛选 |
| `reduce` | ❌ | 归约值 | 求和、分组、扁平化 |
| `find/findIndex` | ❌ | 元素/索引 | 查找第一个匹配 |
| `some/every` | ❌ | boolean | 条件检测 |
| `sort` | ✅ | 原数组 | 排序（原地修改） |
| `toSorted`（ES2023）| ❌ | 新数组 | 不修改原数组的排序 |

**反模式：不要这样做**

```javascript
// ❌ 错误：sort 默认按字符串排序
[10, 5, 2, 1].sort();  // [1, 10, 2, 5]

// ✅ 正确：提供比较函数
[10, 5, 2, 1].sort((a, b) => a - b);  // [1, 2, 5, 10]
```

```javascript
// ❌ 错误：forEach 中尝试 return（无效）
const doubled = [1, 2, 3].forEach(x => x * 2);
console.log(doubled);  // undefined

// ✅ 正确：用 map
const doubled = [1, 2, 3].map(x => x * 2);  // [2, 4, 6]
```

```javascript
// ❌ 错误：在 map/filter 中修改外部变量
let sum = 0;
const result = arr.map(x => { sum += x; return x; });

// ✅ 正确：用 reduce
const sum = arr.reduce((acc, x) => acc + x, 0);
```

### 适用场景

| 场景 | 推荐方法 | 原因 |
|------|---------|------|
| 转换所有元素 | `map` | 返回等长新数组 |
| 筛选子集 | `filter` | 返回符合条件的元素 |
| 求和/分组/统计 | `reduce` | 灵活归约 |
| 查找单个元素 | `find` | 找到即停 |
| 检测是否有匹配 | `some` | 找到一个就返回 true |
| 检测是否全匹配 | `every` | 一个不匹配就返回 false |
| 不可变更新 | `[...arr]` / `slice` | 不修改原数组 |

## L3 专家层：深入

### 数组方法底层原理

```
map/filter/reduce 执行模型：

map([1,2,3], fn):
  创建新数组 [_, _, _]
  → fn(1, 0, arr) → 放入位置 0
  → fn(2, 1, arr) → 放入位置 1
  → fn(3, 2, arr) → 放入位置 2
  返回 [结果1, 结果2, 结果3]

filter([1,2,3,4], fn):
  创建空数组 []
  → fn(1) 为 true → push(1)
  → fn(2) 为 false → 跳过
  → fn(3) 为 true → push(3)
  → fn(4) 为 false → 跳过
  返回 [1, 3]

reduce([1,2,3], fn, 0):
  acc = 0
  → fn(0, 1) → acc = 1
  → fn(1, 2) → acc = 3
  → fn(3, 3) → acc = 6
  返回 6
```

### 性能考量

| 操作 | 时间复杂度 | 说明 |
|------|:---:|------|
| 索引访问 `arr[i]` | O(1) | 直接定位 |
| `push/pop` | O(1) | 队尾操作 |
| `unshift/shift` | O(n) | 需移动所有元素 |
| `splice` | O(n) | 需移动后续元素 |
| `slice` | O(k) | k 为截取长度 |
| `map/filter` | O(n) | 遍历一次 |
| `reduce` | O(n) | 遍历一次 |
| `sort` | O(n log n) | 快速排序/归并排序 |
| `includes/indexOf` | O(n) | 线性查找 |

### 设计动机

**为什么 JavaScript 数组方法这样设计？**

| 设计选择 | 原因 | 对比其他语言 |
|----------|------|-------------|
| 方法返回新数组 | 支持链式调用 | Python 用函数式内置函数 |
| sort 原地修改 | 节省内存 | Python sorted() 返回新列表 |
| reduce 通用性 | 可替代 map/filter | 但 map/filter 语义更清晰 |
| 回调三参数 | 提供元素、索引、原数组 | 比只提供元素更灵活 |

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **数组** | 有序的数值索引集合 | `[1, 2, 3]` |
| **索引** | 元素在数组中的位置，从 0 开始 | `arr[0]` |
| **原地修改** | 方法直接改变原数组 | `push`、`splice`、`sort` |
| **纯方法** | 不修改原数组，返回新数组 | `map`、`filter`、`slice` |
| **map** | 对每个元素执行函数，返回新数组 | `[1,2].map(x=>x*2)` → `[2,4]` |
| **filter** | 筛选符合条件的元素 | `[1,2,3].filter(x=>x>1)` → `[2,3]` |
| **reduce** | 将数组归约为单个值 | `[1,2,3].reduce((a,b)=>a+b)` → `6` |
| **链式调用** | 多个方法连续调用 | `arr.filter(...).map(...)` |
| **解构** | 从数组中提取值赋给变量 | `const [a, b] = [1, 2]` |
| **扁平化** | 将嵌套数组变为一维 | `[[1,2],[3]].flat()` → `[1,2,3]` |

## 实践练习

### 练习：数据处理管道 + 方法实现

```javascript
// 练习 1：电商数据处理
const products = [
    { name: '手机', price: 3999, category: '数码', stock: 50 },
    { name: '电脑', price: 8999, category: '数码', stock: 0 },
    { name: '书', price: 49, category: '图书', stock: 200 },
    { name: '耳机', price: 299, category: '数码', stock: 100 }
];

// 有库存的数码产品，按价格升序，返回名称和价格
const result = products
    .filter(p => p.stock > 0 && p.category === '数码')
    .sort((a, b) => a.price - b.price)
    .map(p => ({ name: p.name, price: p.price }));
console.log(result);
// [{ name: '耳机', price: 299 }, { name: '手机', price: 3999 }]

// 练习 2：实现数组去重
function unique(arr) {
    return [...new Set(arr)];
}
console.log(unique([1, 2, 2, 3, 3, 3]));  // [1, 2, 3]

// 练习 3：手动实现 map
function myMap(arr, fn) {
    const result = [];
    for (let i = 0; i < arr.length; i++) {
        result.push(fn(arr[i], i, arr));
    }
    return result;
}
console.log(myMap([1, 2, 3], x => x * 2));  // [2, 4, 6]
```

## 常见问题

### Q1：map 和 forEach 有什么区别？

**map 返回新数组，forEach 不返回值：**

```javascript
[1, 2, 3].map(x => x * 2);        // [2, 4, 6]
[1, 2, 3].forEach(x => x * 2);    // undefined

// 需要新数组 → map
// 只做副作用（打印、请求）→ forEach
```

### Q2：splice 和 slice 有什么区别？

```javascript
const arr = [1, 2, 3, 4, 5];

arr.slice(1, 3);    // [2, 3]（原数组不变）
console.log(arr);   // [1, 2, 3, 4, 5]

arr.splice(1, 2);   // [2, 3]（原数组被修改）
console.log(arr);   // [1, 4, 5]
```

### Q3：如何不修改原数组进行排序？

```javascript
// ES2023：toSorted（不修改原数组）
const sorted = [3, 1, 2].toSorted((a, b) => a - b);

// 兼容写法：先拷贝再排序
const sorted = [...arr].sort((a, b) => a - b);
```
