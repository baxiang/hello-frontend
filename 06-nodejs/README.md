# Node.js 模块化学习路线

## 章节划分

### 第 1 章：Node.js 基础
- Node.js 简介和环境配置
- 全局对象和模块系统
- 核心模块（path、fs、events、util）
- CommonJS 和 ES Module

### 第 2 章：HTTP 与 Web 开发
- 原生 HTTP 模块
- Express 框架基础
- 路由和中间件
- RESTful API 设计

### 第 3 章：数据库集成
- MongoDB 和 Mongoose
- MySQL 和 mysql2
- 数据库连接池
- ORM 基础

### 第 4 章：用户认证与安全
- 密码加密（bcrypt）
- JWT 令牌认证
- 认证中间件
- 安全最佳实践

### 第 5 章：实用技术
- 文件上传（multer）
- 日志记录
- 错误处理
- 环境变量配置

### 第 6 章：项目实战
- 博客 API 项目
- 用户管理系统
- 完整的 CRUD 操作
- 项目部署基础

---

## 学习建议

1. **按顺序学习** - 章节之间有依赖关系
2. **动手实践** - 每章都有代码示例，务必亲手练习
3. **理解原理** - 不仅要会用，还要理解背后的原理
4. **项目驱动** - 学完后尝试做完整的项目

---

## 开发环境准备

### 必备工具
- **Node.js**: [下载地址](https://nodejs.org/) (建议 LTS 版本)
- **npm/yarn**: 包管理工具
- **Postman**: API 测试工具
- **MongoDB/MySQL**: 数据库

### VS Code 推荐插件
- ESLint - 代码检查
- Prettier - 代码格式化
- Thunder Client - API 测试
- MongoDB for VS Code - 数据库管理
