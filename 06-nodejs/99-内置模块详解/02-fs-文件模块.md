# fs 文件模块

> 读取、写入、删除文件和目录

---

## 学习目标

- 掌握 fs.promises 异步文件操作
- 学会读写文件和目录遍历
- 理解流式读写大文件

---

## L1 理解层

**最简示例（1-3行）：**

```javascript
const fs = require('fs').promises;
const content = await fs.readFile('file.txt', 'utf-8');
await fs.writeFile('output.txt', 'Hello');
```

**详细示例：**

```javascript
const fs = require('fs').promises;
const path = require('path');

// 读取
const text = await fs.readFile('file.txt', 'utf-8');

// 写入（覆盖）
await fs.writeFile('out.txt', 'Hello World');

// 追加
await fs.appendFile('log.txt', 'new line\n');

// 目录
await fs.mkdir('./new-dir', { recursive: true });
const files = await fs.readdir('./src');

// 删除
await fs.unlink('old.txt');
await fs.rm('./dir', { recursive: true });

// 文件信息
const stat = await fs.stat('file.txt');
stat.isFile();       // true
stat.isDirectory();  // false
stat.size;           // 字节
stat.mtime;          // 修改时间
```

## L2 实践层

### 异步 vs 同步

| 方式 | 方法 | 特点 |
|------|------|------|
| 异步（推荐） | `fs.promises.readFile()` | 不阻塞 |
| 同步 | `fs.readFileSync()` | 阻塞，仅用于启动时 |

### 反模式

```javascript
// ❌ 错误：同步读取阻塞主线程
const data = fs.readFileSync('large.json');

// ✅ 正确：异步
const data = await fs.readFile('large.json', 'utf-8');
```

## L3 专家层

### 流式读写（大文件）

```javascript
const fs = require('fs');

// 读取流
const readStream = fs.createReadStream('large.txt', 'utf-8');
readStream.on('data', chunk => console.log(chunk));
readStream.on('end', () => console.log('完成'));

// 写入流
const writeStream = fs.createWriteStream('out.txt');
writeStream.write('line1\n');
writeStream.end();
```
