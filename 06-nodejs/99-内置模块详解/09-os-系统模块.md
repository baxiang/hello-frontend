# os 系统模块

> 获取操作系统信息

---

## L1 理解层

**最简示例：**

```javascript
const os = require('os');
os.platform();    // 'darwin' / 'linux' / 'win32'
os.cpus().length; // CPU 核心数
```

**详细示例：**

```javascript
const os = require('os');

os.platform();      // 操作系统平台
os.arch();          // CPU 架构
os.hostname();      // 主机名
os.homedir();       // 用户目录
os.tmpdir();        // 临时目录
os.cpus();          // CPU 信息
os.totalmem();      // 总内存（字节）
os.freemem();       // 可用内存
os.uptime();        // 运行时间（秒）
os.loadavg();       // 系统负载
```
