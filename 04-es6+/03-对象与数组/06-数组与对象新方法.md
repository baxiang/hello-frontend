# ES6+ 数组与对象新方法 ⭐

> find、flat、Object.fromEntries 等现代 API

---

## 学习目标

- 掌握 ES6+ 新增的数组方法（find、includes、flat、flatMap）
- 掌握 Object 新方法（fromEntries、values、entries）
- 了解 ES2023 数组新方法（findLast、toSorted、toReversed）

---

## L1 理解层：会用

### 数组新方法

**最简示例：**

```javascript
[1,2,3].find(x => x > 1);          // 2（第一个匹配）
[1,2,3].includes(2);               // true
[1,[2,[3]]].flat(2);               // [1,2,3]
['a b','c d'].flatMap(s => s.split(' '));  // ['a','b','c','d']
```

**详细示例：**

```javascript
// find / findIndex — 查找
const users = [{ id: 1, name: '张三' }, { id: 2, name: '李四' }];
users.find(u => u.id === 2);       // { id: 2, name: '李四' }
users.findIndex(u => u.id === 2);  // 1

// includes — 判断包含（比 indexOf 更好，能正确处理 NaN）
[1, 2, NaN].includes(NaN);  // true（indexOf 返回 -1）

// flat — 扁平化
[1, [2, [3, [4]]]].flat(2);         // [1, 2, 3, [4]]
[1, [2, [3, [4]]]].flat(Infinity);  // [1, 2, 3, 4]

// flatMap — 先 map 再 flat（一次遍历）
['hello world', 'foo bar'].flatMap(s => s.split(' '));
// ['hello', 'world', 'foo', 'bar']

// ES2023: findLast / findLastIndex
[1, 2, 3, 2].findLast(x => x === 2);       // 2（最后一个）
[1, 2, 3, 2].findLastIndex(x => x === 2);  // 3
```

### Object 新方法

**最简示例：**

```javascript
Object.entries({ a: 1, b: 2 });   // [['a',1], ['b',2]]
Object.fromEntries([['a',1]]);    // { a: 1 }
Object.values({ a: 1, b: 2 });    // [1, 2]
```

**详细示例：**

```javascript
const obj = { name: '张三', age: 25, city: '北京' };

// Object.entries — 键值对数组
Object.entries(obj);  // [['name','张三'], ['age',25], ['city','北京']]

// Object.fromEntries — 键值对数组转对象
const entries = [['name', '李四'], ['age', 30]];
Object.fromEntries(entries);  // { name: '李四', age: 30 }

// 实用：过滤对象属性
const filtered = Object.fromEntries(
    Object.entries(obj).filter(([k, v]) => v > 20)
);
// { age: 25 }

// URL 参数转对象
const params = new URLSearchParams('name=张三&age=25');
Object.fromEntries(params);  // { name: '张三', age: '25' }
```

## L2 实践层：用好

### 方法选择指南

| 需求 | 推荐方法 | 示例 |
|------|---------|------|
| 查找第一个匹配 | `find` | `arr.find(x => x > 5)` |
| 查找最后一个匹配 | `findLast`（ES2023） | `arr.findLast(x => x > 5)` |
| 判断是否包含 | `includes` | `arr.includes(3)` |
| 嵌套数组展平 | `flat` | `arr.flat(Infinity)` |
| map + 展平一步 | `flatMap` | `arr.flatMap(fn)` |
| 对象遍历 | `Object.entries` | `for (const [k,v] of Object.entries(obj))` |
| 对象转 Map | `new Map(Object.entries(obj))` | |
| Map 转对象 | `Object.fromEntries(map)` | |

### 反模式：不要这样做

```javascript
// ❌ 错误：用 indexOf 判断包含（NaN 判断错误）
[1, 2, NaN].indexOf(NaN) !== -1;  // false

// ✅ 正确：用 includes
[1, 2, NaN].includes(NaN);  // true
```

```javascript
// ❌ 错误：手动递归扁平化
function flatten(arr) {
    let result = [];
    for (const item of arr) {
        if (Array.isArray(item)) result = result.concat(flatten(item));
        else result.push(item);
    }
    return result;
}

// ✅ 正确：用 flat
arr.flat(Infinity);
```

## L3 专家层：深入

### 不修改原数组的新方法（ES2023）

```javascript
// 传统方法修改原数组
const arr = [3, 1, 2];
arr.sort((a, b) => a - b);  // arr 被修改

// ES2023：返回新数组
arr.toSorted((a, b) => a - b);  // [1, 2, 3]，arr 不变
arr.toReversed();                // [3, 2, 1]
arr.toSpliced(1, 1);             // [3, 2]
arr.with(0, 99);                 // [99, 1, 2]
```

## 术语定义

| 术语 | 定义 | 示例 |
|------|------|------|
| **find** | 返回第一个满足条件的元素 | `[1,2,3].find(x => x > 1)` → 2 |
| **includes** | 判断数组是否包含某值 | `[1,2].includes(2)` → true |
| **flat** | 将嵌套数组展平指定层数 | `[1,[2]].flat()` → `[1,2]` |
| **flatMap** | 先 map 再 flat（一层） | `['a b'].flatMap(s => s.split(' '))` |
| **Object.fromEntries** | 键值对数组转对象 | `Object.fromEntries([['a',1]])` → `{a:1}` |

## 实践练习

```javascript
// 练习 1：数据处理管道
const data = [
    { name: '张三', scores: [80, 90, 70] },
    { name: '李四', scores: [60, 55, 85] }
];

// 找出所有及格分数
const passingScores = data
    .flatMap(d => d.scores)
    .filter(s => s >= 60);
// [80, 90, 70, 60, 85]

// 练习 2：对象属性过滤
const config = { theme: 'dark', lang: 'zh', debug: false, version: '1.0' };
const stringValues = Object.fromEntries(
    Object.entries(config).filter(([_, v]) => typeof v === 'string')
);
// { theme: 'dark', lang: 'zh', version: '1.0' }
```
