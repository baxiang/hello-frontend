# 解构赋值与展开运算符

> 从零开始理解 ES6 的解构赋值和展开运算符

## 学习目标

- ✅ 掌握数组和对象的解构赋值
- ✅ 理解嵌套解构和默认值
- ✅ 掌握展开运算符的使用
- ✅ 能够使用参数解构

---

## 3.0 为什么要学解构和展开？

### 3.0.1 故事背景

在 ES6 之前，从数组或对象中提取值需要写很多重复代码：

```javascript
// 从对象中提取值
const user = { name: '张三', age: 25, city: '北京' };

// ES5 写法
const name = user.name;
const age = user.age;
const city = user.city;

// 从数组中提取值
const point = [10, 20];

// ES5 写法
const x = point[0];
const y = point[1];
```

这种写法：
1. 非常冗长
2. 容易出错
3. 可读性差

### 3.0.2 解构和展开解决的问题

```
┌─────────────────────────────────────────────────────────────┐
│                    解构与展开的作用                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  解构 = 从数组/对象中快速提取值                              │
│  → 一行代码提取多个值                                        │
│  → 代码更简洁、更易读                                        │
│                                                             │
│  展开 = 将数组/对象"展开"铺平                               │
│  → 快速合并数组/对象                                         │
│  → 快速复制数据                                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.0.3 本章学习路径

```
第一步：理解解构的概念（什么是解构）
    ↓
第二步：学习数组解构（从数组提取值）
    ↓
第三步：学习对象解构（从对象提取值）
    ↓
第四步：学习嵌套解构（复杂数据结构）
    ↓
第五步：学习展开运算符（合并和复制）
    ↓
第六步：掌握实际应用场景
```

## 3.1 数组解构详解

> 从数组中提取值的最简洁方式

### 3.1.1 基础语法

```javascript
// 传统方式：一个一个拿
const arr = [1, 2, 3];
const first = arr[0];
const second = arr[1];
const third = arr[2];

// 解构方式：一行搞定
const [first, second, third] = [1, 2, 3];

console.log(first, second, third);  // 1, 2, 3
```

**语法图解：**

```
┌─────────────────────────────────────────────────────────────┐
│  const [a, b, c] = [1, 2, 3];                               │
│                                                             │
│  数组 [1, 2, 3]                                            │
│  │  │  │                                                    │
│  ▼  ▼  ▼                                                    │
│  a  b  c                                                    │
│  │  │  └── 第三个值 → c = 3                                │
│  │  └────── 第二个值 → b = 2                               │
│  └───────── 第一个值 → a = 1                               │
└─────────────────────────────────────────────────────────────┘
```

### 3.1.2 跳过某些元素

```javascript
// 跳过中间的元素
const [first, , third] = [1, 2, 3, 4, 5];
console.log(first, third);  // 1, 3
//      ↑  ↑
//      │  └── 跳过索引 1（值 2）
//      └───── 索引 0（值 1）
```

### 3.1.3 只取部分元素

```javascript
// 只取第一个
const [head] = [1, 2, 3, 4, 5];
console.log(head);  // 1

// 只取前两个
const [a, b] = [1, 2, 3, 4, 5];
console.log(a, b);  // 1, 2
```

### 3.1.4 剩余模式（...rest）

```javascript
// 使用 ... 收集剩余元素
const [first, ...rest] = [1, 2, 3, 4, 5];
console.log(first);  // 1
console.log(rest);   // [2, 3, 4, 5]

// 实际应用：分离头部和尾部
const [head, ...tail] = [1, 2, 3, 4, 5];
console.log(`头部: ${head}, 尾部: ${tail.join(', ')}`);
// 头部: 1, 尾部: 2, 3, 4, 5
```

### 3.1.5 默认值

```javascript
// 当对应位置的值为 undefined 时，使用默认值
const [a = 1, b = 2, c = 3] = [10];
console.log(a, b, c);  // 10, 2, 3
//           ↑  ↑  ↑
//           │  │  └── c = 3（默认值，因为原数组没有第三个元素）
//           │  └───── b = 2（默认值，因为原数组第二个元素是 undefined）
//           └──────── a = 10（使用实际值）

// null 不会触发默认值
const [x = 'default'] = [null];
console.log(x);  // null（null 不是 undefined，不会触发默认值）
```

### 3.1.6 交换变量

```javascript
// 传统方式：需要临时变量
let a = 1;
let b = 2;
let temp = a;
a = b;
b = temp;

// 解构方式：一行搞定
let c = 1;
let d = 2;
[c, d] = [d, c];

console.log(c, d);  // 2, 1
```

## 3.2 对象解构详解

> 从对象中提取值的最简洁方式

### 3.2.1 基础语法

```javascript
// 传统方式
const user = { name: '张三', age: 25, city: '北京' };
const name = user.name;
const age = user.age;
const city = user.city;

// 解构方式
const { name, age, city } = { name: '张三', age: 25, city: '北京' };

console.log(name, age, city);  // '张三', 25, '北京'
```

**语法图解：**

```
┌─────────────────────────────────────────────────────────────┐
│  const { name, age } = { name: '张三', age: 25 };           │
│                                                             │
│  对象 { name: '张三', age: 25 }                             │
│       │        │                                            │
│       ▼        ▼                                            │
│      name     age                                           │
│       │        │                                            │
│       ▼        ▼                                            │
│    '张三'    25                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2.2 变量重命名

```javascript
// 变量名与属性名不同时，使用冒号重命名
const { name: userName, age: userAge } = { name: '张三', age: 25 };

console.log(userName);  // '张三'
console.log(userAge);   // 25
// console.log(name);  // ReferenceError: name is not defined
```

**使用场景：避免命名冲突**

```javascript
// 当外部已有同名变量时
const name = '外部变量';

const user = { name: '用户变量' };

// 使用重命名避免冲突
const { name: userName } = user;
console.log(userName);  // '用户变量'
console.log(name);      // '外部变量'（不受影响）
```

### 3.2.3 默认值

```javascript
// 属性不存在或为 undefined 时使用默认值
const { name = '匿名用户', age = 0, city = '未知' } = { age: 25 };

console.log(name);   // '匿名用户'（默认值）
console.log(age);    // 25（使用实际值）
console.log(city);   // '未知'（默认值）
```

### 3.2.4 剩余模式

```javascript
// 使用 ... 收集剩余属性
const { name, ...others } = { name: '张三', age: 25, city: '北京', email: 'test@example.com' };

console.log(name);   // '张三'
console.log(others); // { age: 25, city: '北京', email: 'test@example.com' }
```

## 3.3 嵌套解构

> 处理复杂的数据结构

### 3.3.1 对象嵌套解构

```javascript
const user = {
    name: '张三',
    profile: {
        age: 25,
        address: {
            city: '北京',
            district: '朝阳区'
        }
    }
};

// 嵌套解构
const {
    name,
    profile: {
        age,
        address: { city }
    }
} = user;

console.log(name);   // '张三'
console.log(age);    // 25
console.log(city);   // '北京'
```

**逐步分解：**

```
user 对象结构：
{
    name: '张三',
    profile: {
        age: 25,
        address: {
            city: '北京',
            district: '朝阳区'
        }
    }
}

解构过程：
1. 从 user 中提取 profile
2. 从 profile 中提取 age 和 address
3. 从 address 中提取 city
```

### 3.3.2 数组嵌套解构

```javascript
const data = [1, [2, [3, [4]]]];

// 嵌套解构
const [a, [b, [c, [d]]]] = data;

console.log(a, b, c, d);  // 1, 2, 3, 4
```

### 3.3.3 混合嵌套

```javascript
const config = {
    server: {
        hosts: ['host1', 'host2', 'host3'],
        port: 8080,
        settings: {
            timeout: 30
        }
    }
};

// 混合解构
const {
    server: {
        hosts: [primary, ...backups],
        port,
        settings: { timeout }
    }
} = config;

console.log(primary);   // 'host1'
console.log(backups);   // ['host2', 'host3']
console.log(port);      // 8080
console.log(timeout);   // 30
```

## 3.4 展开运算符详解

> 用三个点（...）展开数组或对象

### 3.4.1 什么是展开运算符？

```
┌─────────────────────────────────────────────────────────────┐
│  展开运算符 ... 的作用：                                     │
│                                                             │
│  数组展开：把 [1, 2, 3] 变成 1, 2, 3（三个独立的值）         │
│  对象展开：把 { a: 1 } 变成 a: 1（属性展开）                 │
│                                                             │
│  就像把包装盒打开，把里面的东西拿出来                         │
└─────────────────────────────────────────────────────────────┘
```

### 3.4.2 数组展开

**合并数组：**

```javascript
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

// 传统方式
const merged1 = arr1.concat(arr2);

// 展开方式
const merged2 = [...arr1, ...arr2];

console.log(merged2);  // [1, 2, 3, 4, 5, 6]
```

**在数组中插入：**

```javascript
const arr = [1, 2, 3];

// 在开头插入
const withStart = [-1, 0, ...arr];
console.log(withStart);  // [-1, 0, 1, 2, 3]

// 在中间插入
const withMiddle = [1, ...arr, 4];
console.log(withMiddle);  // [1, 2, 3, 4]

// 在末尾插入
const withEnd = [...arr, 4, 5];
console.log(withEnd);  // [1, 2, 3, 4, 5]
```

**复制数组（浅拷贝）：**

```javascript
const original = [1, 2, 3];

// 传统方式
const copy1 = original.slice();

// 展开方式
const copy2 = [...original];

console.log(copy2);  // [1, 2, 3]
console.log(original === copy2);  // false（不同的数组）
```

### 3.4.3 对象展开

**合并对象：**

```javascript
const defaults = { theme: 'light', lang: 'zh', timeout: 5000 };
const userPrefs = { theme: 'dark' };

// 传统方式
const merged1 = Object.assign({}, defaults, userPrefs);

// 展开方式
const merged2 = { ...defaults, ...userPrefs };

console.log(merged2);  // { theme: 'dark', lang: 'zh', timeout: 5000 }
```

**属性覆盖规则：**

```javascript
const obj1 = { a: 1, b: 1 };
const obj2 = { b: 2, c: 2 };

// 后面的属性会覆盖前面的
const merged = { ...obj1, ...obj2 };
console.log(merged);  // { a: 1, b: 2, c: 2 }
//                 ↑    ↑
//                 │    └── obj2 的 b 覆盖 obj1 的 b
//                 └────── obj1 的 a
```

**复制对象（浅拷贝）：**

```javascript
const original = { a: 1, b: { c: 2 } };

// 浅拷贝
const copy = { ...original };

console.log(copy);  // { a: 1, b: { c: 2 } }
console.log(original === copy);  // false
// 注意：嵌套对象是引用，不是拷贝！
copy.b.c = 3;
console.log(original.b.c);  // 3（original 也被修改了）
```

### 3.4.4 函数调用展开

```javascript
function sum(a, b, c) {
    return a + b + c;
}

const args = [1, 2, 3];

// 传统方式
console.log(sum.apply(null, args));  // 6

// 展开方式
console.log(...args);  // 1 2 3（展开为 1, 2, 3）
console.log(sum(...args));  // 6
```

## 3.5 参数解构

> 在函数参数中使用解构

### 3.5.1 参数对象解构

```javascript
// 传统方式
function greet(user) {
    const name = user.name;
    const age = user.age;
    console.log(`你好，我叫${name}，今年${age}岁`);
}

// 参数解构
function greet({ name, age }) {
    console.log(`你好，我叫${name}，今年${age}岁`);
}

greet({ name: '张三', age: 25 });
```

### 3.5.2 参数数组解构

```javascript
function sum([a, b, c]) {
    return a + b + c;
}

console.log(sum([1, 2, 3]));  // 6
```

### 3.5.3 带默认值的参数解构

```javascript
function createUser({ name = '匿名', age = 0, city = '未知' }) {
    return { name, age, city };
}

console.log(createUser({ name: '张三' }));
// { name: '张三', age: 0, city: '未知' }

console.log(createUser({}));
// { name: '匿名', age: 0, city: '未知' }
```

## 3.6 实际应用场景

### 3.6.1 API 响应处理

```javascript
// 模拟 API 响应
const apiResponse = {
    status: 'success',
    data: {
        user: {
            id: 1,
            name: '张三',
            email: 'zhang@example.com',
            profile: {
                avatar: '/avatar.jpg',
                bio: '你好'
            }
        }
    },
    timestamp: '2024-01-01'
};

// 使用解构提取需要的数据
const {
    status,
    data: {
        user: { name, email }
    }
} = apiResponse;

console.log(status);  // 'success'
console.log(name);    // '张三'
console.log(email);   // 'zhang@example.com'
```

### 3.6.2 函数返回值

```javascript
// 函数返回多个值
function getStats() {
    const min = 1;
    const max = 100;
    const avg = 50;
    
    return { min, max, avg };
}

const { min, max, avg } = getStats();
console.log(min, max, avg);  // 1, 100, 50
```

### 3.6.3 交换变量

```javascript
let x = 10;
let y = 20;

// 一行交换
[x, y] = [y, x];

console.log(x, y);  // 20, 10
```

const person = { name: '李四', age: 30, city: '上海' };

### 练习 3：展开运算符

## 3.8 常见问答

### Q1: 解构和展开有什么区别？

```javascript
// 解构：从数组/对象中提取值
const [a, b] = [1, 2];  // a=1, b=2

// 展开：将数组/对象展开
const arr = [1, 2, 3];
console.log(...arr);  // 1 2 3

// 记忆技巧：
// 解构 = 提取（从右到左）
// 展开 = 铺平（从左到右）
```

### Q2: 什么时候用解构？

```javascript
// 1. 从函数返回的多个值中提取
const { min, max } = getRange();

// 2. 从 API 响应中提取数据
const { data: { user } } = await fetchUser(id);

// 3. 处理配置对象
const { debug = false, timeout = 5000 } = config;
```

### Q3: 展开运算符和 Object.assign 哪个好？

```javascript
// 功能类似，但展开运算符更简洁
const obj1 = Object.assign({}, defaults, userPrefs);
const obj2 = { ...defaults, ...userPrefs };

// 推荐使用展开运算符（更简洁、更易读）
```
