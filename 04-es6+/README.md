- let 和 const 的基本用法
- 块级作用域概念
- 暂时性死区（TDZ）
- var vs let vs const 对比
- 实践练习

### 第 2 章：箭头函数
- 箭头函数语法
- this 绑定规则
- 箭头函数与普通函数区别
- 使用场景和注意事项
- 实践练习

### 第 3 章：解构赋值与展开运算符
- 数组解构
- 对象解构
- 嵌套解构
- 展开运算符（数组/对象）
- 参数解构
- 实践练习

### 第 4 章：模板字符串与 Symbol
- 模板字符串基础
- 标签模板
- Symbol 类型
- 内置 Symbol
- 实践练习

### 第 5 章：Map、Set 与 WeakMap
- Map 数据结构
- Set 数据结构
- WeakMap 和 WeakSet
- 应用场景
- 实践练习

### 第 6 章：Class 类与继承
- class 基础语法
- 静态方法
- getter 和 setter
- 继承（extends/super）
- 实践练习

### 第 7 章：模块化
- ES6 Module 语法
- export/import
- 默认导出

- 实践练习

### 第 8 章：Promise 与 async/await
- Promise 基础
- Promise 链式调用
- Promise 组合（all/race/allSettled/any）
- async/await 语法
- 错误处理
- 实践练习

### 第 9 章：ES2017-ES2022 新特性
- ES2017: Object.values/entries, padStart/padEnd

- ES2020: Optional chaining, Nullish coalescing, BigInt

- ES2022: 顶层 await, 私有字段，Array.at()
- 实践练习

---

## 学习建议

1. **按顺序学习** - 章节之间有依赖关系
2. **每章练习** - 完成每章的实践代码
3. **对比学习** - 对比 ES5 和 ES6+ 写法差异
4. **实际应用** - 在项目中尝试使用新特性
