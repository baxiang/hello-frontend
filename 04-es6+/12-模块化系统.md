## 12.0 为什么要模块化？

### 12.0.1 故事背景

在 ES6 之前，JavaScript 没有官方模块系统：

```javascript
// 全局变量污染
var name = '张三';
var name = '李四';  // 覆盖！
```

社区出现了各种模块方案：
- CommonJS（Node.js）
- AMD
- CMD

ES6 引入了官方模块系统：ES6 Module。

## 12.1 export 导出详解

### 12.1.1 具名导出

```javascript
// 方式1：声明时导出
export const PI = 3.14159;
export function add(a, b) {
    return a + b;
}

// 方式2：先声明，后导出
function subtract(a, b) {
    return a - b;
}
export { subtract };

// 方式3：重命名导出
function multiply(a, b) {
    return a * b;
}
export { multiply as mul };
```

### 12.1.2 默认导出

```javascript
// 每个文件只能有一个默认导出
export default function() {
    // ...
}

// 或
function createApp() { }
export default createApp;
```

### 12.1.3 混合导出

```javascript
// 混合使用
export const PI = 3.14159;
export function add(a, b) { return a + b; }

export default function multiply(a, b) {
    return a * b;
}
```

## 12.2 import 导入详解

### 12.2.1 具名导入

```javascript
// 从模块导入
import { PI, add } from './math.js';

console.log(PI);   // 3.14159
console.log(add(1, 2));  // 3

// 重命名导入
import { add as sum } from './math.js';
console.log(sum(1, 2));  // 3

// 导入所有
import * as math from './math.js';
console.log(math.PI);  // 3.14159
console.log(math.add(1, 2));  // 3
```

### 12.2.2 默认导入

```javascript
// 导入默认导出
import myFunction from './calculator.js';
myFunction();

// 混合导入
import defaultExport, { PI, add } from './math.js';
```

## 12.3 动态导入

### 12.3.1 import()

```javascript
// 动态导入（按需加载）
button.addEventListener('click', async () => {
    const { default: Modal } = await import('./Modal.js');
    new Modal().show();
});
```

### 12.3.2 使用场景

```javascript
// 1. 路由懒加载
const Home = () => import('./Home.js');
const About = () => import('./About.js');

// 2. 条件加载
if (condition) {
    const module = await import('./heavy-module.js');
}
```

## 12.4 CommonJS vs ES6 Module

### 12.4.1 对比表

| 特性 | CommonJS | ES6 Module |
|------|----------|-------------|
| 语法 | require/exports | import/export |
| 加载 | 同步 | 异步 |
| 运行时 | 运行时 | 编译时 |
| this | 当前模块 | undefined |
| 循环依赖 | 支持 | 支持 |

### 12.4.2 代码对比

```javascript
// CommonJS
const { PI, add } = require('./math.js');
module.exports = { PI, add };

// ES6 Module
import { PI, add } from './math.js';
export { PI, add };
```

## 12.5 常见问答

### Q1: 默认导出和具名导出哪个好？

**答：** 根据场景选择。

```javascript
// 工具函数：具名导出
export function add(a, b) { return a + b; }

// 类或函数式组件：默认导出
export default function App() { }
```

### Q2: 什么时候用动态导入？

```javascript
// 大模块按需加载
const heavyModule = await import('./heavy.js');

// 条件加载
if (isMobile) {
    const mobileUI = await import('./mobile.js');
}
```
