# 箭头函数

## 学习目标
- 掌握箭头函数的语法
- 理解箭头函数的 this 绑定
- 了解箭头函数与普通函数的区别
- 能够正确使用箭头函数

---

## 2.1 箭头函数语法

### 基础语法

```javascript
// 普通函数
function add(a, b) {
    return a + b;
}

// 箭头函数
const add = (a, b) => {
    return a + b;
};

// 简写形式（隐式返回）
const add = (a, b) => a + b;
```

### 各种语法形式

```javascript
// 无参数
const sayHi = () => console.log('Hi');

// 单个参数（可省略括号）
const double = x => x * 2;

// 多个参数（需要括号）
const add = (a, b) => a + b;

// 返回对象（需要括号）
const getUser = () => ({ name: '张三', age: 25 });

// 多语句（需要花括号和 return）
const process = (data) => {
    const result = data.map(item => item * 2);
    return result;
};

// 异步箭头函数
const fetchData = async (url) => {
    const response = await fetch(url);
    return response.json();
};
```

## 2.2 this 绑定

### 箭头函数的 this

```javascript
// 箭头函数的 this 指向定义时的上下文
// 而不是调用时的上下文

const obj = {
    name: '张三',
    // 普通方法
    regularFn() {
        console.log(this.name);  // '张三'（this 指向 obj）
    },
    // 箭头函数
    arrowFn: () => {
        console.log(this.name);  // undefined（this 指向 window）
    }
};

obj.regularFn();  // '张三'
obj.arrowFn();    // undefined
```

### 经典应用场景

```javascript
// 场景 1：定时器
function Timer() {
    this.seconds = 0;

    // 普通函数（this 指向 window，错误）
    // setInterval(function() {
    //     this.seconds++;  // this 不是 Timer
    // }, 1000);

    // 箭头函数（this 指向 Timer 实例，正确）
    setInterval(() => {
        this.seconds++;
    }, 1000);
}

const timer = new Timer();
// timer.seconds 会不断增加

// 场景 2：数组回调
const numbers = [1, 2, 3, 4, 5];

// 普通函数
const squared1 = numbers.filter(function(n) {
    return n * n;
});

// 箭头函数（更简洁）
const squared2 = numbers.filter(n => n * n);
```

### this 陷阱

```javascript
// 陷阱 1：箭头函数不能作为方法
const obj = {
    name: '张三',
    // 错误：箭头函数没有自己的 this
    // getName: () => this.name
};

// 正确做法：使用普通函数
const obj = {
    name: '张三',
    getName() {
        return this.name;
    }
};

// 陷阱 2：嵌套箭头函数
const outer = {
    name: '外层',
    inner: {
        name: '内层',
        getName: () => {
            // 这里的 this 指向 window，不是 inner
            return this.name;
        }
    }
};
```

## 2.3 箭头函数 vs 普通函数

### 核心区别

| 特性 | 普通函数 | 箭头函数 |
|------|----------|----------|
| this 绑定 | 动态绑定 | 词法绑定 |
| 可作为构造函数 | ✅ | ❌ |
| arguments 对象 | ✅ | ❌ |
| 可用作生成器 | ✅ | ❌ |
| 有 prototype | ✅ | ❌ |

### 不能使用箭头函数的场景

```javascript
// 1. 对象方法（需要动态 this）
const obj = {
    name: '张三',
    // 错误
    // getName: () => this.name,
    // 正确
    getName() {
        return this.name;
    }
};

// 2. 构造函数（需要 new）
// const Person = (name) => { this.name = name; };  // 错误
const Person = function(name) {
    this.name = name;
};

// 3. 需要 arguments 对象
// const sum = () => arguments.reduce((a, b) => a + b, 0);  // 错误
const sum = function() {
    return [...arguments].reduce((a, b) => a + b, 0);
};

// 使用 rest 参数替代 arguments
const sumArrow = (...args) => args.reduce((a, b) => a + b, 0);

// 4. 事件处理器（需要动态 this）
const button = {
    element: document.querySelector('button'),
    clickCount: 0,
    // 错误：箭头函数的 this 不是 button
    // init() {
    //     this.element.addEventListener('click', () => {
    //         this.clickCount++;  // this 不是 button
    //     });
    // }
    // 正确
    init() {
        this.element.addEventListener('click', (e) => {
            this.clickCount++;  // this 是 button（词法作用域）
        });
    }
};
```

## 2.4 实用技巧

### 链式调用

```javascript
const numbers = [1, 2, 3, 4, 5];

// 普通函数链式调用
const result1 = numbers
    .filter(function(n) { return n % 2 === 0; })
    .map(function(n) { return n * 2; })
    .reduce(function(sum, n) { return sum + n; }, 0);

// 箭头函数（更简洁）
const result2 = numbers
    .filter(n => n % 2 === 0)
    .map(n => n * 2)
    .reduce((sum, n) => sum + n, 0);
```

### 函数组合

```javascript
// 箭头函数适合函数式编程
const compose = (...fns) => x => fns.reduceRight((acc, fn) => fn(acc), x);

const add1 = x => x + 1;
const double = x => x * 2;
const add1ThenDouble = compose(double, add1);

console.log(add1ThenDouble(3));  // 8
```

### 事件处理

```javascript
// Vue/React 中的箭头函数
class Counter {
    constructor() {
        this.count = 0;
        // 箭头函数自动绑定 this
        this.increment = () => {
            this.count++;
        };
    }
}

// React 组件中的常见模式
class Button extends React.Component {
    // 类属性语法（需要 Babel）
    handleClick = () => {
        console.log(this.props.onClick);
    };

    render() {
        return <button onClick={this.handleClick}>点击</button>;
    }
}
```

## 2.5 实践练习

### 练习 1：数组操作

```javascript
const users = [
    { name: '张三', age: 25, city: '北京' },
    { name: '李四', age: 30, city: '上海' },
    { name: '王五', age: 25, city: '广州' },
    { name: '赵六', age: 35, city: '深圳' }
];

// 1. 筛选年龄大于 25 的用户
const adults = users.filter(user => user.age > 25);

// 2. 提取所有用户名
const names = users.map(user => user.name);

// 3. 按城市分组
const byCity = users.reduce((acc, user) => {
    if (!acc[user.city]) {
        acc[user.city] = [];
    }
    acc[user.city].push(user);
    return acc;
}, {});

// 4. 计算平均年龄
const averageAge = users.reduce((sum, user) => sum + user.age, 0) / users.length;
```

### 练习 2：Promise 链

```javascript
// 使用箭头函数简化 Promise 链
function getUser(userId) {
    return fetch(`/api/users/${userId}`)
        .then(response => response.json())
        .then(user => user);
}

function getPosts(user) {
    return fetch(`/api/posts?userId=${user.id}`)
        .then(response => response.json())
        .then(posts => posts);
}

// 链式调用
getUser(1)
    .then(user => getPosts(user))
    .then(posts => {
        posts.forEach(post => console.log(post.title));
    })
    .catch(error => console.error(error));

// async/await + 箭头函数
const loadUserData = async (userId) => {
    try {
        const user = await getUser(userId);
        const posts = await getPosts(user);
        return { user, posts };
    } catch (error) {
        console.error(error);
        throw error;
    }
};
```

### 练习 3：柯里化

```javascript
// 箭头函数实现柯里化
const curry = fn => {
    const curried = (...args) => {
        if (args.length >= fn.length) {
            return fn(...args);
        }
        return (...more) => curried(...args, ...more);
    };
    return curried;
};

// 使用示例
const add = (a, b, c) => a + b + c;
const curriedAdd = curry(add);

console.log(curriedAdd(1, 2, 3));    // 6
console.log(curriedAdd(1, 2)(3));    // 6
console.log(curriedAdd(1)(2, 3));    // 6
console.log(curriedAdd(1)(2)(3));    // 6
```

## 2.6 常见问答

### Q1: 箭头函数的 this 到底是什么？

**答：** 箭头函数没有自己的 this，它捕获的是定义时所在上下文的 this 值。

```javascript
// this 继承自外层
const obj = {
    name: '外层',
    method: function() {
        const arrow = () => {
            console.log(this.name);  // this 是 method 的 this
        };
        arrow();
    }
};

obj.method();  // '外层'
```

### Q2: 箭头函数可以用 new 调用吗？

**答：** 不可以。箭头函数没有 [[Construct]] 内部方法，也没有 prototype。

```javascript
const Person = (name) => {
    this.name = name;
};

// const p = new Person('张三');  // TypeError: Person is not a constructor
```

### Q3: 箭头函数有 arguments 对象吗？

**答：** 没有。使用 rest 参数替代。

```javascript
// 错误
// const fn = () => arguments[0];

// 正确
const fn = (...args) => args[0];
```

### Q4: 如何判断一个函数是箭头函数？

```javascript
// 箭头函数没有 prototype
const arrow = () => {};
const normal = function() {};

console.log(arrow.prototype);   // undefined
console.log(normal.prototype);  // {}

// 使用 toString
console.log(arrow.toString());  // "() => {}"
console.log(normal.toString()); // "function() {}"
```
