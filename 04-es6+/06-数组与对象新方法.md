## 6.0 通俗理解

### 这些方法是什么？

```
ES6+ 新方法 = 更方便的数组/对象操作

就像：
- 以前：手动遍历、过滤、映射
- 现在：内置方法一行搞定
```

### 常用方法一览

| 方法 | 作用 | 通俗理解 |
|------|------|----------|
| map | 映射 | 每个元素加工 |
| filter | 过滤 | 筛选符合条件的 |
| reduce | 汇总 | 聚合成一个值 |

| 可选链 ?. | 安全访问 | 深层属性不怕空 |
| 空值合并 ?? | 默认值 | null/undefined 用默认值 |

## 6.1 Array.from 与 Array.of

### Array.from()

将类数组对象或可迭代对象转换为真正的数组。

```javascript
// 从类数组对象创建数组
const arrayLike = {
    length: 3,
    0: 'a',
    1: 'b',
    2: 'c'
};
const arr = Array.from(arrayLike);
console.log(arr);  // ['a', 'b', 'c']

// 从字符串创建数组
const str = 'hello';
console.log(Array.from(str));  // ['h', 'e', 'l', 'l', 'o']

// 从 Set 创建数组
const set = new Set([1, 2, 3]);
console.log(Array.from(set));  // [1, 2, 3]

// 从 Map 创建数组
const map = new Map([['a', 1], ['b', 2]]);
console.log(Array.from(map));  // [['a', 1], ['b', 2]]

// 第二个参数：映射函数
const numbers = Array.from({ length: 5 }, (_, i) => i * 2);
console.log(numbers);  // [0, 2, 4, 6, 8]

// 实用场景：生成指定范围的数组
const range = (start, end) => Array.from({ length: end - start + 1 }, (_, i) => start + i);
console.log(range(1, 5));  // [1, 2, 3, 4, 5]
```

### Array.of()

将一组值转换为数组，弥补 `Array()` 构造函数的不足。

```javascript
// Array() 构造函数的问题
console.log(Array(3));      // [empty × 3] - 长度为 3 的空数组
console.log(Array(1, 2));   // [1, 2]

// Array.of() 行为一致
console.log(Array.of(3));     // [3] - 单个元素
console.log(Array.of(1, 2));  // [1, 2]

// 实用场景：创建单元素数组
const singleElement = Array.of(undefined);
console.log(singleElement);  // [undefined]
```

## 6.2 查找方法：find 与 findIndex

### find()

返回数组中第一个满足条件的元素。

```javascript
const users = [
    { id: 1, name: '张三', age: 25 },
    { id: 2, name: '李四', age: 30 },
    { id: 3, name: '王五', age: 35 }
];

// 查找年龄大于 28 的用户
const user = users.find(u => u.age > 28);
console.log(user);  // { id: 2, name: '李四', age: 30 }

// 查找不存在的元素
const notFound = users.find(u => u.age > 100);
console.log(notFound);  // undefined

// 第二个参数：thisArg
const finder = {
    minAge: 30,
    checkAge(user) {
        return user.age >= this.minAge;
    }
};
const result = users.find(finder.checkAge, finder);
console.log(result);  // { id: 2, name: '李四', age: 30 }
```

### findIndex()

返回数组中第一个满足条件的元素索引。

```javascript
const numbers = [1, 5, 10, 15, 20];

// 查找第一个大于 10 的元素索引
const index = numbers.findIndex(n => n > 10);
console.log(index);  // 3

// 查找不存在的元素
const notFound = numbers.findIndex(n => n > 100);
console.log(notFound);  // -1

// 实用场景：查找对象在数组中的位置
const userId = users.findIndex(u => u.id === 2);
console.log(userId);  // 1
```

### findLast 与 findLastIndex (ES2023)

从后向前查找。

```javascript
const numbers = [1, 5, 10, 15, 10, 20];

// 从后向前查找第一个大于 10 的元素
const last = numbers.findLast(n => n > 10);
console.log(last);  // 20

// 从后向前查找的索引
const lastIndex = numbers.findLastIndex(n => n === 10);
console.log(lastIndex);  // 4
```

## 6.3 includes()

判断数组是否包含某个值。

```javascript
const arr = [1, 2, 3, NaN, Infinity];

// 基本用法
console.log(arr.includes(2));      // true
console.log(arr.includes(100));    // false

// 能正确判断 NaN（对比 indexOf）
console.log(arr.includes(NaN));    // true
console.log(arr.indexOf(NaN));     // -1（无法判断）

// 第二个参数：起始索引
console.log(arr.includes(3, 2));   // true（从索引 2 开始）
console.log(arr.includes(1, 2));   // false

// 负数索引（从后往前数）
console.log(arr.includes(Infinity, -1));  // true

// 实用场景：权限检查
const permissions = ['read', 'write'];
const hasPermission = (action) => permissions.includes(action);
console.log(hasPermission('read'));   // true
console.log(hasPermission('delete')); // false
```

## 6.4 entries()、keys()、values()

### 数组的遍历方法

```javascript
const arr = ['a', 'b', 'c'];

// keys() - 遍历索引
for (const key of arr.keys()) {
    console.log(key);  // 0, 1, 2
}

// values() - 遍历值
for (const value of arr.values()) {
    console.log(value);  // 'a', 'b', 'c'
}

// entries() - 遍历 [索引，值] 对
for (const [index, value] of arr.entries()) {
    console.log(`${index}: ${value}`);  // 0: a, 1: b, 2: c
}

// 转换为数组
console.log([...arr.keys()]);     // [0, 1, 2]
console.log([...arr.values()]);   // ['a', 'b', 'c']
console.log([...arr.entries()]);  // [[0, 'a'], [1, 'b'], [2, 'c']]
```

## 6.5 flat() 与 flatMap()

### flat()

数组扁平化，将嵌套数组展平。

```javascript
// 基本用法
const arr1 = [1, [2, 3], [4, [5, 6]]];
console.log(arr1.flat());        // [1, 2, 3, 4, [5, 6]] - 默认展平一层
console.log(arr1.flat(2));       // [1, 2, 3, 4, 5, 6] - 展平两层

// 无限展平
const arr2 = [1, [2, [3, [4, [5]]]]];
console.log(arr2.flat(Infinity));  // [1, 2, 3, 4, 5]

// 处理空位
const arr3 = [1, 2, , 4, 5];
console.log(arr3.flat());  // [1, 2, 4, 5] - 空位被移除

// 实用场景：处理接口返回的嵌套数据
const apiResponse = {
    data: [
        { items: [1, 2] },
        { items: [3, 4] },
        { items: [5, 6] }
    ]
};
const allItems = apiResponse.data.map(item => item.items).flat();
console.log(allItems);  // [1, 2, 3, 4, 5, 6]
```

### flatMap()

先 map 再 flat（只展平一层）。

```javascript
const arr = ['hello', 'world', 'foo', 'bar'];

// 将每个字符串拆分为字符数组，然后扁平化
const result = arr.flatMap(s => s.split(''));
console.log(result);  // ['h', 'e', 'l', 'l', 'o', 'w', 'o', 'r', 'l', 'd', 'f', 'o', 'o', 'b', 'a', 'r']

// 实用场景：过滤并转换
const sentences = ['Hello world', 'JavaScript is fun', ''];
const words = sentences.flatMap(s => s.split(' '));
console.log(words);  // ['Hello', 'world', 'JavaScript', 'is', 'fun']

// 对比：使用 map + flat
const result2 = arr.map(s => s.split('')).flat();
console.log(result2);  // 结果相同，但 flatMap 更高效
```

## 6.6 at()

使用负数索引访问数组元素。

```javascript
const arr = ['a', 'b', 'c', 'd', 'e'];

// 正数索引（与传统方式相同）
console.log(arr.at(0));   // 'a'
console.log(arr.at(2));   // 'c'

// 负数索引（从后往前数）
console.log(arr.at(-1));  // 'e'（最后一个）
console.log(arr.at(-2));  // 'd'（倒数第二个）

// 替代传统写法
console.log(arr[arr.length - 1]);  // 'e'（传统方式）
console.log(arr.at(-1));           // 'e'（现代方式，更简洁）

// 实用场景：获取最后一个元素
const getLastItem = (arr) => arr.at(-1);
console.log(getLastItem([1, 2, 3]));  // 3

// 实用场景：获取数组的最后 N 个元素
const getLastN = (arr, n) => arr.slice(-n);
console.log(getLastN(arr, 2));  // ['d', 'e']
```

## 6.7 ES2023 数组新方法

### toSorted()

排序并返回新数组（不修改原数组）。

```javascript
const numbers = [3, 1, 4, 1, 5, 9, 2, 6];

// sort() 会修改原数组
const sorted1 = numbers.sort((a, b) => a - b);
console.log(numbers);  // [1, 1, 2, 3, 4, 5, 6, 9] - 原数组被修改

// toSorted() 不修改原数组
const numbers2 = [3, 1, 4, 1, 5, 9, 2, 6];
const sorted2 = numbers2.toSorted((a, b) => a - b);
console.log(numbers2);  // [3, 1, 4, 1, 5, 9, 2, 6] - 原数组不变
console.log(sorted2);   // [1, 1, 2, 3, 4, 5, 6, 9]
```

### toReversed()

反转并返回新数组。

```javascript
const arr = [1, 2, 3, 4, 5];

// reverse() 会修改原数组
const reversed1 = arr.reverse();
console.log(arr);  // [5, 4, 3, 2, 1] - 原数组被修改

// toReversed() 不修改原数组
const arr2 = [1, 2, 3, 4, 5];
const reversed2 = arr2.toReversed();
console.log(arr2);      // [1, 2, 3, 4, 5] - 原数组不变
console.log(reversed2); // [5, 4, 3, 2, 1]
```

### toSpliced()

 splice 的不修改原数组版本。

```javascript
const arr = [1, 2, 3, 4, 5];

// splice() 会修改原数组
const removed = arr.splice(2, 2, 'a', 'b');
console.log(arr);     // [1, 2, 'a', 'b', 5]
console.log(removed); // [3, 4]

// toSpliced() 不修改原数组
const arr2 = [1, 2, 3, 4, 5];
const spliced = arr2.toSpliced(2, 2, 'a', 'b');
console.log(arr2);    // [1, 2, 3, 4, 5] - 原数组不变
console.log(spliced); // [1, 2, 'a', 'b', 5]

// 只删除
console.log(arr2.toSpliced(1, 2));  // [1, 4, 5]

// 只添加
console.log(arr2.toSpliced(1, 0, 'x', 'y'));  // [1, 'x', 'y', 2, 3, 4, 5]
```

### with()

替换指定位置的元素并返回新数组。

```javascript
const arr = [1, 2, 3, 4, 5];

// 传统方式会修改原数组
arr[2] = 100;
console.log(arr);  // [1, 2, 100, 4, 5]

// with() 不修改原数组
const arr2 = [1, 2, 3, 4, 5];
const replaced = arr2.with(2, 100);
console.log(arr2);     // [1, 2, 3, 4, 5] - 原数组不变
console.log(replaced); // [1, 2, 100, 4, 5]

// 支持负数索引
console.log(arr2.with(-1, 999));  // [1, 2, 3, 4, 999]

// 实用场景：不可变更新
const updateItem = (arr, index, value) => arr.with(index, value);
```

## 6.8 Object.assign()

将源对象的所有可枚举属性复制到目标对象。

```javascript
// 基本用法
const target = { a: 1 };
const source = { b: 2, c: 3 };
const result = Object.assign(target, source);
console.log(result);  // { a: 1, b: 2, c: 3 }
console.log(target);  // { a: 1, b: 2, c: 3 } - target 被修改

// 多个源对象
const result2 = Object.assign({}, { a: 1 }, { b: 2 }, { c: 3 });
console.log(result2);  // { a: 1, b: 2, c: 3 }

// 后面的属性覆盖前面的
const result3 = Object.assign(
    { a: 1, b: 1 },
    { b: 2, c: 2 },
    { c: 3, d: 3 }
);
console.log(result3);  // { a: 1, b: 2, c: 3, d: 3 }

// 实用场景：对象浅拷贝
const original = { name: '张三', age: 25 };
const copy = Object.assign({}, original);
console.log(copy);  // { name: '张三', age: 25 }

// 实用场景：合并配置
const defaults = { theme: 'light', lang: 'zh' };
const userConfig = { theme: 'dark' };
const config = Object.assign({}, defaults, userConfig);
console.log(config);  // { theme: 'dark', lang: 'zh' }

// 注意：浅拷贝
const nested = { user: { name: '张三' } };
const shallowCopy = Object.assign({}, nested);
shallowCopy.user.name = '李四';
console.log(nested.user.name);  // '李四' - 原对象也被修改了
```

## 6.9 Object.entries() 与 Object.values()

### Object.entries()

返回对象自身的可枚举属性键值对数组。

```javascript
const obj = { a: 1, b: 2, c: 3 };

// 获取键值对数组
console.log(Object.entries(obj));
// [['a', 1], ['b', 2], ['c', 3]]

// 遍历对象
for (const [key, value] of Object.entries(obj)) {
    console.log(`${key}: ${value}`);
}
// a: 1
// b: 2
// c: 3

// 与 Object.fromEntries() 配合使用
const obj2 = Object.fromEntries(Object.entries(obj).filter(([k, v]) => v !== 2));
console.log(obj2);  // { a: 1, c: 3 }

// 实用场景：对象转 Map
const map = new Map(Object.entries(obj));
console.log(map);  // Map(3) { 'a' => 1, 'b' => 2, 'c' => 3 }
```

### Object.values()

返回对象自身的可枚举属性值数组。

```javascript
const obj = { a: 1, b: 2, c: 3 };

// 获取值数组
console.log(Object.values(obj));  // [1, 2, 3]

// 实用场景：计算对象值的总和
const prices = { apple: 3, banana: 2, orange: 4 };
const total = Object.values(prices).reduce((sum, price) => sum + price, 0);
console.log(total);  // 9

// 实用场景：检查对象是否有重复值
const hasDuplicateValues = (obj) => {
    const values = Object.values(obj);
    return values.length !== new Set(values).size;
};
console.log(hasDuplicateValues({ a: 1, b: 2 }));     // false
console.log(hasDuplicateValues({ a: 1, b: 1 }));     // true
```

## 6.10 Object.fromEntries()

将键值对数组转换为对象，是 `Object.entries()` 的逆操作。

```javascript
// 基本用法
const entries = [['a', 1], ['b', 2], ['c', 3]];
const obj = Object.fromEntries(entries);
console.log(obj);  // { a: 1, b: 2, c: 3 }

// 与 Object.entries() 配合
const original = { apple: 3, banana: 2, cherry: 5 };
const doubled = Object.fromEntries(
    Object.entries(original).map(([key, value]) => [key, value * 2])
);
console.log(doubled);  // { apple: 6, banana: 4, cherry: 10 }

// 实用场景：过滤对象属性
const user = { name: '张三', age: 25, password: 'secret', email: 'zhang@example.com' };
const safeUser = Object.fromEntries(
    Object.entries(user).filter(([key]) => key !== 'password')
);
console.log(safeUser);  // { name: '张三', age: 25, email: 'zhang@example.com' }

// 实用场景：Map 转对象
const map = new Map([['name', '李四'], ['age', 30]]);
const obj2 = Object.fromEntries(map);
console.log(obj2);  // { name: '李四', age: 30 }
```

## 6.11 Object.hasOwn() (ES2022)

检查对象是否有自有属性（不检查原型链）。

```javascript
const obj = {
    name: '张三',
    age: 25
};

// 设置原型属性
Object.prototype.inherited = '继承的属性';

// 传统方式：in 操作符（会检查原型链）
console.log('name' in obj);       // true
console.log('inherited' in obj);  // true（检查到原型链）

// 传统方式：hasOwnProperty（需要从 Object.prototype 调用）
console.log(obj.hasOwnProperty('name'));       // true
console.log(obj.hasOwnProperty('inherited'));  // false

// 问题：如果 obj 没有原型或覆盖了 hasOwnProperty
const nullProto = Object.create(null);
nullProto.name = '张三';
// nullProto.hasOwnProperty('name');  // TypeError: nullProto.hasOwnProperty is not a function

// 现代方式：Object.hasOwn()
console.log(Object.hasOwn(obj, 'name'));       // true
console.log(Object.hasOwn(obj, 'inherited'));  // false
console.log(Object.hasOwn(nullProto, 'name')); // true（安全）

// 推荐使用 Object.hasOwn() 替代 hasOwnProperty
```

## 6.12 可选链操作符 ?.

安全地访问深层嵌套属性。

```javascript
const user = {
    name: '张三',
    address: {
        city: '北京',
        street: {
            name: '长安街',
            number: '100 号'
        }
    }
};

// 传统方式：多层判断
let city;
if (user && user.address) {
    city = user.address.city;
}
console.log(city);  // '北京'

// 传统方式：逻辑运算符
city = user && user.address && user.address.city;

// 现代方式：可选链
city = user?.address?.city;
console.log(city);  // '北京'

// 访问不存在的属性返回 undefined（不会报错）
const zip = user?.address?.zipCode;
console.log(zip);  // undefined

// 不存在嵌套对象的情况
const emptyUser = {};
console.log(emptyUser?.address?.city);  // undefined（不会报错）

// 可选链调用函数
const printer = {
    log: (msg) => console.log(msg)
};
printer?.log?.('Hello');  // 'Hello'

const noPrinter = {};
noPrinter?.log?.('Hello');  // 什么都不发生（不会报错）

// 可选链访问数组
const items = [1, 2, 3];
console.log(items?.[0]);  // 1
console.log(items?.[10]); // undefined

// 与空值合并操作符配合
const cityOrDefault = user?.address?.city ?? '未知城市';
console.log(cityOrDefault);  // '北京'
```

## 6.13 空值合并操作符 ??

为 null 或 undefined 提供默认值。

```javascript
// 基本用法
const value1 = null ?? '默认值';
const value2 = undefined ?? '默认值';
const value3 = 0 ?? '默认值';
const value4 = '' ?? '默认值';
const value5 = false ?? '默认值';

console.log(value1);  // '默认值'
console.log(value2);  // '默认值'
console.log(value3);  // 0（不是默认值）
console.log(value4);  // ''（不是默认值）
console.log(value5);  // false（不是默认值）

// 与 || 的区别
const count = 0;
console.log(count || 10);   // 10（0 被视为 falsy）
console.log(count ?? 10);   // 0（只有 null/undefined 才用默认值）

// 实用场景：配置默认值
function createComponent(options = {}) {
    const config = {
        size: options.size ?? 16,      // 默认字号
        color: options.color ?? '#000', // 默认颜色
        bold: options.bold ?? false,   // 默认不加粗
        text: options.text ?? '',      // 默认空文本
    };
    return config;
}

console.log(createComponent({ size: 20 }));
// { size: 20, color: '#000', bold: false, text: '' }

// 实用场景：API 响应处理
const apiResponse = { data: null };
const data = apiResponse?.data ?? [];
console.log(data);  // []

// 实用场景：表单输入
function getUserInput(input) {
    return {
        name: input?.name ?? '匿名用户',
        age: input?.age ?? 0,
        // 如果明确输入空字符串，保留空字符串
        bio: input?.bio ?? '',
    };
}
```

## 6.14 实践练习

### 练习 1：数据处理管道

```javascript
// 需求：处理用户数据
const users = [
    { id: 1, name: '张三', age: 25, tags: ['前端', 'Vue'] },
    { id: 2, name: '李四', age: 17, tags: ['后端', 'Java'] },
    { id: 3, name: '王五', age: 30, tags: ['全栈'] },
    { id: 4, name: '赵六', age: 15, tags: [] },
];

// 1. 筛选成年用户
// 2. 提取他们的技能标签并扁平化
// 3. 去重
// 4. 按字母排序

const adultSkills = users
    .filter(u => u.age >= 18)
    .flatMap(u => u.tags)
    .filter((skill, index, arr) => arr.indexOf(skill) === index)
    .toSorted();

console.log(adultSkills);  // ['前端', '全栈', '后端', 'Vue', 'Java']
```

### 练习 2：安全访问嵌套对象

```javascript
// 需求：安全获取 API 响应数据
const apiResponse = {
    code: 200,
    data: {
        user: {
            profile: {
                name: '张三',
                contact: {
                    email: 'zhang@example.com'
                }
            }
        }
    }
};

// 使用可选链和空值合并安全获取数据
const getUserEmail = (response) => {
    return response?.data?.user?.profile?.contact?.email ?? 'no-email@example.com';
};

console.log(getUserEmail(apiResponse));  // 'zhang@example.com'
console.log(getUserEmail({}));           // 'no-email@example.com'
console.log(getUserEmail(null));         // 'no-email@example.com'
```

### 练习 3：对象属性过滤

```javascript
// 需求：过滤对象中的敏感属性
const userData = {
    id: 1,
    name: '张三',
    email: 'zhang@example.com',
    password: 'secret123',
    token: 'abc123',
    createdAt: '2024-01-01'
};

// 方法 1：使用 Object.fromEntries
const SENSITIVE_KEYS = ['password', 'token'];
const sanitizeUser = (obj) => Object.fromEntries(
    Object.entries(obj).filter(([key]) => !SENSITIVE_KEYS.includes(key))
);

console.log(sanitizeUser(userData));
// { id: 1, name: '张三', email: 'zhang@example.com', createdAt: '2024-01-01' }

// 方法 2：只保留允许的字段
const ALLOWED_KEYS = ['id', 'name', 'email'];
const pickUserFields = (obj) => Object.fromEntries(
    Object.entries(obj).filter(([key]) => ALLOWED_KEYS.includes(key))
);

console.log(pickUserFields(userData));
// { id: 1, name: '张三', email: 'zhang@example.com' }
```

## 6.15 常见问答

### Q1: flat() 的性能如何？深度扁平化会影响性能吗？

**答：** `flat()` 会创建新数组，对于大型数组或深度嵌套的数组，性能开销较大。建议：
- 优先使用 `flatMap()` 代替 `map() + flat()`
- 明确指定扁平深度，避免使用 `Infinity`
- 对于超大数据集，考虑使用生成器逐个产出元素

### Q2: toSorted/toReversed/toSpliced 与原地方法如何选择？

**答：**
- 如果需要保留原数组（函数式编程、React 状态等），使用 `to*` 方法
- 如果不需要保留原数组且关注性能，使用原地方法
- 在 Redux、React 等不可变数据场景中，优先使用 `to*` 方法

### Q3: 可选链和空值合并可以混合使用吗？

**答：** 可以，而且经常一起使用：

```javascript
const value = obj?.property ?? 'default';
```

但注意运算符优先级，必要时加括号。

### Q4: Object.hasOwn() 在所有浏览器都支持吗？

**答：** Object.hasOwn() 是 ES2022 特性，旧浏览器需要 polyfill：

```javascript
if (!Object.hasOwn) {
    Object.hasOwn = function(obj, prop) {
        return Object.prototype.hasOwnProperty.call(obj, prop);
    };
}
```
