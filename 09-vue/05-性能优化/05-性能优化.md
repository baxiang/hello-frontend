# Vue 性能优化 ⭐⭐⭐

> 响应式优化、组件懒加载、虚拟列表

---

## L1 理解层

**最简示例（1-3行）：**

```vue
<!-- 组件懒加载 -->
const About = () => import('./About.vue')

<!-- 条件渲染优化 -->
<div v-if="show">内容</div>  <!-- 销毁/创建 -->
<div v-show="show">内容</div>  <!-- CSS 切换 -->
```

**详细示例：**

```vue
<!-- 路由懒加载 -->
const router = createRouter({
  routes: [
    { path: '/about', component: () => import('./views/About.vue') }
  ]
})

<!-- v-for 优化 -->
<li v-for="item in items" :key="item.id">{{ item.name }}</li>

<!-- 避免不必要的响应式 -->
<script setup>
import { ref, shallowRef, markRaw } from 'vue'

// shallowRef — 只追踪 .value 变化
const config = shallowRef({ settings: { theme: 'dark' } })

// markRaw — 标记为不可响应
const largeData = markRaw(fetchLargeData())
</script>
```

## L2 实践层

### 优化策略速查

| 策略 | 方法 | 效果 |
|------|------|------|
| 组件懒加载 | `() => import()` | 减少首屏体积 |
| 避免深度响应式 | `shallowRef` | 减少响应式开销 |
| 列表优化 | `:key` 用唯一 ID | 减少 DOM 操作 |
| 计算属性缓存 | `computed` | 避免重复计算 |
| 虚拟列表 | `vue-virtual-scroller` | 大数据渲染 |
