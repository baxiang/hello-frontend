## 1.1 Vue 简介

### 什么是 Vue？

Vue 是一个渐进式 JavaScript 框架，用于构建用户界面。

```
┌─────────────────────────────────────────────────────────────┐
│  Vue 核心特点                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  渐进式      → 可逐步采用，从简单到复杂                     │
│  响应式      → 自动追踪依赖，高效更新                       │
│  组件化      → 可复用的独立 UI 单元                         │
│  模板语法    → 直观的 HTML 模板                             │
│  单文件组件  → HTML、CSS、JS 组织在一起                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Vue vs 传统开发

```javascript
// 传统方式：命令式编程
const app = document.getElementById('app');
app.innerHTML = '<h1>Hello</h1>';
app.querySelector('h1').addEventListener('click', function() {
    this.textContent = 'Clicked';
});

// Vue 方式：声明式编程
<template>
    <h1 @click="message = 'Clicked'">{{ message }}</h1>
</template>

<script setup>
import { ref } from 'vue'
const message = ref('Hello')
</script>
```

## 1.2 环境搭建

### 使用 Vite 创建项目


## 1.3 第一个组件

### 单文件组件 (SFC)

```vue
<!-- src/App.vue -->
<script setup>
import { ref } from 'vue'

// 响应式数据
const message = ref('Hello Vue!')

// 方法
const handleClick = () => {
  message.value = 'Clicked!'
}
</script>

<template>
  <div class="app">
    <h1>{{ message }}</h1>
    <button @click="handleClick">点击我</button>
  </div>
</template>

<style scoped>
.app {
  padding: 20px;
}
h1 {
  color: #42b883;
}
</style>
```

### 组件的基本规则

```vue
<!-- ✅ 正确：单文件组件结构 -->
<script setup>
// JavaScript 逻辑
</script>

<template>
<!-- HTML 模板 -->
</template>

<style scoped>
/* CSS 样式 */
</style>

<!-- ✅ 正确：使用 setup 语法糖 -->
<script setup>
import { ref } from 'vue'
const count = ref(0)
</script>

<template>
  <button @click="count++">{{ count }}</button>
</template>
```

## 1.4 模板语法

### 文本插值

```vue
<template>
  <div>
    <!-- 使用 {{ }} 插值 -->
    <p>{{ message }}</p>
    
    <!-- 使用表达式 -->
    <p>{{ count + 1 }}</p>
    <p>{{ message.toUpperCase() }}</p>
    <p>{{ count > 10 ? '大于10' : '小于10' }}</p>
  </div>
</template>

<script setup>
import { ref } from 'vue'
const message = ref('Hello')
const count = ref(5)
</script>
```

### 属性绑定

```vue
<template>
  <div>
    <!-- v-bind 绑定属性 -->
    <img v-bind:src="imageUrl" />
    
    <!-- 简写 : -->
    <img :src="imageUrl" :alt="title" />
    
    <!-- 动态属性名 -->
    <a :[attributeName]="url">链接</a>
    
    <!-- class 绑定 -->
    <div :class="{ active: isActive }">内容</div>
    <div :class="['base', isActive ? 'active' : '']">内容</div>
    
    <!-- style 绑定 -->
    <div :style="{ color: textColor, fontSize: '16px' }">内容</div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
const imageUrl = ref('photo.jpg')
const title = ref('照片')
const isActive = ref(true)
const textColor = ref('red')
</script>
```

## 1.5 条件渲染

```vue
<template>
  <div>
    <!-- v-if 完全移除元素 -->
    <div v-if="isVisible">显示内容</div>
    <div v-else>隐藏内容</div>
    
    <!-- v-else-if -->
    <div v-if="type === 'A'">A</div>
    <div v-else-if="type === 'B'">B</div>
    <div v-else>其他</div>
    
    <!-- v-show 只是隐藏（display: none） -->
    <div v-show="isVisible">显示/隐藏</div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
const isVisible = ref(true)
const type = ref('A')
</script>
```

### v-if vs v-show

| 特性 | v-if | v-show |
|------|------|------|
| 渲染方式 | 条件为真才渲染 | 总是渲染 |
| 切换开销 | 高（销毁/创建） | 低（CSS 切换） |
| 适用场景 | 条件很少改变 | 频繁切换 |

## 1.6 列表渲染

```vue
<template>
  <ul>
    <!-- v-for 渲染列表 -->
    <li v-for="item in items" :key="item.id">
      {{ item.name }} - ¥{{ item.price }}
    </li>
    
    <!-- 使用索引 -->
    <li v-for="(item, index) in items" :key="item.id">
      {{ index + 1 }}. {{ item.name }}
    </li>
    
    <!-- 遍历对象 -->
    <li v-for="(value, key) in object" :key="key">
      {{ key }}: {{ value }}
    </li>
    
    <!-- 遍历数字 -->
    <span v-for="n in 10" :key="n">{{ n }}</span>
  </ul>
</template>

<script setup>
import { ref } from 'vue'
const items = ref([
  { id: 1, name: '苹果', price: 5 },
  { id: 2, name: '香蕉', price: 3 },
  { id: 3, name: '橙子', price: 4 }
])
const object = ref({ name: '张三', age: 25, city: '北京' })
</script>
```

## 1.7 事件处理

```vue
<template>
  <div>
    <!-- v-on 绑定事件 -->
    <button v-on:click="handleClick">点击</button>
    
    <!-- 简写 @ -->
    <button @click="handleClick">点击</button>
    
    <!-- 内联处理器 -->
    <button @click="count++">{{ count }}</button>
    
    <!-- 传递参数 -->
    <button @click="handleClick('参数')">点击</button>
    
    <!-- 访问事件对象 -->
    <button @click="handleClick($event)">点击</button>
    
    <!-- 事件修饰符 -->
    <form @submit.prevent="handleSubmit">
      <button type="submit">提交</button>
    </form>
    
    <!-- 按键修饰符 -->
    <input @keyup.enter="handleEnter" />
  </div>
</template>

<script setup>
import { ref } from 'vue'
const count = ref(0)

const handleClick = (param) => {
  console.log('点击', param)
}

const handleSubmit = () => {
  console.log('表单提交')
}

const handleEnter = () => {
  console.log('按下回车')
}
</script>
```

### 常用事件修饰符

```vue
<!-- 阻止默认行为 -->
<form @submit.prevent="onSubmit"></form>

<!-- 阻止事件冒泡 -->
<div @click.stop="onClick"></div>

<!-- 只触发一次 -->
<button @click.once="onClick"></button>

<!-- 捕获模式 -->
<div @click.capture="onClick"></div>

<!-- 只当事件在该元素本身触发 -->
<div @click.self="onClick"></div>

<!-- 链式调用 -->
<div @click.stop.prevent="onClick"></div>
```

## 1.8 双向绑定

```vue
<template>
  <div>
    <!-- v-model 双向绑定 -->
    <input v-model="message" />
    <p>{{ message }}</p>
    
    <!-- 不同输入类型 -->
    <textarea v-model="content"></textarea>
    
    <input type="checkbox" v-model="checked" />
    
    <input type="radio" v-model="picked" value="one" />
    <input type="radio" v-model="picked" value="two" />
    
    <select v-model="selected">
      <option value="a">A</option>
      <option value="b">B</option>
    </select>
    
    <!-- v-model 修饰符 -->
    <input v-model.lazy="message" />    <!-- 惰性更新 -->
    <input v-model.number="age" />      <!-- 转为数字 -->
    <input v-model.trim="text" />       <!-- 去除空格 -->
  </div>
</template>

<script setup>
import { ref } from 'vue'
const message = ref('')
const content = ref('')
const checked = ref(false)
const picked = ref('one')
const selected = ref('a')
const age = ref(0)
const text = ref('')
</script>
```

## 1.9 常见问题

### Q1: Vue 2 和 Vue 3 有什么区别？

- Vue 3 使用 Composition API
- Vue 3 响应式系统基于 Proxy
- Vue 3 更好的 TypeScript 支持
- Vue 3 性能更好

### Q2: 为什么需要 key？

key 帮助 Vue 识别列表中哪些元素发生了变化，从而高效更新 DOM。使用索引作为 key 在列表顺序变化时可能导致问题。

### Q3: ref 和 reactive 有什么区别？

- `ref`：用于基础类型，访问时需要 `.value`
- `reactive`：用于对象，直接访问属性
